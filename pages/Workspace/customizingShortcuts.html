Настраивание комбинаций клавиш

Хотя комбинации клавиш по умолчанию во многом соответствуют отраслевым стандартам и повсеместно используются в других приложениях, можно настроить их в соответствии с вашими предпочтениями.

Можно настроить комбинации клавиш для отдельного приложения или режима работы. Комбинации клавиш можно настраивать для инструментов, пунктов меню, включения или выключения панелей и действий отдельного инструмента. Для этого можно либо добавить новую комбинацию клавиш, либо заменить уже существующую.

Дополнительно можно сохранить настроенные комбинации клавиш в файл (.affshortcuts) для последующей передачи другим пользователям либо резервного копирования с целью безопасного хранения.

Ниже представлены доступные параметры.

  • Режим: во всплывающем меню задается режим для настраивания комбинаций клавиш.
  • Элемент интерфейса: во всплывающем меню отображаются меню, команды и действия для текущего выбранного режима. В категории «Разное» сгруппированы комбинации клавиш для часто выполняемых операций; здесь можно настроить включение или выключение панелей посредством определенных комбинаций клавиш.
  • Применить ко всем: если в этом поле установлен флажок, комбинация клавиш, примененная к элементу интерфейса, используется во всех режимах. Если флажок в поле не установлен, в исходном всплывающем меню «Режим» можно назначить уникальную пользовательскую комбинацию клавиш, которая будет работать только в текущем выбранном режиме.
  • Игнорировать модификатор: позволяет создавать комбинации клавиш с помощью однобуквенного обозначения вместо использования модификаторов клавиатуры.
  • Сбросить: используется для сброса всех пользовательских комбинаций клавиш до значений по умолчанию.
  • Очистить все комбинации клавиш: применяется для удаления всех комбинаций клавиш (используемых по умолчанию и пользовательских).
  • Загрузить: используется для загрузки ранее сохраненного файла .affshortcuts. Это действие приведет к перезаписи распределений текущих комбинаций клавиш.
  • Сохранить: используется для сохранения текущего набора комбинаций клавиш в файл .affshortcuts для последующей передачи или резервного копирования.
Чтобы назначить (повторно назначить) комбинацию клавиш, выполните указанные ниже действия.
  1. В меню Affinity Designer выберите Установки.
  2. В меню Правка выберите пункт Установки.
  3. Перейдите на вкладку Комбинации клавиш.
  4. В двух исходных всплывающих меню отфильтруйте комбинации клавиш сначала по режиму, а потом по элементу интерфейса соответственно.
  5. В нижнем списке комбинаций клавиш нажмите поле (с незаполненной или имеющейся комбинацией клавиш) в конце записи о комбинации клавиш, которую необходимо изменить.
  6. Для назначения нажмите выбранную одну или несколько клавиш. Это действие приводит к добавлению новой комбинации клавиш или перезаписыванию назначенной.

Если выбранная комбинация клавиш уже была назначена, рядом с ней появится предупреждающий треугольный знак (). Для устранения этой проблемы назначьте другую комбинацию клавиш.

Рядом с некоторыми комбинациями клавиш отображается символ (). Он означает, что комбинация клавиш используется несколькими инструментами. Наведите курсор на символ, чтобы увидеть, что это за инструменты.

Чтобы удалить комбинацию клавиш, выполните указанные ниже действия.
  • Нажмите значок перекрестья в конце записи о комбинации клавиш.
