Создание пользовательских кистей

Программа Affinity предоставляется изначально с целым рядом предустановленных мазков кисти, которые можно настроить в соответствии с вашими потребностями. Можно также создавать пользовательские кисти с нуля с помощью собственных изображений.

Чтобы изменить предустановленный мазок кисти, выполните указанные ниже действия.
  1. Перейдите на панель Кисти.
    • Выберите предустановку.
    • Нажмите Правка.
  2. Выберите нужные параметры в разделе Начертание.
  3. Выполните одно из перечисленных ниже действий.
    • Нажмите Дублировать, чтобы сохранить измененную предустановку в новой пользовательской предустановке.
    • Нажмите Закрыть, чтобы сохранить пользовательские параметры в текущей предустановке.
    • Нажмите Сбросить, чтобы возвратиться к параметрам предустановки.

Чтобы получить дополнительные сведения об изменении параметров начертания, см. тему Изменение мазков кисти.

Чтобы создать пользовательский предустановленный мазок кисти с нуля, выполните указанные ниже действия.

Перейдите на панель Кисти, нажмите значок установок панели и выберите указанные ниже значения.

  • Создать сплошную кисть: используется для создания базового сплошного векторного мазка.
  • Создать текстурированную кисть интенсивности: используется для создания мазка кисти на основании значений непрозрачности растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.
  • Создать текстурированную кисть с изображением: используется для создания мазка кисти на основании значений цвета растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

Новая кисть добавляется в выбранную категорию с параметрами по умолчанию.

Дополнительные параметры в меню установок панели позволяют создавать, переименовывать или удалять пользовательские категории кистей, а также импортировать наборы кистей в виде новой категории либо экспортировать текущую категорию в виде набора кистей.
