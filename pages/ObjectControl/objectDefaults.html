Настройки объекта по умолчанию

Внешний вид новых объектов при создании изначально определяется настройками по умолчанию, установленными для определенного создаваемого объекта.

Эти настройки по умолчанию можно менять в текущем документе, устанавливать в качестве глобальных параметров для всех будущих документов или сбросить к заводским установкам в любое время. Функция синхронизации настроек по умолчанию позволяет создавать новые настройки по умолчанию, назначенные на основании параметров текущего выделенного объекта, с возможностью их сохранения или возврата к сохраненным ранее параметрам по умолчанию.

Настройки объекта по умолчанию сохраняются отдельно для атрибутов обводки и заливки, атрибутов художественного текста и атрибутов текстовой области.

Для синхронизации настроек по умолчанию текущего выделенного объекта выполните перечисленные далее действия.
  1. Выделите объект с атрибутами, которые следует сохранить в качестве настроек по умолчанию.
  2. На панели инструментов нажмите Синхронизация настроек по умолчанию с выделенным объектом.

Синхронизированные настройки по умолчанию отменяют сохраненные ранее параметры по умолчанию только для этого документа. Этот способ отлично подходит для работы со временным набором настроек по умолчанию для одного документа без необходимости сохранять их.

Для сохранения настроек по умолчанию выполните перечисленные далее действия.
  • В меню Правка выберите пункт Настройки по умолчанию, а затем Сохранить.

Настройки по умолчанию, используемые для этого документа, станут глобальными для всех будущих документов.

Для возврата синхронизированных настроек по умолчанию к сохраненным ранее параметрам выполните перечисленные далее действия.

  • На панели инструментов выберите Отменить изменения настроек по умолчанию.

    Если в текущий момент выделен объект, его атрибуты будут возвращены к параметрам по умолчанию.

Для сброса настроек по умолчанию к заводским установкам выполните перечисленные далее действия.

  • В меню Правка выберите пункт Настройки по умолчанию, а затем Восстановить заводские настройки.

    Если в текущий момент выделен объект, его атрибуты будут возвращены к параметрам по умолчанию.
