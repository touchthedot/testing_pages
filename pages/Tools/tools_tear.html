Капля

Инструмент для рисования фигур Капля позволяет быстро и легко создавать различные формы капель.

Настройка

В инструменте «Капля» есть несколько вариантов выбора формы и параметров на контекстной панели инструментов, с помощью которых можно регулировать размер шара, кривую, положение хвоста и изгиб.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Фиксированный размер шара: по умолчанию при изменении размера объекта масштабируется эллиптический конец фигуры. Если этот параметр включен, можно дополнительно указать фиксированный размер.
    • Размер шара: используется для регулирования кривизны эллиптического конца фигуры. При выборе значения 0 % для фигуры устанавливается абсолютно плоское основание.
  • Кривая: используется для регулирования кривизны краев, соединяющих эллиптический конец фигуры с противоположным острым концом.
  • Положение хвоста: используется для регулирования положения острого конца фигуры слева (0 %) направо (100 %), где 50 % — это центральная точка.
  • Изгиб: используется для регулирования кривизны краев (см. параметр Кривая выше). При выборе положительных значений оба края изгибаются вправо (при этом левый край затрагивается больше); при выборе отрицательных значений оба края изгибаются влево (при этом правый край затрагивается больше).
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
