Сектор

С помощью инструмента для рисования фигур Сектор можно легко создавать секторные фигуры, сегменты и кольца с удаленными сегментами. Поскольку сегменты имеют правильную геометрическую форму, упрощается создание фигур с точным значением угла, который необходим для диаграмм и других целей.

Настройка

В инструменте «Сектор» есть несколько вариантов выбора формы и параметров на контекстной панели инструментов, с помощью которых можно регулировать начальный и конечный углы, а также размер отверстия кольца.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Радиус отверстия: используется для регулирования ширины отверстия в центре фигуры.
  • Начальный / конечный угол: используется для установки положения начальной / конечной точек относительно 360°. Чем больше разница между углами, тем больше площадь сектора.
  • Общий угол: используется для установки положения общего угла относительно начального угла.
  • Инвертировать углы: используется для переключения значений начального и конечного углов. С помощью этого параметра можно преобразовать фигуры с острыми углами в фигуры с тупыми углами (и наоборот), а также отразить фигуру по горизонтали и (или) по вертикали.
  • Закрыть сектор: используется для замыкания фигуры в случае, если круг не полный, и создания замкнутого круга.
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
