Упорядочение объектов

Каждый объект на странице занимает определенное положение в z-последовательности этого проекта (порядок элементов в стеке). В случае перекрывающихся объектов с помощью этой функции можно задать, какие части объекта остаются видимыми, а какие — скрытыми.

Каждому нарисованному или вставленному объекту присваивается определенное положение в z-последовательности текущего выбранного слоя. Исходное положение объекта определяется с помощью функции нацеливания. Однако при необходимости положение всех объектов в z-последовательности слоя можно изменять.

Для изменения положения объекта в z-последовательности выполните одно из указанных ниже действий.
  1. Выделите один или несколько объектов.
  2. На панели инструментов выберите один из перечисленных ниже вариантов.
    • Переместить на задний план: используется для перемещения выделенных объектов в нижнюю часть слоя.
    • На 1 позицию назад: используется для перемещения выделенных объектов на одну позицию вниз в слое.
    • На 1 позицию вперед: используется для перемещения выделенных объектов на одну позицию вверх в слое.
    • Переместить на передний план: используется для перемещения выделенных объектов сверху слоя.

    Описанные выше параметры также доступны в подменю Упорядочить в меню Слой.

Можно также менять положение объектов или вкладывать их друг в друга с помощью панели Слои.
