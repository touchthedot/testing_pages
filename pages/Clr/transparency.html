Прозрачность

В Affinity Designer прозрачность применяется самыми разнообразными способами.

Ниже представлены наиболее типичные способы использования этого параметра:

  • как сплошная непрозрачность, примененная к обводке, заливке и мазкам кисти;
  • как градиент прозрачности, опять же примененный к обводке или заливке;
  • как непрозрачность слоя;
  • как свойство эффекта слоя.

В процессе работы в Affinity Designer вам будет встречаться термин «непрозрачность». Этот параметр противопоставляется прозрачности. Объект, для которого установлено значение прозрачности 0 %, является абсолютно непрозрачным (100 % непрозрачности). А значение прозрачности 100 % означает, что объект невидимый (0 % непрозрачности).

Клавиши быстрого доступа к управлению непрозрачностью

Чтобы установить непрозрачность слоя для выделенных слоев или объектов, можно использовать клавиши с цифрами.

Этот способ быстрого доступа к настройкам непрозрачности также можно использовать при работе с инструментами на основе кистей в режиме «Пиксельный».

Чтобы изменить сплошную непрозрачность объекта, выполните приведенные далее действия.
  1. На панели «Цвет» выберите обводку или заливку с помощью селектора цвета обводки или заливки.
  2. Перетащите ползунок Непрозрачность.
Для того чтобы применить градиент прозрачности к объекту, выполните перечисленные далее действия.
  1. На панели Инструменты выберите инструмент «Прозрачность».
  2. Для того чтобы применить градиент прозрачности, перетаскивайте курсор по объекту.
Чтобы применить непрозрачность слоя, выполните перечисленные далее действия.
  1. Выделите слой на панели Слои.
  2. Чтобы отрегулировать параметр Непрозрачность нажмите стрелку вниз на панели и перетащите ползунок.
Использование клавиш быстрого доступа к управлению непрозрачностью.
  1. Выполните одно из перечисленных ниже действий.
    • Выделите один или несколько слоев или объектов.
    • Выберите инструмент «Кисть» в режиме «Пиксельный».
  2. Нажмите клавишу с цифрой или две клавиши с цифрами подряд, чтобы установить непрозрачность. Пример.
    • Нажмите 4, чтобы установить значение непрозрачности 40 %.
    • Нажмите 0, чтобы установить значение непрозрачности 100 %.
    • Нажмите 4 и 5, чтобы установить значение непрозрачности 45 %.
    • Нажмите 0 и 7, чтобы установить значение непрозрачности 7 %.
Чтобы применить эффект непрозрачности слоя, выполните перечисленные далее действия.
  1. На панели Слои нажмите Эффекты слоя.
  2. В диалоговом окне отрегулируйте параметр Непрозрачность с помощью соответствующего ползунка (при наличии). Параметр «Непрозрачность» предусмотрен не для всех эффектов слоя.
