Параллельные проекции

В Affinity Designer используется сетка проекции с широкими возможностями настройки, идеально подходящая для проектирования интерфейса пользователя и игр, цифрового проектирования и создания моделей.

О параллельных проекциях

Можно включить наборы плоскостей, чтобы работать над проектом в передней, боковых и верхней плоскостях. Переключение между плоскостями может быть выполнено с помощью одной клавиши-модификатора.

Вместо обычно используемой изометрической сетки проекции можно выбрать одну из предустановленных сеток проекта (например, диметрическую или треугольную). Кроме того, для более продвинутого использования можно даже создавать пользовательские сетки проекции с параметрами изменения размера сетки, настройки пропорций и интервалов.

Рисование на сетках проекций

После установки определенного режима и выбора плоскости можно рисовать линии и фигуры с помощью инструмента «Перо» или предварительно спроектированных геометрических фигур.

Чтобы рисовать пером прямые линейные сегменты, воспользуйтесь режимами инструмента «Перо» «Многоугольник» или «Линия». Чтобы создавать геометрические фигуры, изменяйте их форму так, чтобы она соответствовала сетке.

Об элементах управления привязкой

Наибольшая эффективность достигается при использовании сеток в сочетании с функцией привязки (включается на верхней панели инструментов), в частности, когда во всплывающем меню кнопки «Привязка» выбран параметр «Привязать к сетке». Маркеры объекта с высокой точностью привязываются к любой линии сетки или пересечению линий.

Для сеток можно установить любые единицы измерения в документе, отображаемые при включенном представлении линеек.

О пользовательских сетках

Если вам требуются более продвинутые сетки, чем те, что предусмотрены предустановками режима проекции (например, изометрическая), можно настроить сетку в соответствии с вашими предпочтениями.

Чтобы установить сетку проекции, выполните указанные ниже действия.
  1. В меню Вид выберите пункт Диспетчер сеток и осей.
  2. Снимите флажок Использовать автоматическую сетку.
  3. Для перехода к сеткам проекции нажмите Расширенные.
  4. Во всплывающем меню Тип сетки выберите режим проекции, например «Изометрическая».

Диспетчер сеток и осей также идеально подходит для размещения двухмерных фиксированных сеток и сеток проекций.

Для создания набора плоскостей выполните указанные ниже действия.
  • В диалоговом окне установите флажок Создать набор плоскостей.

Теперь можно чередовать плоскости при необходимости работать над проектом в верхней, боковых или передней плоскостях рисунка.

Для переключения плоскостей выполните указанные ниже действия.
  • Нажмите '.
Для настройки сетки проекции выполните указанные ниже действия.
  1. Во всплывающем меню Тип сетки выберите вариант Двухосевая пользовательская или Треугольная пользовательская.
  2. Выберите Создать набор плоскостей, чтобы активировать и настроить ось вертикально вверх.
  3. Выберите нужные параметры в диалоговом окне.
  4. Нажмите Закрыть.
