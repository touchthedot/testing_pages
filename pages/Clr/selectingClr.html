Выбор цветов

В приложении Affinity Designer существует несколько способов выбора цветов для композиции.

О выборе цветов

Важным требованием для обеспечения эффективности работы по созданию композиции является простой и интуитивно понятный интерфейс доступа к цветам. Поскольку у дизайнеров могут быть свои предпочтения в отношении методов выбора цветов, в приложении Affinity Designer предусмотрен целый ряд описанных ниже вариантов отбора проб.

  • Контекстная панель инструментов: используется для выбора цветов заливки и (или) обводки на контекстной панели инструментов, которая отображается при выделении векторного объекта.
  • Панель «Цвет»: используется для выбора цвета с помощью цветового круга, палитр или путем перемещения ползунков с возможностью выбора различных цветовых моделей.
  • Панель «Образцы»: используется для выбора цветов по образцам из различных предустановленных или пользовательских категорий, включая цвета PANTONE®.
  • Инструмент «Пипетка»: используется для выбора образцов цветов в любом месте на экране. Отлично подходит для выполнения дополнительных операций при работе с цветом.
  • Инструмент «Заливка»: используется для применения цветного градиента к объекту и его редактирования. Выберите нужный вариант на контекстной панели инструмента.

Использование контекстной панели инструментов объекта

Для получения быстрого доступа к образцам цветов заливки и обводки можно воспользоваться контекстной панелью инструментов любой выделенной фигуры, линии или обводки.

Для того чтобы применить цвет заливки или обводки к векторному объекту, выполните перечисленные далее действия.
  1. Выделите объект.
  2. На контекстной панели инструментов нажмите образец «Обводка» или «Заливка».
  3. Во всплывающем подменю выберите панель Образцы, Цвет или Градиент.

На панели «Цвет» можно выбирать цвета с помощью цветового круга HSL по умолчанию, путем перемещения ползунков для цветов моделей RGB, HSL, CMYK, Lab, шестнадцатеричной модели RGB и оттенков серого, а также по мере необходимости работать в режиме «8 бит», «16 бит» или «Процент» и регулировать значения независимых параметров «Оттенок», «Насыщенность» или «Светлота».

Использование панели «Цвет»

Несколькими нажатиями на панели «Цвет» можно применять цвета к объекту или задать цвет для инструмента. Кроме того, можно применять дополнительные атрибуты цвета, например непрозрачность и шум. Цвета могут быть сохранены в палитре на панели «Образцы».

Для выбора цвета обводки или заливки (используя цветовой круг HSL по умолчанию) выполните перечисленные далее действия.
  1. Нажмите образец для обводки или заливки в верхней части панели слева.
  2. Нажмите для определения цветового оттенка на внешней окружности.
  3. Нажмите расположенный внутри треугольник для комбинированной настройки значений насыщенности и яркости.

Соответствующий образец будет обновлен в зависимости от выбранного цвета.

Нажмите образец Нет для того, чтобы сделать цвет полностью прозрачным.

Раскройте меню «Установки панели» на панели «Цвет» для того, чтобы выбрать цвет с помощью ползунков цветов RGB, HSL, CMYK, LAB или оттенков серого, а также по мере необходимости работать в режиме «8 бит», «16 бит» или «Процент» и менять значения параметров «Оттенок», «Насыщенность» или «Светлота».

Для переключения селекторов образцов цветов выполните перечисленные далее действия.
  • Нажмите двунаправленную стрелку. Цвета поменяются местами (но активный селектор образца не изменится).
Чтобы отрегулировать непрозрачность или настроить параметры шума, выполните указанные ниже действия.
  1. Нажмите кнопку переключения Переключить на непрозрачность / шум в нижней части панели слева.
  2. Перетащите ползунок для того, чтобы установить значение.
Для использования другой цветовой модели выполните перечисленные далее действия.
  1. В меню «Установки панели» выберите пункт Ползунки.
  2. Справа в верхней части панели нажмите текущий цветовой режим, например RGB, для открытия всплывающего меню.
  3. Выберите другую цветовую модель в меню.
Для того чтобы применить оттенок цвета, выполните перечисленные далее действия.
  1. В меню «Установки панели» выберите Оттенок.
  2. Для увеличения или уменьшения интенсивности оттенка цвета перетаскивайте ползунок влево или вправо соответственно.
Для использования другого цветового режима выполните перечисленные далее действия.
  • В меню «Установки панели» выберите 8 бит, 16 бит или Процент.
Для сохранения цветов на панели «Образцы» выполните перечисленные далее действия.
  • В меню «Установки панели» выберите один из приведенных далее вариантов.
    • Добавить цвет в образцы: используется для добавления текущего цвета в загруженную палитру на панели «Образцы».
    • Выберите тип хорды во всплывающем меню Добавить хорду в образцы: этот параметр используется для добавления хорды текущего цвета в загруженную палитру на панели «Образцы».

Использование панели «Образцы»

На панели «Образцы» представлены предустановки образцов цветов, объединенные в различные категории. С помощью предустановок можно с легкостью применять оттенки серого, сплошные цвета или цвета градиента.

Для использования образца цвета выполните перечисленные далее действия.

Выполните одно из перечисленных ниже действий.

  • Выберите категорию палитры во всплывающем меню категорий, а затем выберите образец цвета в палитре.
  • Среди недавно использованных образцов выберите и нажмите нужный образец.
  • Нажмите образец Нет для того, чтобы сделать цвет полностью прозрачным.

Использование инструмента «Заливка»

Используйте инструмент «Заливка» для того, чтобы применять пользовательские градиентные контуры к обводке или заливке объекта. Также можно применять сплошные заливки и заливки растровыми рисунками. На контекстной панели инструментов линейную градиентную заливку можно заменить на радиальную, эллиптическую или коническую.

Для использования инструмента «Заливка» выполните перечисленные далее действия.
  1. Выделите объект.
  2. Выберите инструмент «Заливка» на панели «Инструменты».
  3. Во всплывающем меню Контекст контекстной панели инструментов выберите вариант «Обводка» или «Заливка».
  4. Во всплывающем меню Тип контекстной панели инструментов выберите тип заливки.

  5. Перетаскивайте курсор по объекту, для которого следует создать обводку или заливку (в зависимости от того, что было выбрано ранее).

    Нажмите и удерживайте для того, чтобы ограничить угол градиентного контура до 45°.

Выделите и перетаскивайте ограничители контура для того, чтобы изменить его длину и направление. На панели «Цвет» можно изменить цвет и уменьшить непрозрачность ограничителей. С помощью можно изменить положение градиента, не влияя на его длину или направление.

Создавайте собственные градиентные заливки, нажимая образец цвета рядом со всплывающим меню «Тип» на контекстной панели инструментов.

Доступ к цветам PANTONE®

Палитры цветов PANTONE® доступны на главной панели «Образцы». Они также доступны на всплывающей панели «Образцы» контекстной панели инструментов при выделении фигуры, линии или обводки.

Для доступа к цветам PANTONE® выполните перечисленные далее действия.
  • На панели Образцы (или на всплывающей панели «Образцы») раскройте всплывающее меню категорий и выберите нужную палитру PANTONE.

Приводочный цвет

Функция печати в PDF-формате позволяет включать метки принтера, в том числе метки приводки, которые будут напечатаны черным цветом (С = 100 %, М = 100 %, Y = 100 % и K = 100 %). Тем не менее этот приводочный цвет можно добавить в качестве образца, который может быть назначен для объекта на странице, что приведет к созданию метки приводки.

Для добавления приводочного цвета в качестве образца выполните перечисленные далее действия.
  • На панели «Образцы» нажмите Установки панели, а затем выберите Добавить приводочный цвет.

После этого образец может быть применен к объекту на странице.
