Мазки пиксельной кисти для рисования

Можно перейти в режим «Пиксельный» для добавления фотореалистичных текстур к векторным проектам с помощью пиксельных кистей. Либо можно воспользоваться базовыми пиксельными кистями с жесткими или размытыми краями для прорисовки теней и светлых участков в проекте.

Кисти

В Affinity Designer есть потрясающий набор различных вариантов пиксельных кистей, которые можно использовать с кистью рисования. В каждой категории содержатся кисти с различными свойствами и характеристиками.

Можно также создавать собственные пиксельные кисти, которые помогут в работе над проектом. Набор параметров для кисти может быть сохранен в виде пользовательской предустановки на панели.

Обрезка векторных объектов с помощью мазков пиксельной кисти помогает эффективно создавать тени и светлые участки.

Для нанесения мазков пиксельной кисти для рисования выполните перечисленные далее действия.
  1. Используйте панель Слои для выбора растрового слоя, на котором хотите работать, или создания нового растрового слоя.
  2. На панели Инструменты выберите Кисть рисования.
  3. На панели Кисти выберите пиктограмму кисти на ваш выбор. По умолчанию используется мягкая кисть округлой формы.
  4. Отрегулируйте параметры контекстной панели инструментов.
  5. На панели Цвет выберите цвет мазка.
  6. Перетащите инструмент по странице в направлении желаемого следования мазка кисти.
Чтобы сгладить мазки кисти по мере рисования, выполните приведенные далее действия.
  • На контекстной панели инструментов включите параметр Стабилизатор и выполните одно из перечисленных ниже действий.
    • Режим веревки: потяните конец мазка с помощью «веревки», сглаживающей мазок, но позволяющей обеспечить острые углы при увеличении значений параметра Длина (Радиус) путем изменения направления ослабленной веревки.
    • Режим окна: сглаживание мазка посредством усреднения положения мазка в Окне с настраиваемым размером.

Клавиши-модификаторы

При использовании инструмента «Кисть рисования» можно применять приведенные далее клавиши-модификаторы ().

  • : используется для проведения мазка кисти по прямой линии с конца предыдущего мазка до следующей позиции, определяемой нажатием кнопки.
  • Нажмите и, удерживая левую кнопку мыши, выберите новый цвет рисования в точке под кистью.
  • Нажмите и и выполните перетаскивание инструмента по странице. При перетаскивании влево или вправо размер кисти будет уменьшаться или увеличиваться соответственно. При перетаскивании вверх или вниз жесткость кисти будет уменьшаться или увеличиваться соответственно.
  • Нажмите правую кнопку мыши, , а затем левую кнопку и выполните перетаскивание на страницу. При перетаскивании влево или вправо размер кисти будет уменьшаться или увеличиваться соответственно.
