Художественный текст

Само название «художественный текст» подсказывает, что этот инструмент подходит для визуального выделения отдельных слов (или фраз) или применения декоративных оформительских элементов.

Художественный текст можно изменить множеством способов, подобных тем, что используются для изменения нарисованных объектов, с сохранением при этом стандартных свойств текста. Благодаря этому он идеально подходит для оформления отдельных слов, заголовков, броских цитат и т. д.

При изменении размера художественный текст растягивается или сжимается для соответствия новым размерам. Относительный размер его шрифта автоматически обновляется на контекстной панели инструментов.

Для художественного текста можно установить свойства абзаца, но, скорее всего, вы будете вносить изменения в свойства знаков и положение на странице с помощью средств проектирования и творческих решений.

Для использования необычных декоративных оформительских элементов можно преобразовать художественный текст в кривые. После преобразования художественного текста в кривые невозможно будет использовать стандартные текстовые свойства для его изменения.

Чтобы создать художественный текст, выполните приведенные далее действия.

Выберите инструмент «Художественный текст».

  1. Нажмите кнопку мыши или перетащите курсор по странице, чтобы сохранить размер текста по умолчанию или изменить его в соответствии с потребностями.
  2. Выполните одно из перечисленных ниже действий.
    • Введите текст.
    • Вставьте предварительно скопированный текст.
    • В меню Текст выберите Вставить текст из файла. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

При копировании текста из внешних приложений воспользуйтесь функцией вставки без форматирования (меню Правка) для сохранения «чистого» неформатированного текста.

Изменение размера художественного текста

После выбора художественного текста выполните одно из перечисленных ниже действий.

  • Чтобы одновременно изменять высоту и ширину (сохраняя соотношение сторон), перетягивайте угловые маркеры объекта.
  • Чтобы изменять отдельно высоту и ширину (без сохранения соотношения сторон), перетягивайте центральный и боковые маркеры объекта.

Клавиши-модификаторы

При использовании инструмента «Художественный текст» можно использовать приведенные далее клавиши-модификаторы.

  • С помощью клавиши можно инвертировать нормы изменения размера (как описано выше).
  • С помощью клавиши можно изменить размер текста из центра.
  • С помощью клавиши можно вращать размещенный текст вокруг противоположного маркера.
  • С помощью нажатия правой кнопки мыши можно вращать размещенный текст вокруг противоположного маркера.

При активном инструменте «Текст» для применяется описанная далее логическая схема.

  1. Если используется при выбранных глифах (знаках), с глифов снимается выделение, но каретка (курсор) остается внутри текстового объекта. (Можно продолжать редактирование текста в текстовом объекте.)
  2. Если каретка находится внутри текстового объекта, но глифы не выбраны, использование приведет к удалению каретки и выделению текстового объекта. (Нажмите V для перехода к инструменту «Перемещение» и изменению положения текстового объекта.)
  3. При выбранном текстовом объекте нажмите , чтобы отменить выделение текстового объекта.
