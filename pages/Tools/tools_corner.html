Инструмент «Угол»

Инструмент «Угол» позволяет округлять острые углы геометрических фигур или замкнутых фигур, нарисованных с помощью инструмента «Перо».

Скругленные углы создаются из узла острого угла посредством перетаскивания по направлению внутрь или наружу для формирования кольца, вокруг которого будет формироваться скругленный угол. Чем дальше вы будете перетаскивать указатель, увеличивая радиус, тем более выраженным будет угол.

Клавиша быстрого доступа к инструменту : C

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Заливка: нажмите на образец цвета для отображения всплывающей панели и обновите цвет заливки.
  • Обводка: нажмите на образец цвета для отображения всплывающей панели и обновите цвет обводки.
  • Ширина: толщина линии в пикселях. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Тип угла: позволяет выбрать тип угла, который будет использоваться для скругления угла. Можно выбрать следующие параметры: Нет, Скругленный (угловой), Прямой (скошенный), Вогнутый (дугообразный) или Вырезанный.
  • Радиус угла: позволяет установить расстояние от острого узла до скругленного угла.
  • Скруглить углы: преобразует закругленные углы в кривые. Если выбран этот параметр, созданный угол больше нельзя будет изменять.
