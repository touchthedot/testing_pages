Сохранить

При сохранении документа вся информация проекта сохраняется, что позволяет редактировать его позднее. Проект сохраняется в виде отдельного файла с расширением .afdesign.

При первом сохранении документа вам будет предложено дать проекту имя и указать место сохранения. При последующих сохранениях новые файлы будут перезаписывать предыдущие версии.

По желанию текущий проект можно сохранить в качестве нового проекта. В этом случае новый файл не будет перезаписывать сохраненную версию.

Историю можно сохранить вместе с документом для того, чтобы можно было вернуться на более ранние этапы редактирования даже в случае, если документ был закрыт и вновь открыт.

Для сохранения документа в качестве нового проекта выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Сохранить как.
  2. Отрегулируйте параметры диалогового окна по мере необходимости.
  3. Нажмите Сохранить.
Для сохранения текущего проекта выполните перечисленные далее действия.
  • В меню Файл выберите пункт Сохранить.

Если были внесены только те изменения, которые поддерживаются типом файла изображения (например, JPG или PNG), тогда при использовании функции Сохранить новый файл перезапишет текущий файл. Если были внесены изменения, которые не поддерживаются, например добавление слоев, то вы можете сохранить изображение как файл проекта или выполнить сведение и объединение изменений в существующем файле.

Для сохранения истории вместе с документом выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Сохранить документ с историей.
  2. В диалоговом окне Сохранение с историей нажмите Да для того, чтобы согласиться с условиями, описанными в диалоговом окне.
  3. Сохраните документ.

Сохранение документа вместе с историей может привести к значительному увеличению размера файла проекта.

Функцию сохранения истории можно отключить, повторив шаг 1 описанной выше процедуры.
