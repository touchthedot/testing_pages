Настройка документа

В Affinity Designer можно просмотреть и изменить настройку документа в любой момент. Выбор этой настройки осуществляется на этапе создания документа.

Для изменения настройки документа выполните указанные ниже действия.
  • В меню Файл выберите пункт Настройка документа.

Параметры представлены ниже.

  • Тип: используется для определения назначения и конечного результата проекта (для быстрой настройки приведенных ниже параметров). Помимо печати готовых для вывода на печать изображений CMYK, вы можете работать с конкретными размерами фотопечати и определенными разрешениями экрана для отображения веб-страниц, а благодаря функции «Устройства» также можете создавать документы, соответствующие спецификациям iPad, iPhone, Nexus, Kindle Fire HD и Galaxy S5.
  • Предустановка страницы: используется для настройки размеров в зависимости от характера использования экрана и типа мобильных устройств, а также параметров фотопечати и печати документов по стандартам, принятым в США и Великобритании. Параметр «Пользовательский» позволяет определить, если для страницы указаны нестандартные размеры.
  • Единицы измерения документа: используется для отображения линеек и размеров объекта в пикселях, точках или физических единицах измерения.

Диалоговое окно разбито на отдельные вкладки, в которых можно настраивать текущую предустановку или применять расширенные параметры.

  • Размеры.
    • Размеры: используется для изменения значений с целью настройки пользовательского размера страницы. (Является альтернативой выбору описанного выше параметра «Предустановка страницы».)
    • Разрешение: используется для установки разрешения документа. Например, для получения профессионального качества печати установите разрешение 300 тчк/дюйм.
    • Портретная: если флажок не установлен, документ будет представлен в альбомной ориентации.
    • Масштабирование «Фактический размер»: используется для определения устройства, которое имитируется при представлении документа в фактическом размере (меню Вид). Этот параметр доступен только тогда, когда в качестве единиц измерения документа указаны пиксели или точки. Выберите этот вариант во всплывающем меню.
    • Нажмите С изменением масштаба объекта, чтобы увеличить или уменьшить размеры объекта относительно изменения ширины и (или) высоты страницы.
    • Используйте параметр С привязкой объекта к странице, чтобы сохранить размер объекта неизменным при изменении размера страницы.
    • Точка привязки: используется для управления масштабом объектов (если выбран параметр «С изменением масштаба объекта») относительно текущей заданной точки привязки.
    • Передискретизация: используется для выбора способа передискретизации объектов (если выбран параметр «С изменением масштаба объекта»).
  • Цвет (управление).
    • Формат цвета: используется для установки цветового режима RGB или «Серый» (8 или 16 бит), CMYK (8 бит) или Lab (16 бит).
    • Цветовой профиль: используется для установки цветовой гаммы для ранее выбранного формата цвета.
    • Чтобы применить другой цветовой профиль, используйте параметр Назначить. Будет применен новый профиль с неизменными значениями цветов и пикселей. Преобразовать: используется для преобразования каждого цвета старого профиля в новый, и, в результате, значения цветов и пикселей могут изменяться.
    • Прозрачный фон: если для этого параметра установлен флажок, фон страницы будет прозрачным.
  • Поля.
    • Учитывать поля: если для этого параметра установлен флажок, включено отображение полей страниц.
    • Левое поле / Верхнее поле / Правое поле/ Нижнее поле: используется для установки полей принтера, отображаемых в виде непечатаемых синих линий. Нажмите Использовать для поля настройки принтера для того, чтобы применить настройки принтера по умолчанию.
  • Вылет под обрез.
    • Левый вылет под обрез / Верхний вылет под обрез / Правый вылет под обрез / Нижний вылет под обрез: при профессиональной печати вылет под обрез используется для расширения области печати за край страницы, чтобы обеспечить возможность подрезания художественного изображения.
