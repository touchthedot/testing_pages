Трехмерный эффект

Трехмерный эффект используется для имитации текстурированной поверхности.

Параметры

На панели Эффекты представлены указанные ниже параметры.

  • Непрозрачность: используется для регулирования степени прозрачности эффекта.
  • Радиус: используется для регулирования степени проявления эффекта.
  • Эффекты слоя: обеспечивает доступ к диалоговому окну Эффекты слоя, где представлены расширенные параметры и элементы управления.

В диалоговом окне Эффекты слоя можно изменить приведенные далее расширенные параметры.

  • Глубина: используется для установки глубины эффекта. Этот параметр можно привязать к радиусу или устанавливать независимо от него.
  • Смягчение: используется для размытия теней и светлых участков изображения.
  • Профиль: используется для определения способа падения света и прорисовывания неровностей, углублений и выступов, затеняемых в процессе.
  • Удалить профиль: используется для возврата настроек профиля по умолчанию, когда свет падал равномерно.
  • Диффузия: используется для установки объема цветного рассеивающегося света, отражаемого от поверхности объекта при падении от источника света. При высоком значении параметра появится матовость, а при низком — глянец.
  • Отражение: используется для установки интенсивности цветного зеркального отблеска, отражаемого от поверхности объекта при падении от источника света. При высоком значении параметра появится глянец, а при низком — матовость.
  • Блеск: используется для установки объема цветного зеркального отблеска, отражаемого от поверхности объекта при падении от источника света. При высоком значении параметра появятся светлые участки с более жестким переходом от света к тени, а при низком — с менее жестким переходом.
  • Отражающий цвет: используется для установки цветного зеркального отблеска. Нажмите поле с цветом для выбора цвета на всплывающей панели.
  • Окружающий свет: используется для установки интенсивности окружающего света.
  • Цвет окружающего света: используется для установки цвета окружающего света. Нажмите поле с цветом для выбора цвета на всплывающей панели.
  • Источник света: используется для выбора источника света во всплывающем меню. Впоследствии пользователь может подкорректировать параметры ниже для выбранного источника света.
  • Добавить: используется для применения дополнительного источника света к эффекту.
  • Удалить: используется для удаления выбранного источника света. Если используется только один источник света, его удаление невозможно.
  • Направление: используется для обозначения положения источника света, тени или градиента. Для корректировки направления нажмите поле или перетащите указатель (и установите значение параметров Азимут и Возвышение автоматически).
  • Азимут: используется для определения направления источника света, тени или градиента.
  • Возвышение: используется для определения «высоты» источника света.
  • Цвет: используется для установки цвета выбранного источника света. Нажмите поле с цветом для выбора цвета на всплывающей панели.
  • Масштабировать относительно объекта: при выборе этого параметра (он установлен по умолчанию) в случае изменения размера объекта эффект масштабируется пропорционально объекту. Если этот параметр не выбран, при изменении размера объекта масштаб эффекта остается неизменным.
  • Непрозрачность заливки: используется для установки непрозрачности содержимого слоя, которая не влияет на примененные эффекты.
Чтобы применить (пользовательский) профиль, выполните указанные ниже действия.
  1. Нажмите пиктограмму профиля.
  2. Выберите на всплывающей панели один из стандартных профилей.
  3. (Необязательно) Нажмите точку на кривой, чтобы добавить узел, а затем перетащите узел кривой для придания ей формы.
  4. Установите флажок в поле «Линейная» для создания заостренной кривой или снимите флажок для создания сглаженной кривой.
  5. Для того чтобы применить профиль, нажмите в любом месте за пределами всплывающей панели.
