Изменение мазков кисти

Мазок кисти, примененный к странице, является полностью редактируемым, и его можно изменить (предварительно выделив) с помощью параметров на контекстной панели инструментов.

Чтобы задать расширенные параметры кисти, нажмите кнопку Изменить на панели «Кисти» и сохраните параметры мазка как новую предустановку кисти.

Параметры

В диалоговом окне «Кисть», в разделе Мазок доступны приведенные далее параметры.

  • Ширина кисти: используется для установки размера мазка.
  • Отклонение размера: используется для установки величины допустимого отклонения ширины кисти в процессе рисования мазка.
  • Отклонение непрозрачности: используется для установки величины допустимого отклонения прозрачности в процессе рисования мазка.
  • Во всплывающем меню контроллера размер и непрозрачность кисти меняются в соответствии с заданными параметрами: «Нажим» для графических планшетов и устройств с технологией Force Touch и «Скорость» или «Инверсия скорости» при использовании мышей или трекпадов. Во всплывающем меню контроллера размер и непрозрачность кисти меняются в соответствии с заданными параметрами: «Нажим» для графических планшетов и «Скорость» или «Инверсия скорости» при использовании мышей. Последние два параметра используются для имитации нажима относительно скорости движения мыши. Нажмите значок профиля смежного линейного градиента, чтобы выбрать стандартный профиль среди расположенных ниже пиктограмм или создать собственный, используя диаграмму линейного градиента. Перемещайте узлы круглой формы для изменения формы линейного градиента, добавляйте узлы, нажимая точки на линии, или выделяйте узлы для их удаления с помощью (упрощение линейного градиента). Установите флажок для параметра Линейная для того, чтобы узлы были соединены прямыми линиями. Если флажок не установлен (нелинейная), узлы будут соединены плавными кривыми.
  • Во всплывающем меню профиля можно задать поведение контроллера.
  • Основная часть: определяет способ рисования центральной части мазка по всей длине.
  • Углы: определяет способ прорисовки углов мазка.
  • Смещение головки: устанавливает точку перехода, в которой головка переходит в основную часть.
  • Смещение хвоста: устанавливает точку перехода, в которой основная часть переходит в хвост.
  • Чтобы установить длину основной части кисти, перетащите красные пунктирные линии на нижнюю часть области просмотра. Значения параметров «Смещение головки» и «Смещение хвоста» будут обновляться в соответствии с изменением положения.
  • Сбросить: используется для возврата всех параметров мазка к настройкам сохраненной ранее предустановки кисти.
  • Дублировать: используется для сохранения текущих параметров мазка в качестве предустановки кисти.
  • Закрыть: используется для выхода из диалогового окна и применения параметров мазка к выбранному объекту.
Для изменения мазков кисти выполните перечисленные далее действия.
  1. Выберите инструмент «Кисть» (если он еще не выбран).
  2. Удерживая , наведите указатель на мазок кисти и нажмите.
  3. На контекстной панели инструментов нажмите Дополнительно.
  4. Выберите нужные параметры в разделе Начертание.
  5. Нажмите Готово.

Можно рисовать на странице, пока открыто диалоговое окно «Кисть». К мазку будут применены текущие параметры, заданные в диалоговом окне. Таким образом, можно регулировать все параметры кисти в процессе рисования для повышения эффективности.

Дополнительную информацию о сохранении параметров в виде предустановок см. в разделе Создание пользовательских кистей.
