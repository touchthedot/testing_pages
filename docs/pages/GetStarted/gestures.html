Жесты

Продукты Affinity предназначены для использования в сочетании с мультисенсорными устройствами (мышь Magic Mouse, трекпад), в том числе устройствами с технологией распознавания силы нажатия Force Touch (трекпад Magic Trackpad 2 и компьютеры MacBook более поздних версий).

Для выполнения операций точечного наведения и нажатия, прокрутки и масштабирования с помощью трекпада можно использовать приведенные далее жесты, определенные по умолчанию (жесты могут быть изменены в установках устройства).

    • Касание для нажатия
      Создание выделения.
    • Поиск
      Отображение контекстного меню (только для устройств Force Touch).
    • Повторное нажатие
      Отображение контекстного меню.
    • Направление прокрутки
      Панорамирование представления документа.
    • Увеличение или уменьшение масштаба
      Увеличение или уменьшение масштаба посредством сведения или разведения двух пальцев.
    • Интеллектуальное масштабирование
      Переключение двойным касанием между текущим масштабом и масштабом 100 %.
    • Направление прокрутки (модификатор)
      Увеличение и уменьшение масштаба. Отслеживание движений пальцев.
    • Повернуть
      Поворот холста относительно текущего положения курсора.
