Корректировка «Раздельное тонирование»

Используется для придания оттенка и изменения цвета светлых участков и теней.

Параметры

Параметры, которые могут быть изменены, описаны ниже.

  • Оттенок: используется для регулирования оттенка цвета светлых участков и теней. Перетащите ползунок для того, чтобы установить значение.
  • Насыщенность: используется для регулирования интенсивности цветов светлых участков и теней. Для уменьшения интенсивности цвета перетащите ползунок влево, а вправо — для увеличения интенсивности.
  • Баланс: используется для контроля выделения корректируемых пикселей. Для выделения цвета светлых участков перетащите ползунок влево, а вправо — для выделения теней.

Первый набор ползунков «Оттенок/Насыщенность» предназначен для корректировки светлых участков, а второй — теней.
