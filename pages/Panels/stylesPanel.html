Панель «Стили»

Панель «Стили» позволяет применять предварительно разработанные стили путем их перетаскивания на объект.

О панели «Стили»

Стили состоят из эффектов, свойств мазков или текстовых атрибутов и их комбинаций. Панель «Стили» позволяет с легкостью применять эти свойства.

На панели «Стили» также можно сохранить любой стиль объекта для использования в будущем.

На панели отображается приведенное далее содержимое.

  • Всплывающее меню категории. Отображение пиктограмм стилей выбранной категории.
  • Пиктограммы стилей текущей категории.
  • Поиск: используется для фильтрации стилей на панели по названиям, соответствующим критериям поиска. Вводите текст непосредственно в текстовое поле.

Параметры

В меню «Установки панели» доступны приведенные далее параметры.

  • Добавить категорию стилей: используется для создания новой категории, в которую могут быть сохранены пользовательские стили.
  • Удалить категорию: используется для удаления текущей активной пользовательской категории.
  • Переименовать категорию: используется для переименования текущей активной пользовательской категории.
  • Импортировать категорию стилей: если стили были ранее экспортированы в файл, вы можете импортировать их с помощью этого параметра.
  • Экспортировать категорию стилей: используется для экспорта текущей категории и ее стилей в файл.
  • Добавить стиль из выделенного: если объект выделен, выберите этот параметр для того, чтобы сохранить стиль объекта в текущую категорию на панели «Стили».
