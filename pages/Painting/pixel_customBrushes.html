Создание пользовательских пиксельных кистей

Можно создавать пользовательские кисти на основе предустановки или с нуля на основе растровых изображений.

Чтобы создать пользовательский предустановленный мазок кисти с нуля, выполните указанные ниже действия.

Перейдите на панель Кисти, нажмите значок установок панели и выберите указанные ниже значения.

  • Создать кисть интенсивности: используется для создания мазка кисти на основании значений непрозрачности растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.
  • Создать круглую кисть: используется для создания мазка кисти округлой формы.
  • Создать квадратную кисть: используется для создания мазка кисти прямоугольной формы.
  • Создать кисть с изображением: используется для создания мазка кисти на основе значений цвета растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

Новая кисть добавляется в выбранную категорию с параметрами по умолчанию. Для изменения параметров по умолчанию выполните процедуру, представленную ниже в шаге 3.

Чтобы создать пользовательский предустановленный мазок кисти на основе предустановки, выполните указанные ниже действия.
  1. На панели Кисти выберите кисть и нажмите «Изменить кисть».
  2. В диалоговом окне нажмите Дублировать, а затем Закрыть.
  3. Выберите новую кисть в нижней части панели и нажмите «Изменить кисть».
  4. Выберите нужные параметры в диалоговом окне. Дополнительную информацию см. в разделе Изменение кистей.
  5. Нажмите Закрыть.

Дополнительные параметры, доступные пользователю после нажатия значка установок панели , позволяют создавать, переименовывать, удалять, импортировать и экспортировать категории кистей.

Переместите кисти в любую созданную категорию, нажав и выбрав имя категории в пункте меню Переместить кисть в категорию. Пользовательским категориям присваивается условное имя «Кисти», «Кисти 2», «Кисти 3» и т. д.
