Корректировка «Микширование каналов»

Откорректируйте цвет отдельных каналов, чтобы добиться эффектов, которых не так легко достичь с помощью других инструментов корректировки цвета.

Корректировки можно выполнять в любом цветовом режиме, вне зависимости от текущего цветового режима документа.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Выходной канал.
    • Выберите цветовой режим из первого всплывающего меню.
    • Укажите один цветовой канал, к которому будут применяться корректировки; можно выбрать в том числе альфа-канал слоя. Выберите вариант во втором всплывающем меню.
  • С помощью ползунков можно регулировать процент представления указанного цвета в выбранном выходном канале. Для снижения уровня определенного цвета перетащите ползунок влево, а для его повышения — вправо.
  • Смещение: контролирует общее влияние выбранного выходного канала на все изображение. Для снижения уровня влияния выходного канала перетащите ползунок влево, а для его повышения — вправо.

Изменение альфа-канала приведет к достижению матового эффекта или рирпроекции.
