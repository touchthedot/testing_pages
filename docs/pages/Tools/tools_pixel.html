Пиксель

С помощью инструмента «Пиксель» можно рисовать линии с попиксельным выравниванием и резкими краями. В этом его отличие от инструмента «Кисть рисования», в котором возможно небольшое колебание количества пикселей, связанное со сглаживанием.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Ширина: толщина линии в пикселях. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Непрозрачность: используется для определения степени прозрачности мазков кисти. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Поток: используется для определения скорости наложения эффектов кисти (1 % — очень медленно, 100 % — мгновенно). Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Жесткость: используется для определения степени жесткости краев линии, проведенной кистью. При уменьшении процентного значения жесткость кисти снижается. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Дополнительно: при нажатии этой кнопки отображается диалоговое окно Кисти, в котором можно установить дополнительные параметры кисти.
  • Принудительный нажим для регулирования размера: используется для регулирования размера мазка кисти посредством нажима в случае применения чувствительных к нажиму устройств. При выборе этого параметра настройки кисти по умолчанию отменяются.
  • Стабилизатор: стабилизация мазка с помощью либо стабилизатора «Веревка», либо стабилизатора «Окно». При использовании первого инструмента потяните конец мазка с помощью «веревки», сглаживающей мазок, но позволяющей обеспечить острые углы при увеличении значений параметра Длина (Радиус) путем изменения направления ослабленной веревки. При использовании второго инструмента мазок сглаживается благодаря усреднению отобранных положений ввода в рамках окна с настраиваемым размером.
  • Режим наложения: используется для изменения характера взаимодействия пикселей примененного фильтра с имеющимися пикселями слоя. Выберите этот вариант во всплывающем меню.
  • Защитить альфа-канал: при выборе этого параметра нельзя рисовать на прозрачных областях текущего слоя.
