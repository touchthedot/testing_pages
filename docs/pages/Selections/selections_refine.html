Уточнение краев пиксельного выделения

После создания выделения можно уточнять его края, чтобы обеспечить необходимую степень точности выделения.

Для очень точного выделения, например для выделения волос на цветном фоне, используйте корректирующую кисть в процессе уточнения, чтобы подправить мелкие детали.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Предпросмотр: определяет способ отображения выделения и страницы. Выберите этот вариант во всплывающем меню.
  • Заполнить края: при выборе этого параметра (он установлен по умолчанию) область выделения практически совпадает с краями изображения. Если этот параметр не выбран, выделение не совпадает с краями изображения. Это позволяет выполнить уточнение краев прямого выделения с более высокой точностью.
  • Ширина границы: используется для увеличения выделения путем регулирования ширины границы. Перетащите ползунок для того, чтобы установить значение.
  • Плавный: определяет кривизну угла выделения. Перетащите ползунок для того, чтобы установить значение.
  • Размытие краев: определяет мягкость (непрозрачность) перехода по краю выделения. Перетащите ползунок для того, чтобы установить значение.
  • Линейный градиент: используется для изменения размера выделения. Перетащите ползунок для того, чтобы установить значение. При выборе отрицательных значений выделение будет сжато, а при выборе положительных значений — увеличено.
  • Корректирующая кисть: используется для определения режима уточнения корректирующей кисти.
    • Подложка: используется для повторного анализа выделения и разделения деталей переднего плана и фона. Этот параметр прекрасно подходит для выделения линии волос по краю фотографий и т. д.
    • Передний план: добавление в выделение (при этом отображается больше переднего плана).
    • Фон: удаление из выделения (при этом отображается больше фона).
    • Размытие краев: смягчение края выделения в альфа-канале.
  • Ширина: используется для определения ширины кончика кисти. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Вывод: определяет, каким образом применяется выделение при закрытии диалогового окна. Выберите этот вариант во всплывающем меню.
Для уточнения краев пиксельного выделения выполните приведенные далее действия.
  1. Создав выделение, в меню Выделить выберите Уточнить края.
  2. Выберите нужные параметры в диалоговом окне.
  3. Если необходимо откорректировать края выделения путем рисования, перетаскивайте края в режиме предварительного просмотра.
  4. Нажмите Применить.
