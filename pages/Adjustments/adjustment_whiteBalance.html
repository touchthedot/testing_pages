Корректировка «Баланс белого»

Уберите нежелательные оттенки цветов или добавьте в проект больше теплых или холодных оттенков.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Баланс белого: используется для контроля «температуры» проекта. Перетащите ползунок влево для того, чтобы добавить холодные оттенки (добавляя синий), а вправо чтобы добавить теплые оттенки (добавляя желтый).
  • Оттенок: используется для придания пурпурного или зеленого оттенка проекту.
  • Пипетка: используется для выбора на странице образца цвета для установки белой точки, на основе которого будет рассчитан баланс белого. Можно выбирать образец с помощью приведенных далее способов.
    • Нажмите кнопку мыши, чтобы выбрать в качестве образца пиксель, находящийся под курсором.
    • Перетаскивайте курсор по всему изображению, чтобы выбрать все пиксели, находящиеся под курсором. Будет использован средний цвет этих пикселей.
    • Нажмите и перетащите курсор с нажатой кнопкой мыши, чтобы выделить прямоугольную область. Как и в предыдущем случае, в качестве образца будет выбран средний цвет этих пикселей.
