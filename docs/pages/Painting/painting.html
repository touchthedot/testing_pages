Мазки кисти рисования

Используйте инструмент «Векторная кисть» для того, чтобы применять мазки векторной кисти к своим композициям. Выбирайте кисти из широкого ассортимента инструментов, систематизированных по категориям. В каждой категории содержатся кисти с различными свойствами и характеристиками.

Набор параметров для кисти может быть сохранен в виде пользовательской предустановки на панели.

Рисование

В Affinity Designer есть потрясающий набор кистей, которые можно использовать с инструментами на основе кисти. Как и при использовании инструмента «Векторная кисть», для того чтобы применять произвольные мазки векторной кисти на странице, можно наносить мазки по контуру объектов и по периметру фотографий.

Для создания дополнительных фотореалистичных текстур и улучшения векторной композиции можно использовать инструмент «Кисть рисования» (например, кисти распыления) и другие инструменты на основе кисти.

Для наложения мазков кисти на страницу используются инструмент «Векторная кисть» (или «Кисть рисования»), панель «Кисти» и контекстная панель инструментов.

Для рисования мазками векторной кисти выполните перечисленные далее действия.
  1. На панели Инструменты выберите инструмент «Векторная кисть».
  2. На панели Кисти выберите пиктограмму кисти на ваш выбор.
  3. Отрегулируйте параметры контекстной панели инструментов.
  4. Перетащите инструмент по странице в направлении желаемого следования мазка кисти.
Чтобы сгладить мазки кисти по мере рисования, выполните приведенные далее действия.
  • На контекстной панели инструментов включите параметр Стабилизатор и выполните одно из перечисленных ниже действий.
    • Режим веревки: потяните конец мазка с помощью «веревки», сглаживающей мазок, но позволяющей обеспечить острые углы при увеличении значений параметра Длина (Радиус) путем изменения направления ослабленной веревки.
    • Режим окна: сглаживание мазка посредством усреднения положения мазка в Окне с настраиваемым размером.
Чтобы применить мазок векторной кисти к выделенной фигуре, выполните перечисленные далее действия.
  1. На панели Обводка выберите Текстурная линия и Ширина.
  2. На панели Кисти выберите пиктограмму кисти на ваш выбор.
  3. При необходимости нажмите Свойства, чтобы изменить кисть, используемую при выборе стиля «Текстурная линия», с помощью диалогового окна «Кисть».
Для изменения нарисованных мазков кисти выполните перечисленные далее действия.
  • Нажмите для перемещения отображаемых узлов и регулирования управляющих маркеров. На панели «Цвет» можно изменить цвет и значение непрозрачности.

Нажмите и удерживайте , а в это время редактируйте другие мазки кисти таким же образом, нажимая их.

Одновременно нажмите и и выполните перетаскивание на страницу. При перетаскивании влево или вправо размер кисти будет уменьшаться или увеличиваться соответственно.

Нажмите правую кнопку мыши, , а затем левую кнопку и выполните перетаскивание на страницу. При перетаскивании влево или вправо размер кисти будет уменьшаться или увеличиваться соответственно.
