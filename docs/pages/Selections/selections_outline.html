Создание контурных выделений

Функция Контур позволяет создать новое выделение по внешнему краю ранее созданного выделения.

Параметры

В соответствующих диалоговых окнах можно настроить указанные ниже параметры.

  • Радиус: используется для регулирования ширины области выделения.
  • Круглые края: при выборе этого параметра по мере увеличения значения радиуса края выделения будут становиться все более и более округлыми.
  • Выравнивание: используется для определения исходной точки, откуда берет начало область выделения. Выберите этот вариант во всплывающем меню.
Для создания контурного выделения выполните перечисленные далее действия.
  1. Создав выделение, в меню Выделить выберите Контур.
  2. Выберите нужные параметры в диалоговом окне.
  3. Нажмите Применить.
