Привязка

Функция привязки упрощает позиционирование новых и существующих объектов, так как ее принцип действия основан на том, что перемещенные объекты или объекты с измененным размером «притягиваются» к другим объектам или элементам документа.

Благодаря привязке происходит выравнивание изображений, мазков кисти, линий, фигур и областей выделений по отношению к соседним линиям сетки, направляющим, полям, монтажным областям, краям страницы или любой комбинации этих элементов. Кроме того, можно выполнить привязку к ограничительным рамкам и контуру объекта, а также ключевым точкам фигуры. Текст можно также привязать к базовой линии другого текстового блока (только первая строка текстовых областей), а художественные текстовые объекты могут быть привязаны к наивысшей точке ранее созданного художественного текста.

Для понимания механизма «магнитного притяжения» ориентируйтесь на цветные динамические направляющие и целевые узлы, которые отображаются во время привязки к объектам. Используемые цвета приведены далее.

  • Красная линия. Привязка к целевому объекту по горизонтали.
  • Зеленая линия. Привязка к целевому объекту по вертикали.
  • Узел желтого цвета. Привязка к ключевым точкам фигуры по горизонтали и вертикали.
  • Узел фиолетового цвета. Привязка узла кривой к ключевым точкам фигуры.
  • Синяя линия. Привязка объекта к третьей плоскости в случае использования треугольной проекционной сетки.
  • Оранжевая линия. Привязка к целевому объекту по горизонтали или вертикали в случае, если включено отображение проекционной сетки.

Динамические направляющие, которые отображаются во время выполнения привязки, служат визуальным ориентиром для выравнивания объектов. На динамических направляющих также отображаются метки, которые позволяют получить представление о расстоянии между объектами привязки (измеряется в единицах, установленных для документа).

При создании веб-графики целесообразно использовать параметр Принудительное выравнивание пикселей. Он может применяться в качестве дополнения к любым активным параметрам привязки.

Кандидаты привязки

Кандидаты привязки — это объекты на странице, к которым может применяться привязка. Используя приведенные далее параметры, можно задать, каким образом должно осуществляться создание кандидатов.

  • Список кандидатов: количество объектов, которые являются кандидатами привязки, не превышает установленное пользователем значение. В этом случае для назначения кандидата привязки следует создать новый объект или навести курсор мыши на уже существующий объект. Привязка может быть выполнена только к активным кандидатам.
  • Ближайшие слои: кандидатами привязки могут быть только объекты на текущем слое.
  • Ближайшие и дочерние слои: кандидатами привязки могут быть только объекты на текущем слое или любом из его дочерних слоев.
  • Все слои: неограниченное количество кандидатов привязки в документе.
Для включения или выключения привязки выполните перечисленные ниже действия.
  • На главной панели инструментов нажмите Привязка.
Для временного переопределения привязки выполните перечисленные ниже действия.
  • Нажмите во время размещения объекта. Если не нажать , привязка не выполнится.
Чтобы исключить объект из привязки, выполните приведенные далее действия.
  • На панели Слои нажмите и кнопку мыши на записи объекта и выберите Исключить из привязки.

На записи объекта отображается символ, указывающий на исключение из привязки.

Предустановки привязки

Пользователям доступен широкий выбор функциональных возможностей и параметров привязки. Для того чтобы легко и быстро выполнить привязку, можно воспользоваться одной из предустановок, каждая из которых включает набор рекомендуемых настроек привязки, определенных на основании вида работы (например, для проектирования интерфейса, рисования кривых и т. д.).

Для выбора предустановки привязки выполните перечисленные ниже действия.
  1. Нажмите стрелку параметра Привязка.
  2. Во всплывающем меню Предустановка выберите предустановку.
Для настройки параметров предустановки привязки выполните перечисленные ниже действия.
  1. Нажмите стрелку параметра Привязка.
  2. Выберите предустановку, на основании которой должен быть создан новый набор настроек для функции привязки.
  3. Устанавливайте или снимайте флажки отдельных параметров для переопределения текущих настроек предустановки.

Параметры вступят в силу немедленно.

Для сохранения пользовательской предустановки для использования в будущем выполните перечисленные ниже действия.
  1. Нажмите кнопку рядом со всплывающим меню Предустановки.
  2. Выберите Создать набор настроек.

Пользовательская предустановка вступит в силу немедленно.

Параметры привязки

Можно включать или отключать определенные параметры привязки в зависимости от ваших потребностей, стиля рисования и типа текущего проекта. Настройка параметров изначально выбранной предустановки будет осуществляться по ходу работы.

Во всплывающем диалоговом окне доступны приведенные ниже параметры.

  • Допуск экрана: используется для регулирования расстояния до объекта, которое должно выдерживаться для выполнения привязки.
  • Включить привязку: если флажок для этого параметра установлен, то привязка объектов будет выполняться на основании определенного критерия. Для изменения других настроек необходимо выбрать этот параметр.
  • Предустановки: используется для выбора предустановки, которая является набором настроек привязки и предназначена для выполнения определенного вида работы.
    • Макеты страницы: используется для целей печати композиций, когда привязка к установленным направляющим, полям или краям страницы играет важное значение.
    • Макеты страницы с объектами: этот параметр похож на описанный выше, однако предусматривает дополнительные возможности по выравниванию объекта по отношению к другому объекту.
    • Создание объектов: идеально подходит для простого выравнивания объекта по отношению к ограничительным рамкам другого объекта и их средним точкам, в том числе выравнивания некоторых фигур по ключевым точкам. Ключевые точки — это точки, которые автоматически добавляются на некоторые объекты (например, при выполнении округления угла фигуры ключевые точки расположены в начале и в конце сегмента, образующего закругленный угол).
    • Рисование кривой: эта настройка предназначена для работы с негеометрическими объектами (то есть рисования пером, карандашом или кистями).
    • Проектирование интерфейса: используется в работе над созданием элементов интерфейса и веб-графики в целях обеспечения пиксельной точности при привязке к неподвижным направляющим и сетке.
  • Кандидаты: используется для определения способа создания кандидатов. Выберите этот вариант во всплывающем меню.
    • Максимальный: используется для установки предельного количества активных кандидатов, если выбран параметр Список кандидатов (см. выше). Если предельное значение превышено, ранее созданные кандидаты будут заменены новыми в хронологическом порядке.
    • Показать кандидатов привязки: если флажок для этого параметра установлен, то будут выделены активные кандидаты привязки, то есть объекты, к которым можно привязать другие элементы путем предварительного выделения или наведения курсора мыши. Кандидаты будут отображаться с фиолетовым ореолом.
  • Привязать только к видимым объектам: если флажок для этого параметра установлен, то привязка будет выполняться только к видимым объектам.
  • Принудительное выравнивание пикселей: этот параметр похож на параметр панели инструментов (см. Принудительное выравнивание пикселей).
    • Переместить целыми пикселями: этот параметр похож на параметр панели инструментов (см. Принудительное выравнивание пикселей).
  • Привязать к сетке: если флажок для этого параметра установлен, то содержимое будет привязано к сетке линий (если в меню Вид включено ее отображение). Эта функция недоступна, если используется «Принудительное выравнивание пикселей».
  • Привязать к направляющим: если флажок для этого параметра установлен, то содержимое будет привязано к направляющим (если в меню Вид включено их отображение).
  • Привязать к ширине: если флажок для этого параметра установлен, то содержимое будет привязано к краю страницы (поля не учитываются).
    • Включить средние точки ширины: если флажок для этого параметра установлен, то содержимое будет привязано к вертикальному или горизонтальному центру страницы. Эта функция доступна, только если выбран указанный выше параметр.
  • Привязать к полю: если флажок для этого параметра установлен, то содержимое будет привязано к полям страницы (если в меню Вид включено их отображение).
    • Включить средние точки поля: если флажок для этого параметра установлен, то содержимое будет привязано к вертикальному или горизонтальному центру поля страницы. Эта функция доступна, только если выбран указанный выше параметр.
  • Привязать к ограничительным рамкам объекта: если флажок для этого параметра установлен, то выравнивание объектов может выполняться по ограничительным рамкам.
    • Включить средние точки ограничительной рамки: если флажок для этого параметра установлен, то объекты будут привязаны к вертикальному или горизонтальному центру целевого объекта. Эта функция доступна, только если выбран указанный выше параметр.
    • Привязать к промежуткам и размерам: если флажок для этого параметра установлен, то сопоставленные промежутки между кандидатами привязки и сопоставленные размеры по горизонтали и (или) вертикали обозначаются стрелками.
  • Привязать к ключевым точкам фигуры: если флажок для этого параметра установлен, то объекты могут быть выровнены по ключевым точкам на фигурах (например, по точкам в начале и в конце сегмента, образующего закругленный угол).
  • Привязать к контуру объекта: если флажок для этого параметра установлен, то объекты могут быть привязаны к вершинам других объектов, а не просто к ограничительной рамке или ключевым точкам. Вершины — это углы и узлы пересечения объекта, например лучи звезды, в пределах ограничительной рамки фигуры.

Во время привязки всегда используются единицы измерения, установленные на текущий момент в меню Файл > Настройка документа.

Получить доступ к описанным выше параметрам также можно с помощью диспетчера привязки в меню Вид.

Если вы работаете с документом, который содержит большое количество объектов или слоев, и используете параметр Все слои для кандидатов привязки, могут возникнуть проблемы с производительностью при перемещении объектов или изменении их размеров по причине слишком большого числа кандидатов привязки.
