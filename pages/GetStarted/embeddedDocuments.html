Встраивание документов

Функция встраивания документов позволяет добавлять любой документ Affinity (Designer, Photo и т. д.), Photoshop, Illustrator, Freehand, SVG, EPS или PDF в текущий документ без необходимости открывать каждый файл по очереди.

После добавления встроенных документов на страницу каждый документ можно будет редактировать, не выходя из текущего документа.

Для того чтобы встроить документ, выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Разместить.
  2. Во всплывающем диалоговом окне перейдите к файлу, выберите документ и нажмите кнопку Открыть.
  3. Выполните одно из перечисленных ниже действий.
    • Нажмите для того, чтобы поместить документ, сохранив его размер по умолчанию.
    • Перетаскивайте курсор по странице для того, чтобы установить размер и отрегулировать положение документа.

Встроенные документы представлены на панели «Слои», где отображаются первоначальное название документа и суффикс «Встроенный документ».

Если вы встраиваете файл Designer с несколькими монтажными областями, на контекстной панели инструментов документа появится параметр Монтажная область, так что вы сможете выбирать, какая монтажная область должна отображаться.

Редактирование встроенного документа
  1. Выполните одно из перечисленных ниже действий.
    • Два раза нажмите добавленный документ.
    • На контекстной панели инструментов выберите пункт Редактировать документ.

Изменения сохраняются в основном документе, не затрагивая исходный файл.

Возможности для редактирования встроенного документа определяются тем, каким образом исходный документ был сохранен в собственном приложении.

Для замены документа выполните перечисленные далее действия.
  1. Выберите документ.
  2. На контекстной панели инструментов выберите пункт Заменить документ.
