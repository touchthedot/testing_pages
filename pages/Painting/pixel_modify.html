Изменение пиксельных кистей

Можно менять настройки пиксельных кистей перед использованием их в работе над композицией. Базовые изменения можно сделать на контекстной панели инструментов, а более продвинутые — на панели Кисти.

Параметры

Доступны приведенные далее параметры.

Общие

  • Размер: используется для установки ширины мазка по умолчанию. Однако этот параметр может быть изменен для отдельных типов мазков кисти на контекстной панели инструментов и на панели «Обводка».
  • Накопление: используется для установки значения отклонения для непрозрачности или видимости мазка во время рисования.
  • Жесткость: используется для определения степени жесткости краев линии, проведенной кистью. При уменьшении процентного значения жесткость кисти снижается.
  • Интервал: используется для установки расстояния между точками наконечника кисти. При установке более низкого процентного значения происходит наложение точек наконечника, благодаря чему обеспечивается плавность мазка. При установке более высокого процентного значения расстояние между точками наконечника увеличивается, в результате чего возникает эффект распыления.
  • Поток: используется для установки скорости, с которой накладывается цвет при проведении мазка.
  • Форма: используется для установки диаметра наконечника кисти.
  • Поворот: используется для установки угла поворота наконечника кисти. Отлично подходит для кистей некруглой формы, используемых, например, при создании каллиграфических эффектов.
  • Режим наложения: используется для изменения характера взаимодействия примененных цветных пикселей с имеющимися пикселями слоя. Выберите этот вариант во всплывающем меню.
  • Влажные края: используется для установки значения по умолчанию для эффекта «мокрой» кисти. Выберите этот вариант во всплывающем меню. Установите флажок Пользовательский и примените предустановленный или пользовательский профиль. После этого немного изменится степень размытости мазка.
    При выборе этого параметра происходит скопление цвета вдоль контуров мазка, нанесенного пиксельной кистью, в результате чего возникает эффект акварельного рисунка.

Динамика

  • Параметры колебания определяют степень влияния выбранного контроллера кисти (нажим, скорость, поворот и т. д.) на описанные выше общие параметры. Выберите контроллер во всплывающем меню и нажмите значок профиля смежного линейного градиента для того, чтобы выбрать стандартный профиль среди расположенных ниже пиктограмм или создать собственный, используя диаграмму линейного градиента. Перемещайте узлы круглой формы для изменения формы линейного градиента, добавляйте узлы, нажимая точки на линии, или выделяйте узлы для их удаления с помощью (упрощение линейного градиента). Установите флажок для параметра Линейная для того, чтобы узлы были соединены прямыми линиями. Если флажок не установлен (нелинейная), узлы будут соединены плавными кривыми.
  • Распределение по оси X: используется для установки значения отклонения во время проведения мазка в горизонтальном направлении в соответствии с настройками предустановки.
  • Распределение по оси Y: используется для установки значения отклонения во время проведения мазка в вертикальном направлении в соответствии с настройками предустановки.

Параметры оттенка, насыщенности и изменения светимости влияют на цвет кисти, который устанавливается с помощью панели «Цвет». Аналогичным образом, параметр «Изменение плотности» влияет на уровень непрозрачности кисти, который также устанавливается на панели «Цвет».

Текстура

Управление наконечником

  • Наконечники кисти: используется для просмотра наконечников, которые в настоящее время назначены для текущей кисти.
  • Добавить: используется для добавления дополнительного наконечника в предустановку.
  • Удалить: используется для удаления выбранного наконечника из предустановки.

Управление базовой текстурой

  • Базовая текстура: используется для отображения базовой текстуры или узора текущей кисти.
  • Задать текстуру: используется для открытия диалогового окна, предназначенного для добавления базовой текстуры кисти (как изображения). Например, чтобы имитировать шероховатую поверхность бумаги или холста.
  • Удалить: используется для удаления базовой текстуры текущей кисти.
  • Инвертировать: используется для создания инвертированной текстуры.
  • Режим: используется для управления наложением базовой текстуры на отпечаток текущей кисти. Выберите этот вариант во всплывающем меню.
    • Нет: базовая текстура не применяется, используются только наконечники кисти.
    • Наконечник: используется для управления наложением цвета наконечников на базовую текстуру в зависимости от того, как меняются показатели потока и непрозрачности.
    • Окончательный: плотность базовой текстуры не меняется, как и показатели потока и непрозрачности.
  • Масштаб: используется для установки размера, при котором отображается текстура. При установке более низкого процентного значения текстура будет отображаться в большем размере. При установке более высокого процентного значения текстура будет отображаться в меньшем размере.

Изображения для базовых текстур должны быть в формате JPG или PNG. Могут использоваться любые изображения с приемлемыми размерами, поскольку базовая текстура может быть масштабирована (см. выше).

Дополнительные параметры

  • Сбросить: используется для возврата всех параметров мазка к настройкам сохраненной ранее предустановки кисти.
  • Дублировать: используется для сохранения текущих параметров мазка в качестве новой предустановки.
  • Закрыть: используется для выхода из диалогового окна и применения параметров мазка к выбранной предустановке.
Изменение параметров пиксельной кисти
  1. Выберите одно из перечисленных ниже действий.
    • Выберите кисть рисования и на контекстной панели инструментов нажмите Дополнительно.
    • На панели Кисти нажмите «Изменить кисть».
  2. Выберите нужные параметры в диалоговом окне.
  3. Нажмите Закрыть.

Дополнительную информацию о создании пользовательских предустановок для пиксельных кистей см. в разделе Создание пользовательских пиксельных кистей.
