Инструмент «Разместить изображение»

Инструмент «Разместить изображение» позволяет добавить изображение на страницу с сохранением размера по умолчанию или использованием пользовательских размеров.

Чтобы разместить изображение, выполните указанные ниже действия.
  1. Нажмите кнопку Инструмент «Разместить изображение».
  2. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.
  3. Выполните одно из перечисленных ниже действий.
    • Нажмите для того, чтобы разместить изображение, сохранив его отображаемый размер по умолчанию.
    • Перетаскивайте курсор по странице для того, чтобы установить размер и отрегулировать положение изображения.
