Рисование кривых и фигур

Векторные линии, кривые и фигуры можно легко создавать с помощью инструментов «Перо» или «Карандаш». Инструмент «Перо» имеет несколько режимов, которые меняют способ рисования контура.

Чтобы с точностью нарисовать кривые, используя инструмент «Перо», выполните указанные ниже действия.
  1. Выберите инструмент «Перо».
  2. На контекстной панели инструментов выберите один из режимов:
    • Режим «Перо». Нажмите левую кнопку мыши и, не отпуская ее, перетащите инструмент вдоль страницы для создания дублируемых узлов. Если изменить положение отображающихся управляющих маркеров в каждом узле, то при прокладывании узлов определяется форма следующего сегмента.
    • Интеллектуальный режим. Для прокладывания узлов необходимо несколько раз щелкнуть левой кнопкой мыши по странице. Наиболее подходящая кривая создается без необходимости регулировать управляющий маркер.
    • Режим «Многоугольник». Для прокладывания узлов необходимо несколько раз щелкнуть левой кнопкой мыши по странице. В результате создается контур с острыми узлами, состоящими из прямых сегментов.
    • Режим «Линия». Нажмите левую кнопку мыши и, не отпуская ее, перетащите инструмент вдоль страницы для создания простой прямой линии, состоящей из одного сегмента.
  3. Чтобы завершить рисование кривой, не замыкая ее, нажмите . Чтобы замкнуть кривую, щелкните начальный узел.

Если один раз нажать для замыкания кривой, а затем снова начать использовать инструмент «Перо», узлы предыдущей кривой будут временно отображаться меньшими по размеру, чтобы показать отсутствие соединения между предыдущей кривой и новой, которую вы собираетесь нарисовать.

Чтобы продлить существующую кривую, выполните указанные ниже действия.
  1. Поместите указатель пера в конечный узел контура, который необходимо продлить.
  2. Нажмите один раз для выбора узла.
  3. При необходимости продолжайте размещать новые узлы.
Чтобы нарисовать кривые от руки с помощью инструмента «Карандаш», выполните указанные ниже действия.
  • Перетащите инструмент по странице в направлении желаемого следования контура.
Чтобы замкнуть кривые для создания пользовательской фигуры, выполните указанные ниже действия.

Выполните одно из перечисленных ниже действий.

  • Выберите кривую с помощью инструмента Перо или Узел и нажмите Замкнуть кривую на контекстной панели инструментов.
  • При создании кривой с помощью инструмента «Перо» щелкните начальный узел, чтобы соединить его с конечным узлом и образовать фигуру.
