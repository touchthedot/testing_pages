Поля

Поля страницы позволяют создавать композиции с учетом требований печати.

Благодаря полям страницы можно получить представление о том, как будет выглядеть страница, распечатанная на настольном принтере. Элементы композиции, расположенные за пределами полей, могут быть не напечатаны.

Положение полей можно настроить вручную или с использованием параметров подключенного принтера. Они являются непечатаемыми и не видны на экспортированном документе.

Поля также можно использовать для выполнения привязки при создании или перемещении объектов.

Если вы собираетесь экспортировать композицию, а не сразу отправлять ее на печать, поля можно полностью игнорировать во время работы над проектом. Если это так, то отображение полей можно отключить.

Для изменения полей страницы выполните перечисленные далее действия.
  1. На контекстной панели инструментов по умолчанию или в меню Файл выберите Настройка документа.
  2. Во всплывающем диалоговом окне выберите вкладку Поля.
  3. Выберите вариант Учитывать поля.
  4. Выполните одно из перечисленных ниже действий.
    • Установите значения для параметров Левое поле, Правое поле, Верхнее поле и Нижнее поле.
    • Нажмите Использовать для поля настройки принтера для автоматической настройки полей в соответствии с параметрами настольного принтера.
Для того чтобы скрыть или отобразить поля, выполните перечисленные далее действия.
  • В меню Вид отмените выбор или выберите Показать поля. Когда поля видны, рядом с пунктом меню отображается флажок.
