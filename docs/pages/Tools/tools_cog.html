Зубчатое колесо

Инструмент для рисования фигур Зубчатое колесо является одним из наиболее универсальных инструментов в наборе. Он обеспечивает геометрическую точность и имеет ряд настраиваемых контрольных точек, которые позволяют создавать различные полезные фигуры, от шестеренок до солнечных лучей!

Настройка

В инструменте «Зубчатое колесо» есть несколько вариантов выбора формы и параметров на контекстной панели инструментов, с помощью которых можно устанавливать количество и размер зубов, размер канавки, внутренний радиус и размер отверстия.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Зубья: используется для выбора количества выступов на фигуре.
  • Внутренний радиус: используется для регулирования ширины зубьев относительно ширины фигуры.
  • Радиус отверстия: используется для регулирования ширины отверстия в центре фигуры.
  • Размер зуба: используется для регулирования ширины верхнего края зуба.
  • Размер канавки: используется для регулирования ширины нижнего края зуба.
  • Кривизна: используется для добавления изогнутого края к зубу.
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
