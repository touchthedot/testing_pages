Проверка орфографии

Встроенная функция проверки орфографии позволяет привести текст в идеальное состояние перед его печатью.

Существует несколько способов проверки орфографии, в том числе через диалоговое окно Проверка орфографии или посредством поиска и исправления орфографических ошибок непосредственно на странице.

Орфографические ошибки выделяются красным подчеркиванием.

Параметры

В диалоговом окне Проверка орфографии (средство проверки орфографии) доступны приведенные далее параметры.

  • Текущее слово: используется для выбора и ручного ввода варианта исправления.
  • Список предложений: используется для выбора слова из списка предложений для замены текущего слова.
  • Изменить: используется для замены слова с ошибкой текущим словом.
  • Найти далее: используется для отображения следующего слова с ошибкой.
  • Игнорировать: используется для того, чтобы оставить правописание текущего слова без изменений.
  • Добавить в словарь: используется для добавления текущего слова в словарь.
  • Определить: используется для поиска текущего слова в активном словаре приложения.
  • Предположить: используется для отображения списка вариантов замены текущего слова.
  • Проверить орфографию в: используется для определения области проверки орфографии. Выберите этот вариант во всплывающем меню.
Для проверки орфографии выполните перечисленные далее действия.

Откройте меню Текст и выберите один или несколько из приведенных далее параметров в подменю Проверка орфографии.

  • Параметры проверки орфографии: используется для запуска проверки орфографии.
  • Проверять орфографию: используется для отображения и выделения следующего слова с ошибкой.
  • Проверять орфографию при наборе текста: используется для выделения орфографических ошибок при вводе текста.
Для исправления слова с ошибкой на странице выполните перечисленные далее действия.
  • Выберите один из инструментов для работы с текстом и, удерживая , нажмите слово с ошибкой, а затем выберите вариант исправления во всплывающем меню.
Для того чтобы пропустить слово с ошибкой на странице, выполните перечисленные далее действия.
  1. Выделите слово с ошибкой.
  2. Откройте меню Текст и в подменю Проверка орфографии выберите Пропустить слово с ошибкой.
