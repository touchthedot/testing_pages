Выделяющая кисть

С помощью инструмента Выделяющая кисть можно выделить участок изображения посредством рисования.

Действие инструмента «Выделяющая кисть» подобно действию других кистей в режиме «Пиксельный». Можно добавлять участки к выделению или удалять их из него с помощью простого перетаскивания инструмента по изображению. Выделение будет увеличиваться или уменьшаться в зависимости от параметров кисти.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Режим: используется для установки способа обработки выделения по мере рисования кистью.
    • Добавить: используется для добавления областей к текущему выделению при проведении мазков кистью. Если выделение отсутствует, при рисовании будет создано новое выделение.
    • Вычесть: используется для удаления областей из текущего выделения при проведении мазков кистью.
  • Ширина: используется для определения размера кисти (мазка) в пикселях. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Привязать к краям: используется для расширения области выделения до краев объектов, поверх которых пользователь проводит мазок кистью.
  • Все слои: если этот параметр выключен (по умолчанию), выделение применяется только к текущему слою. Если параметр включен, выделение применяется ко всем слоям изображения.
  • Уточнить: при нажатии этой кнопки отображается диалоговое окно Уточнить выделение, в котором можно установить дополнительные параметры выделения.

ТАКЖЕ СМ.:

  • Создание пиксельных выделений
  • Выделение в виде области
