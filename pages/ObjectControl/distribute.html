Распределение объектов

Объекты на странице можно распределить равномерно или расположить с ровным интервалом с помощью команд панели «Упорядочить».

Распределение и расположение с интервалом

Распределение объектов предусматривает выбор равномерного расстояния между определенными точками на объектах.

Расположение объектов с интервалом обеспечивает расположение на равномерном расстоянии между краями каждого выделенного объекта.

Параметры

Можно настроить указанные ниже параметры на всплывающей панели Упорядочить. Распределение всегда выполняется в пределах границ выделения.

  • Интервал по горизонтали: используется для равномерного распределения выделенных объектов по горизонтали.
  • Интервал по вертикали: используется для равномерного распределения выделенных объектов по вертикали.
  • Автораспределение — если этот параметр выбран (по умолчанию), объекты распределяются равномерно в пределах границ выделения. Если этот параметр не выбран, можно задать определенное расстояние между объектами, сопряженное с параметром «Автораспределение».

Используйте другие параметры распределения, настроив панель инструментов (Вид > Настроить панель инструментов).

Чтобы распределить объекты или расположить их с интервалом, выполните указанные ниже действия.
  1. Выберите несколько объектов.
  2. Выполните одно из перечисленных ниже действий.
    • На панели инструментов нажмите Упорядочить, выберите функции на всплывающей панели и нажмите Готово.
    • В подменю Выравнивание меню Слой выберите параметр Пространство или Распределение.
