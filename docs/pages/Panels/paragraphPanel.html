Панель «Абзац»

Панель Абзац позволяет регулировать положение и размещение отдельных абзацев или целых блоков текста.

О панели «Абзац»

Панель «Абзац» обеспечивает ряд уникальных возможностей, описанных ниже.

  • Регулирование размера вертикального интервала между абзацами.
  • Задание отступов слева и справа, включая отступы только для первых строк.
  • Создание нескольких позиций табуляции на различных расстояниях, в результате чего можно устанавливать неравномерные интервалы при нажатии несколько раз.
  • Изменение характеристик выключенного текста, благодаря чему пользователь может управлять процессом вычисления интервала между буквами и словами для того, чтобы текст располагался на одном уровне вдоль левого и правого полей.

Далее приведены параметры, доступные на панели.

  • Выравнивание по левому краю: используется для того, чтобы установить выравнивание абзаца по левому полю.
  • Выравнивание по центру: используется для того, чтобы установить выравнивание абзаца таким образом, чтобы текст был равноудален от левого и правого полей.
  • Выравнивание по правому краю: используется для того, чтобы установить выравнивание абзаца по правому полю.
  • Выключка по левому краю: используется для того, чтобы установить выравнивание абзаца как по левому, так и по правому полю. Последняя строка абзаца выравнивается по левому краю.
  • Выключка по центру: этот параметр похож на параметр «Выключка по левому краю», однако последняя строка абзаца выравнивается по центру.
  • Выключка по правому краю: этот параметр похож на параметр «Выключка по левому краю», однако последняя строка абзаца выравнивается по правому краю.
  • Выключка по ширине: этот параметр похож на параметр «Выключка по левому краю», однако последняя строка абзаца выключается независимо от длины (иногда эту функцию называют «Принудительная выключка»).
  • Стиль текста: позволяет применять определенный стиль текста абзаца к выделенному тексту.
Интервал
  • Междустрочный интервал абзаца: используется для регулирования расстояния между базовыми линиями текста (вертикальный промежуток между линиями) в абзаце. Выберите этот вариант во всплывающем меню. Доступные параметры описаны ниже.
    • По умолчанию: используется для установки межстрочного интервала для шрифта по умолчанию (одинарный интервал).
    • Точно: используется для установки фиксированного интервала (другие атрибуты текста не учитываются для целей назначения корректного межстрочного интервала). Для регулирования этого параметра можно использовать предустановки во всплывающем меню.
    • % высоты: используется для установки интервала в зависимости от процентного значения относительно размера текста. Для регулирования этого параметра можно использовать предустановки во всплывающем меню.
    • Минимум: используется для установки минимального интервала (фактический межстрочный интервал может увеличиваться в зависимости от других атрибутов текста). Для регулирования этого параметра можно использовать предустановки во всплывающем меню.
    • Множитель: используется для регулирования межстрочного интервала в качестве интервала по умолчанию. Для регулирования этого параметра можно использовать предустановки во всплывающем меню.
  • Интервал перед абзацем: используется для регулирования вертикального промежутка перед абзацем.
  • Интервал после абзаца: используется для регулирования вертикального промежутка после абзаца.
  • Отступ первой строки: используется для регулирования отступа первой строки абзаца.
  • Отступ слева: используется для регулирования левого отступа всего абзаца (за исключением первой строки).
  • Отступ справа: используется для регулирования правого отступа всего абзаца.
Позиции табуляции
  • Позиции табуляции по умолчанию: используется для регулирования стандартного горизонтального интервала, который появляется перед символом после вставки табуляции ().
  • Добавить новую позицию табуляции: используется для добавления дополнительной позиции табуляции.
  • Удалить выделенную позицию табуляции: используется для удаления выделенной позиции табуляции.
  • Выравнивание позиции табуляции: используется для выравнивания выделенной позиции табуляции.
  • Заполнитель табуляции: используется для установки отображаемых символов, которые служат указателями выделенных позиций табуляции.

Дополнительную информацию см. в разделе Форматирование абзацев.

Выключка
  • Минимальный интервал между словами: используется для установки минимального разрешенного промежутка.
  • Требуемый интервал между словами: используется для установки предпочтительного промежутка между словами.
  • Максимальный интервал между словами: используется для установки максимального разрешенного промежутка.
  • Минимальный межбуквенный интервал: используется для установки минимального разрешенного трекинга.
  • Требуемый межбуквенный интервал: используется для установки предпочтительного трекинга (интервала между буквами) в пределах слов, когда для абзаца установлено полное выравнивание.
  • Максимальный межбуквенный интервал: используется для установки максимального разрешенного трекинга.
