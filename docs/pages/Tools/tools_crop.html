Инструмент «Кадрирование векторного объекта»

С помощью инструмента «Кадрирование векторного объекта» можно удалить части выделенного объекта (или сгруппированных объектов) неразрушающим способом и с применением конкретных пропорций. Можно также кадрировать изображения.

Для применения кадрирования выполните перечисленные далее действия.

Выберите инструмент «Кадрирование векторного объекта» и выполните любое из перечисленных ниже действий.

  • Перетащите угловой маркер, чтобы применить кадрирование от выделенного угла без сохранения соотношения сторон. Кадрирование от угла позволит сохранить пропорции изображения.
  • Перетащите курсор от маркера края, чтобы применить кадрирование по вертикали или горизонтали от выбранного края.
Для перемещения объектов в области кадрирования выполните перечисленные далее действия.
  • Выберите инструмент «Кадрирование векторного объекта» и перетащите объект в пределах области кадрирования.

Клавиши-модификаторы

При использовании инструмента «Кадрирование векторного объекта» можно использовать перечисленные далее клавиши-модификаторы во время применения кадрирования.

  • : используется для сохранения пропорций объекта при кадрировании. Этот модификатор можно использовать при кадрировании изображения от маркера края.
  • : используется для временного отключения привязки.
  • : используется для применения кадрирования из центра.
  • : используется для поворота области кадрирования вокруг противоположного маркера.
  • С помощью нажатия правой кнопки мыши можно поворачивать область кадрирования вокруг противоположного маркера.

Эти модификаторы можно использовать в комбинации друг с другом.
