Корректировка «Светофильтр»

Используется для имитации применения светофильтра с целью тонирования цвета изображения.

Параметры

Параметры, которые могут быть изменены, описаны ниже.

  • Цвет фильтра: используется для выбора цвета / оттенка светофильтра.
  • Оптическая плотность: используется для корректировки плотности (или интенсивности) светофильтра. Таким образом, можно регулировать количество оттенка, накладываемое на изображение.
  • Сохранить светимость: если флажок установлен, изменение цвета не влияет на канал свечения изображения. Если флажок не установлен, повышение оптической плотности, как правило, приводит к снижению светимости на изображении.
