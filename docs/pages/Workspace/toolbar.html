Панель инструментов

Панель инструментов содержит часто используемые инструменты и функции для быстрого доступа. Параметры находятся в логических группах для повышения удобства использования.

Параметры

На панели инструментов доступны приведенные ниже параметры.

Настройки по умолчанию

  • Синхронизация настроек по умолчанию с выделенным объектом: настройки по умолчанию обновляются до настроек текущего выбранного объекта.
  • Отменить изменения настроек по умолчанию: синхронизированные настройки по умолчанию возвращаются к сохраненным значениям по умолчанию. Если в текущий момент выделен объект, его атрибуты будут возвращены к параметрам по умолчанию.

Режим просмотра

  • Пиксельный режим просмотра: включение пиксельного режима для отображения векторных композиций в виде отдельных пикселей.
  • Пиксельный режим просмотра (Retina): включение пиксельного режима (Retina) для отображения векторных композиций в виде отдельных пикселей.
  • Контурный режим рисования: включение контурного режима для отображения композиции исключительно в виде контуров.

Порядок

  • Переместить на задний план: используется для перемещения выделенных объектов под слой. Либо используется для перемещения выделенных слоев в нижнюю часть панели «Слои».
  • На 1 позицию назад: используется для перемещения выделенных объектов на одну позицию вниз в слое. Либо используется для перемещения выделенных слоев на одну позицию вниз на панели «Слои».
  • На 1 позицию вперед: используется для перемещения выделенных объектов на одну позицию вверх в слое. Либо используется для перемещения выделенных слоев на одну позицию вверх на панели «Слои».
  • Переместить на передний план: используется для перемещения выделенных объектов в верхнюю часть слоя. Либо используется для перемещения выделенных слоев в верхнюю часть панели «Слои».

Описанные выше параметры также доступны в подменю Упорядочить в меню Слой.

Трансформация

  • Отразить по горизонтали: используется для отражения выделенных объектов слева направо.
  • Отразить по вертикали: используется для отражения выделенных объектов сверху вниз.
  • Повернуть по часовой стрелке: используется для поворота выделенных объектов вправо с шагом 90°.
  • Повернуть против часовой стрелки: используется для поворота выделенных объектов влево с шагом 90°.

Описанные выше параметры также доступны в подменю Трансформация в меню Слой.

Выравнивание

  • Упорядочить: используется для отображения всплывающего диалогового окна, которое позволяет выровнять и распределить выделенные объекты.

Привязка

  • Принудительное выравнивание пикселей: выбор этого параметра (по умолчанию) обеспечивает полное вытягивание пикселей при создании, перемещении или изменении векторного содержимого. Если этот параметр выключен, векторное содержимое может охватывать частичные пиксели.
  • Привязка: при выборе этого параметра выделенные объекты будут подчиняться правилам привязки, которые определяются текущими параметрами привязки. Если этот параметр выключен (по умолчанию), привязка отключена.
  • Параметры привязки: настройка параметров во всплывающем меню.

Операции

  • Добавить: используется для создания нового объекта, получаемого путем объединения выделенных объектов.
  • Вычесть: используется для удаления частей самого нижнего объекта на основании перекрытия с объектами, расположенными выше в пределах выделения. Все остальные выделенные объекты удаляются.
  • Пересечение: используется для создания нового объекта, получаемого из перекрывающихся частей выбранных объектов.
  • Разделить: используется для разделения участков объектов на отдельные объекты; объект, полученный из области пересечения, сохраняет цвет верхнего объекта.
  • Объединить: используется для объединения выделенных объектов в один составной объект, на котором в прозрачной области перекрываются участки с заливкой.

На новых объектах сохраняются свойства нижнего объекта в выделении.

Описанные выше параметры также доступны в подменю Геометрия в меню Слой.

Место вставки

  • Вставить позади выделенного: если этот параметр включен, новые объекты добавляются ниже выделенных объектов.
  • Вставить сверху слоя: если этот параметр включен, новые объекты добавляются сверху слоя.
  • Вставить внутри выделенного: если этот параметр включен, новые объекты добавляются внутри текущего выделения.

Если все целевые параметры отключены (по умолчанию), новые объекты добавляются поверх текущих выбранных объектов.
