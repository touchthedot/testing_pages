Корректировка OpenColorIO

Корректировка OpenColorIO предназначена для упрощения рабочих процессов OpenColorIO по управлению цветами посредством преобразования исходного цветового пространства в целевое. Дополнительную информацию см. в разделе Использование OpenColorIO.

Параметры

Параметры, которые могут быть изменены, описаны ниже.

  • Исходное цветовое пространство: позволяет установить параметры исходного цветового пространства для преобразования.
  • Целевое цветовое пространство: позволяет установить параметры целевого цветового пространства.

Для применения этой корректировки требуется действительная конфигурация OpenColorIO. Инструкции по настройке конфигурации см. в разделе Использование OpenColorIO.
