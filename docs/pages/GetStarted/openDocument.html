Открытие документов и изображений

Можно открывать не только ранее созданные документы, но и растровые изображения (изображения с камеры, мобильного устройства или цифровые художественные изображения) и векторные изображения (SVG). Также можно открывать целый ряд популярных форматов необработанных изображений с камеры, включая форматы, которые используются в профессиональных графических редакторах.

Ключевой особенностью продуктов серии Affinity является возможность легко открывать любые файлы Affinity в любом приложении Affinity вне зависимости от того, в какой программе Affinity они были созданы. Тем самым вы сможете расширить свои возможности, используя во время создания композиций функции других приложений Affinity и не беспокоясь по поводу совместимости файлов при их сохранении.

Документы Affinity Designer сохраняются с расширением файла .afdesign.

Для открытия документа или изображения выполните перечисленные далее действия.
  1. Выполните одно из перечисленных ниже действий.
    • В меню Файл выберите пункт Открыть.
    • Дважды нажмите кнопкой мыши в любом месте пустого представления. (Доступно только в режиме «Обычный», если нет других открытых документов.)
  2. Выберите нужный файл и нажмите Открыть.
Для открытия недавно использованного документа выполните перечисленные далее действия.
  • В меню Файл выберите пункт Открыть недавние, а затем выберите файл в подменю.
Для того чтобы открыть изображение как новый документ (через Finder), выполните перечисленные далее действия.
  • Откройте Finder и перетащите файл в область другой страницы вашего рабочего пространства.

При открытии растровых изображений с помощью указанных выше методов будет создан новый документ с одним слоем под названием «Фон». Этот слой будет зафиксирован по умолчанию. Название файла документа и название изображения будут совпадать.

При открытии файла цветовое пространство сохраняется по умолчанию, но вы можете преобразовать его в рабочее цветовое пространство по умолчанию, выбрав Affinity Designer > Установки (параметр «Цвет») и функцию Преобразовать открытые файлы. Текущий цветовой профиль документа отображается в верхней части рабочего пространства слева.

При открытии файла цветовое пространство сохраняется по умолчанию, но вы можете преобразовать его в рабочее цветовое пространство по умолчанию, выбрав Правка > Установки (параметр «Цвет») и функцию Преобразовать открытые файлы. Текущий цветовой профиль документа отображается в верхней части рабочего пространства слева.
Для добавления изображения на слой (через Finder) выполните перечисленные далее действия.
  1. На панели Слои выделите целевой слой.
  2. Перетащите изображение из окна Finder на страницу. Изображение будет добавлено к выделенному слою.

Существующие документы можно встраивать в слои, используя описанную выше процедуру добавления изображений.

Для открытия файлов Adobe PDF выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Открыть.
  2. В диалоговом окне «Параметры PDF» можно выполнять описанные далее действия.
    • Импорт отдельной или всех страниц.
    • Установка разрешения для PDF-содержимого.
    • Определение того, должны ли быть сохранены широкие возможности редактирования за счет снижения точности воспроизведения композиции.
    • Определение того, следует ли рассматривать отдельные строки текста как единое текстовое поле, благодаря чему обеспечивается более корректное положение текста.
    • Замена отсутствующего шрифта соответствующим замещающим шрифтом.

Каждая страница многостраничных PDF-файлов будет размещаться на отдельном слое. Необходимо включить отображение слоя скрытой страницы для просмотра его содержимого.
