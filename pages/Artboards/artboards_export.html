Экспорт монтажных областей

Можно экспортировать монтажные области в виде отдельных изображений или сразу все в одном документе в виде одного изображения.

Если выбрать монтажную область непосредственно перед экспортом, она будет автоматически выбрана в диалоговом окне «Экспорт». Но вне зависимости от выбора все монтажные области в активном документе могут быть экспортированы в любое время из диалогового окна «Экспорт».

Если выбрать экспорт всего документа, все монтажные области в документе будут включены в экспортируемое изображение. Размеры итогового изображения будут зависеть от наименьшего возможного размера, относящегося к размеру и расположению каждой монтажной области на панели вставки. Другими словами, чем меньше размеры монтажных областей и чем ближе они расположены друг к другу, тем меньше будут общие размеры изображения.

Дополнительную информацию об экспорте всего документа см. в теме Экспорт.

Режим «Экспорт»

Для отдельных монтажных областей автоматически создаются фрагменты (области экспорта), благодаря чему пользователи могут получить доступ ко всем функциям, имеющимся в режиме «Экспорт».

Для экспорта отдельной монтажной области выполните указанные ниже действия.
  1. (Необязательно.) Выберите монтажную область.
  2. В меню Файл выберите пункт Экспортировать.
  3. Выберите нужные параметры в диалоговом окне.
  4. Во всплывающем меню Область выберите имя монтажной области. (Это действие осуществлять не нужно, если указанный выше необязательный этап был выполнен.)
  5. Нажмите Экспортировать.
Для экспорта монтажной области с помощью режима «Экспорт» выполните указанные ниже действия.
  1. (Необязательно.) На панели Параметры экспорта установите необходимые параметры экспорта для фрагментов монтажной области, выбранных на панели «Фрагменты».
  2. На панели Фрагменты выберите необходимые параметры для каждого фрагмента монтажной области, развернув запись фрагмента монтажной области. Можно выполнять приведенные ниже действия.
    • Добавить другие форматы файлов для экспорта.
    • Добавить для формата экспорта фрагмента дополнительные варианты масштабирования размеров (например, трехкратное масштабирование) или абсолютные размеры (100w).
    • Переименовать файлы для экспорта.
    • Экспортировать определенные размеры и форматы на выбор.
  3. Нажмите Экспортировать фрагменты (n) в нижней части панели, чтобы экспортировать все отмеченные элементы.
  4. Перейдите к папке для хранения экспортируемых изображений, выберите ее и нажмите кнопку Экспортировать.
  5. (Необязательно.) На панели Фрагменты выберите Непрерывно, если необходимо автоматически выполнять повторный экспорт областей экспорта при последующем изменении документа. Этот параметр доступен только после выбора элемента Экспортировать фрагменты (n) выше.
