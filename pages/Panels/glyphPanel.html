Панель «Средство просмотра глифов»

В контексте типографики глиф — это графическое представление символа в гарнитуре шрифта, например кириллицей или латиницей.

О панели «Средство просмотра глифов»

Панель «Средство просмотра глифов» доступна в режимах «Рисование» и «Пиксельный». Для ее включения перейдите в меню Вид > Студия.

Чтобы добавить глиф, выполните перечисленные ниже действия.
  1. В документе выберите инструмент Художественный текст или Текстовая область, создайте текстовый слой и приступайте к редактированию.
  2. Чтобы добавить глиф в текст, дважды щелкните по нему на панели Средство просмотра глифов.

Далее приведены параметры, доступные на панели.

  • Шрифт: выбранный тип шрифта (перечислены все действительные установленные шрифты).
  • Формат: используется для установки формата шрифта, например Полужирный.
  • Гарнитура шрифта: используется для установки выбранной гарнитуры шрифта для отображения глифов, например Основная латиница, Арабское письмо, Расширенное греческое письмо.

Во всплывающем подменю доступны указанные ниже дополнительные параметры.

  • Размер: используется для выбора размера отображения глифов. Это повышает эффективность использования панели и не влияет на выходной размер глифа.
  • Кодировка: используется для переключения между кодировками Глифы и Юникод.
  • Очистить последние использованные: используемые глифы добавляются в нижней части панели для повторного использования. Этот параметр используется для очистки этого списка.
