Типы стилей текста

В программе Affinity Designer существует три типа стиля текста, которые используются для различных целей.

О типах стилей текста

Не существует ограничений на то, какие текстовые атрибуты и свойства могут быть применены к текстовому стилю, независимо от его типа. Вместо этого тип определяет главное назначение стиля текста.

Существует три типа стиля текста.

  • Абзац: используется для определения свойств абзацев.
  • Символ: используется для определения свойств отдельных глифов (или слов, строк и предложений внутри абзаца).
  • Группа: используется для определения свойств группы стилей текста.

Стили текста «Абзац» и «Символ»

Хотя стилям текста присваивается тип, программа Affinity Designer позволяет гибко применять стили текста абзаца и символа. Например, можно применить стили абзаца к выбранным глифам, как будто они являются стилями символов, что не повлияет на стиль, примененный к абзацу. Чтобы активировать эту функцию, необходимо задать стиль текста Показать на обеих панелях.

Если за стилем текста закреплена комбинация клавиш, такая комбинация применяет стиль, используя его преобладающий характер. Например, если выделить часть текста и использовать комбинацию клавиш, чтобы применить стиль текста абзаца, изменится абзац с выделенным текстом, а не только выделенный текст.

Стиль текста «Группа»

Стиль текста Группа нельзя применить непосредственно к тексту на странице. Вместо этого он определяет принципы построения других стилей текста и позволяет быстро создать иерархию стилей текста. Стили теста «Группа» отображаются только на панели «Стили текста».

Стиль «Группа» можно рассматривать как родительский или основной стиль текста. Все прочие стили, основанные на стиле «Группа», считаются дочерними или подчиненными стилями.

Можно создать подчиненный стиль текста в пределах группы, выбрав стиль текста «Группа» в качестве стиля, на основе которого он был создан.

Сложность иерархии зависит исключительно от предпочтений пользователя — подчиненный стиль может быть основным стилем с собственными подчиненными стилями, группа стилей может быть введена в качестве подчиненной к основной группе стилей.

Для редактирования типа стиля текста выполните указанные ниже действия.
  1. На панели Стили текста нажмите меню параметров перечисленных стилей и выберите Правка.
  2. В разделе Стиль диалогового окна выберите один из пунктов в списке Тип.
  3. Нажмите ОК.
Чтобы обеспечить гибкость использования стиля текста, выполните указанные ниже действия.
  1. На панели Стили текста нажмите меню параметров перечисленных стилей и выберите Правка.
  2. В разделе Стиль диалогового окна выберите параметр Показать на обеих панелях.
  3. Нажмите ОК.
Для создания иерархии стилей текста выполните перечисленные далее действия.
  1. На панели Стили текста выберите Создать стиль группы.
  2. Выберите нужные параметры в диалоговом окне.
  3. Нажмите ОК.
  4. На панели Стили текста нажмите на меню параметров созданного стиля группы и выберите Создать стиль на основе.
  5. Выберите нужные параметры в диалоговом окне.
  6. Нажмите ОК.
