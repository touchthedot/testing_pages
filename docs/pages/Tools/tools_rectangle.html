Инструмент «Прямоугольник»

Прямоугольники часто используются при рисовании. Возможно, это одна из наиболее часто используемых фигур в художественных произведениях и в дизайне. Инструмент «Прямоугольник» позволяет быстро и легко рисовать прямоугольники и квадраты.

Настройка

Параметр Один радиус на контекстной панели инструментов позволяет применить один из нескольких типов угла к любому из углов. После этого можно перетягивать красный маркер, чтобы регулировать радиус угла.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Один радиус: если этот параметр включен (по умолчанию), пользователь может изменять все четыре угла одновременно посредством выбора одного значения радиуса. Если этот параметр выключен, можно установить значение радиуса для каждого угла отдельно, а также изменить тип угла.
  • Абсолютные размеры: по умолчанию радиус угла указывается в виде процентного соотношения для объекта и масштабируется при изменении размера объекта. Если этот параметр включен, можно указать радиус угла в единицах измерения. При изменении размера объекта радиус угла не масштабируется вместе с объектом, а остается неизменным.
  • Угол: используется для выбора стиля и радиуса углов прямоугольника. Можно выбрать следующие параметры: «Скругленный» (угловой), «Прямой» (скошенный), «Вогнутый» (дугообразный) или «Вырезанный».
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделенное при перетягивании: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
