Поворот холста

Вместо поворачивания документа можно повернуть холст; это похоже на физический поворот рабочей области. Благодаря этому можно рассмотреть документ с другого ракурса, не проводя никаких операций с его макетом.

О повороте холста

Возможность поворачивать холст может пригодиться при выполнении различных операций:

  • при работе кистью и проработке мелких деталей;
  • при разработке продукта (например, при создании макетов коробки или упаковки продукта, содержимое на которых располагается под углом 90° или 180°);
  • при проектировании с использованием планшетов (рисование под углом).
Для поворачивания холста выполните указанные ниже действия.
  • В меню Вид выберите пункт Повернуть влево или Повернуть вправо. Холст будет поворачиваться с шагом 15°.

В том же меню выберите пункт Сбросить поворот, чтобы возвратить холст в исходное положение.

Для большей эффективности назначьте собственные пользовательские комбинации клавиш для операций поворота и сброса ориентации холста. Дополнительную информацию см. в теме Настраивание комбинаций клавиш.

Пользователи, применяющие трекпад, могут повернуть холст с помощью жеста.
Для выключения или включения поворота холста (при использовании трекпадов) выполните перечисленные далее действия.
  • Выберите Affinity Designer > Установки (параметр «Инструменты»), а затем установите или снимите флажок в поле Включить поворот холста с помощью трекпада.
