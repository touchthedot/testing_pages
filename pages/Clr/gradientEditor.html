Редактирование градиента

С помощью инструмента «Заливка» можно заполнить объект простым цветным градиентом. Созданный контур заливки можно редактировать непосредственно на объекте (добавить больше, чем два цвета по контуру градиента, настроить непрозрачность, изменить положение добавленных цветов или управлять цветовыми переходами).

При необходимости можно точно определить контур заливки, воспользовавшись образцом цвета на контекстной панели инструмента.

Можно также применить градиент к растровым слоям, корректировочным слоям и маскам слоя, воспользовавшись инструментом «Заливка».

Параметры градиента

Можно изменить приведенные далее параметры на контекстной панели инструментов (образце цвета).

  • Тип: используется для определения типа градиента (линейный, эллиптический и т. д.) во всплывающем меню.
  • Положение: используется для контроля положения цветового узла в градиенте слева (0 %) направо (100 %), где 50 % — это центральная точка.
  • Средняя точка: используется для настройки распределения цветов между выбранными цветовыми узлами слева и справа.
  • Цвет: нажмите на образец цвета, чтобы отобразилась всплывающая панель, на которой можно изменить цвет выбранного узла (а также значение шума).
  • Непрозрачность: используется для определения степени прозрачности узла. Значение параметра 100 % соответствует полной непрозрачности, а 0 % — полной прозрачности.
  • Вставить: используется для добавления нового узла между выбранным и находящимся справа. Этому узлу назначается цвет его нового положения.
  • Копировать: используется для дублирования выбранного узла. Дубликат узла помещается между выбранным и находящимся справа.
  • Удалить: используется для удаления выбранного узла из градиента. Вследствие удаления конечного узла контур укорачивается до соседнего узла.
  • Обратить: используется для инвертирования градиента, то есть создания зеркального отражения.
Чтобы изменить градиент (непосредственно на объекте), выполните приведенные далее действия.

Выберите инструмент «Заливка», нажмите на объект с примененной градиентной заливкой и выполните одно из перечисленных далее действий.

  • Нажмите на точку на контуре градиента, чтобы добавить узел.
  • Чтобы выбрать узел, нажмите на него. Размер выбранных узлов больше размеров остальных.
  • Перетащите узел, чтобы изменить его положение в контуре градиента. Положение конечных узлов можно изменять (посредством перетаскивания) для увеличения или уменьшения длины градиента. Перетаскиванием можно также изменять его угол.
  • Перетащите маркер средней точки для настройки распределения цветов между двумя цветовыми узлами.
  • На панели «Цвет» примените цвет (или значение непрозрачности или шума) к выбранному узлу.
  • Чтобы удалить выбранный узел, нажмите .
Чтобы изменить градиент (на контекстной панели инструментов), выполните приведенные далее действия.
  1. Выберите инструмент «Заливка» и нажмите на объект.
  2. На контекстной панели инструментов выберите образец цвета.
  3. Выберите параметр Градиент, с помощью которого можно изменить градиент, воспользовавшись настройками, приведенными выше.
