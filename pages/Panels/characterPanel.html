Панель «Символ»

Панель Символ позволяет применять локальное форматирование к отдельным буквам, словам, предложениям, абзацам, а также целым блокам текста.

О панели «Символ»

Панель «Символ» обеспечивает ряд уникальных возможностей, описанных ниже.

  • Отображение списка гарнитур шрифтов в пределах отдельных коллекций (включая отсутствующие шрифты).
  • Применение двойного подчеркивания или перечеркивания, а также управление их цветом независимо от цвета текста.
  • Установка кернинга и наклона.
  • Применение горизонтального и вертикального масштабирования.

Далее приведены параметры, доступные на панели.

  • Коллекция: используется для установки гарнитур шрифтов, которые доступны на панели. Выберите этот вариант во всплывающем меню.
  • Гарнитура шрифта: используется для установки гарнитуры шрифта для выделенного текста. Выберите этот вариант во всплывающем меню.
  • Размер: используется для определения размера знаков текста.
  • Шрифт: используется для определения того, какой стиль гарнитуры шрифта применяется к выделенному тексту.
  • Цвет шрифта: используется для установки цвета текста. Выберите этот вариант на всплывающей панели.
  • Цвет фона: используется для установки цвета области позади выделенного текста (таким образом создается эффект выделения). Выберите этот вариант на всплывающей панели.
  • Стиль текста: позволяет применять определенный стиль к выделенному тексту.
Подчеркивание и зачеркивание
  • Подчеркивание: используется для установки режима Без подчеркивания или же выбора одинарного или двойного подчеркивания.
  • Цвет подчеркивания: используется для установки цвета подчеркивания текста. Если установлено значение «Нет», цвет подчеркивания будет соответствовать цвету шрифта. Выберите этот вариант на всплывающей панели.
  • Зачеркивание: используется для установки режима Без зачеркивания или же выбора одинарного или двойного зачеркивания.
  • Цвет зачеркивания: используется для установки цвета зачеркивания текста. Если установлено значение «Нет», цвет зачеркивания будет соответствовать цвету шрифта. Выберите этот вариант на всплывающей панели.
Расположение и трансформация
  • Кернинг: используется для изменения кернинга (расстояния) между символами. Авто: автоматическое использование межсимвольного интервала, установленного по умолчанию. Для увеличения кернинга следует указывать положительные значения, а для его уменьшения — отрицательные.
  • Трекинг: используется для изменения интервалов между символами.
  • Базовая линия: используется для определения положения символов в тексте относительно нижней линии. При увеличении значения базовая линия опускается, а при уменьшении — поднимается.
  • Частота междустрочного интервала: используется для локального применения к выделенному тексту определенного междустрочного интервала для увеличения расстояния между строками по отношению к расстоянию между абзацами.
  • Наклон: используется для управления степенью наклона текста. Положительные значения обеспечивают наклон текста влево, а отрицательные значения — вправо.
  • Масштаб по горизонтали: используется для увеличения ширины символов и интервалов по отношению к размеру точки.
  • Масштаб по вертикали: используется для увеличения символов по отношению к размеру точки.
Типографика
  • Стандартные лигатуры: используется для применения к выделенному тексту любых доступных лигатур для гарнитур шрифта.
  • Контекстные заменители: используется для применения любых альтернативных гарнитур шрифтов, доступных для глифов в зависимости от их относительной позиции в слове или положения относительно соседних глифов.
  • Порядковые номера: используется для автоматического преобразования букв, являющихся окончаниями порядковых числительных, в верхний индекс.
  • Дроби: используется для динамического преобразования дробей в один глиф.
  • Надстрочный: используется для преобразования текстовых символов в верхний индекс (например, символы устанавливаются выше соседних символов, а размер шрифта уменьшается).
  • Подстрочный: используется для преобразования текстовых символов в нижний индекс (например, символы устанавливаются ниже соседних символов, а размер шрифта уменьшается).
  • Все прописные: используется для преобразования всех букв выделенного текста в прописные (или малые прописные, если выбран вариант Малые прописные).
  • Малые прописные: используется для отображения строчных букв в виде миниатюрных заглавных букв.
Язык
  • Выберите язык (для проверки орфографии и справочных целей). Выберите этот вариант во всплывающем меню.
