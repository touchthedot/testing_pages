Многоугольник

Инструмент для рисования фигур «Многоугольник» позволяет создавать полезные многогранные геометрические фигуры.

Настройка

В инструменте «Многоугольник» есть несколько вариантов выбора формы и параметров на контекстной панели инструментов, с помощью которых можно устанавливать количество сторон и боковую кривизну.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Стороны: используется для задания количества сторон (и углов) многоугольника.
  • Кривая: используется для задания выпуклых или вогнутых сторон многоугольника. Кривизна в процентах определяется количеством сторон. Для прямых сторон установите значение 0 %.
  • Точки сглаживания: используются для применения сглаживания к вершинам многоугольника, в результате чего образуется непрерывная кривая. Этот параметр применяется, только если кривизна сторон составляет более 0 %.
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Равносторонние треугольники

Для создания правильного равностороннего треугольника (в котором все три стороны имеют одинаковую длину) используйте инструмент Многоугольник. Убедитесь, что в процессе рисования фигура ограничена с помощью , а затем на контекстной панели инструментов установите значение «3» для количества сторон.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
