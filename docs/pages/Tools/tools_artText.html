Инструмент «Художественный текст»

Инструмент «Художественный текст» позволяет добавлять художественный текст в проект.

Само название «художественный текст» подсказывает, что этот инструмент подходит для визуального выделения отдельных слов (или фраз) или декоративного оформления шрифта.

Инструменты «Художественный текст» и «Текстовая область» сгруппированы на панели Инструменты. Чтобы получить доступ к альтернативному инструменту нажмите треугольник в правом нижнем углу отображаемого значка.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Шрифт: используется для установки шрифта текста. Выберите этот вариант во всплывающем меню.
  • Задать размер текста: для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Задать формат текста: нажмите, чтобы вывести всплывающее меню с дополнительными форматами для выбранного шрифта, или выберите параметры Жирный, Курсив и (или) Подчеркнутый.
  • Стиль символов: используется для применения стиля символов к выделенному тексту. Выберите этот вариант во всплывающем меню.
  • Стиль абзаца: используется для применения стиля абзаца к выделенному тексту или абзацу, в котором в настоящий момент размещен курсор. Выберите этот вариант во всплывающем меню.
  • Задать выравнивание абзаца: используется для выбора параметров Выравнивание по левому краю, Выравнивание по центру, Выравнивание по правому краю или Выключка (с дополнительными параметрами всплывающего меню).
  • Задать междустрочный интервал (расстояние между базовыми линиями текста): введите значение непосредственно в текстовое поле или выберите во всплывающем меню.
  • Базовая линия*: используется для установки положения текста по отношению к контуру. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Обратить контур для текста*: меняет местами две крайние точки контура таким образом, что текст меняет направление.
  • Символ: предоставляет доступ к панели Символ, содержащей расширенные параметры текста.
  • Абзац: обеспечивает доступ к панели Абзац, где расположены расширенные параметры абзаца.

*Доступно только при выбранном параметре Текст по контуру.

Во время выбора междустрочного интервала при указании единицы измерения (например, «мм» или «дюймы») будет определено точное расстояние, тогда как при указании процентного значения будет определен % высоты, основанный на размере шрифта текста.
