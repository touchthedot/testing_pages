Панель инструментов «Режимы»

На панели инструментов «Режимы» представлены режимы, которые можно выбрать в настоящий момент в Affinity Designer.

Параметры

На панели инструментов доступны приведенные далее параметры.

  • Режим «Рисование»: используется для переключения в специализированную среду векторного проектирования и векторной графики.
  • Режим «Пиксельный»: используется для переключения в специализированную среду для коррекции растровых изображений.
  • Режим «Экспорт»: используется для переключения в специализированную среду экспорта.

Активный режим можно также переключить с помощью расположенного сверху меню Affinity Designer.

Активный режим можно также переключить с помощью расположенного сверху меню Правка.
