Инструменты выделения в виде области

Инструменты выделения в виде области позволяют различными способами применять выделения к изображению.

Можно воспользоваться инструментами выделения в виде областей различных форм, включая прямоугольную область, эллиптическую область, область в виде столбца и область в виде строки. Чтобы применить любое выделение в виде области, просто выберите область нужной формы и перетащите ее на изображение.

Также можно использовать инструмент «Выделение произвольной формы», чтобы определить область выделения в виде нарисованной карандашом линии, многоугольника с прямыми сторонами или магнитных узлов с привязкой.

Параметры

Для каждого инструмента используется собственный набор параметров, который может включать некоторые или все указанные ниже параметры, настраиваемые на контекстной панели инструментов.

  • Режим: используется для включения режима Создать, Добавить, Вычесть и Пересечение.
  • Уточнить: при нажатии этой кнопки отображается диалоговое окно Уточнить выделение, в котором можно установить дополнительные параметры выделения.
  • Размытие краев: используется для уменьшения резкости краев выделения путем частичного выделения пикселей по краям.
  • Сглаживание: если этот параметр выключен (по умолчанию), пиксели по краю выделения остаются непрозрачными. Если этот параметр включен, края выделения сглаживаются путем применения прозрачности к пикселям по краям.
  • Ширина: используется для определения ширины элемента Область в виде столбца или Область в виде строки.

О режимах выделения

На контекстной панели инструментов доступно четыре режима выделения.

  • Создать: используется для отмены текущего выделения и создания нового.
  • Добавить: используется для добавления областей к текущему выделению. Если выделение отсутствует, будет создано новое выделение.
  • Вычесть: используется для удаления областей из текущего выделения.
  • Пересечение: используется для создания новой области выделения из перекрытия между нарисованной фигурой и текущим выделением.

Инструмент «Выделение произвольной формы»

Инструмент «Выделение произвольной формы» позволяет выбрать один из трех типов выделения, перечисленных ниже.

  • Произвольная форма: используется для создания выделения, которое в точности повторяет движения курсора.
  • Многоугольник: используется для создания выделения на основе соединенных прямых линий, где каждое нажатие кнопки определяет изменение направления.
  • Магнит: используется для создания выделения путем создания автоматических узлов с привязкой к отчетливым краям по мере перемещения курсора. Нажмите для определения положений пользовательских узлов.

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать для создания выделений при работе с инструментом «Выделение произвольной формы».

  • : используется для временного включения элемента Многоугольник во время работы с элементом Произвольная форма.
  • : используется для временного переключения между элементами Многоугольник и Магнит.
  • Во время работы с элементами Многоугольник и Магнит перетаскивание инструмента временно активирует элемент Произвольная форма.
