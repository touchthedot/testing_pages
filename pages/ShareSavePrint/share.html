Общий доступ

С помощью параметров общего доступа можно упростить передачу документов клиентам, друзьям и родственникам.

Если параметр общего доступа включен, страница документа экспортируется в виде файла JPG и становится доступной для соответствующего приложения или веб-сайта.

Параметры

Доступны приведенные далее параметры общего доступа.

  • Почта: изображение размещается в основной части сообщения электронной почты и готово для отправки соответствующим контактам.
  • Сообщения: изображение размещается в новом сообщении и готово для отправки друзьям.
  • AirDrop: изображение становится доступным для передачи с помощью технологии AirDrop.
  • Flickr: изображение загружается в пользовательскую учетную запись Flickr для передачи подписчикам.
  • Twitter: изображение загружается в пользовательскую учетную запись Twitter для передачи подписчикам.
  • Facebook: изображение загружается в пользовательскую учетную запись Facebook для публикации.
  • Добавить в Photo: изображение импортируется в приложение Apple Photo.
  • Добавить в Aperture: изображение импортируется в программу Aperture (если установлена).
  • Задать фоновое изображение: изображение используется в виде фонового изображения.
Чтобы поделиться проектом, выполните перечисленные далее действия.
