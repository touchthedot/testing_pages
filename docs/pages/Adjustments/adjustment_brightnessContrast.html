Корректировка яркости и контрастности

Меняйте значения параметров теней и светлых участков и управляйте общим тональным диапазоном изображений.

Вы также можете регулировать тональный диапазон изображений с помощью корректировок Уровни и Кривые.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Яркость: используется для регулирования уровня светлоты или затемнения изображения. Для уменьшения яркости перетащите ползунок влево, а вправо — для ее увеличения.
  • Контрастность: используется для регулирования тонального диапазона изображения. Для уменьшения контрастности темных и светлых областей перетащите ползунок влево, а вправо — для ее увеличения.
  • Линейный: по умолчанию эта корректировка позволяет предотвратить отсечение теней и светлых участков путем изменения уровня светлоты пикселей по сравнению с первоначальным значением. При выборе этого параметра для изменения пикселей используются абсолютные значения, в результате чего может произойти отсечение.
