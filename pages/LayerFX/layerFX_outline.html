Контур

Эффект «Контур» используется для добавления цветного контура к объектам.

Параметры

На панели Эффекты представлены указанные ниже параметры.

  • Цвет: используется для установки цвета эффекта. Нажмите поле с цветом для выбора цвета на всплывающей панели.
  • Непрозрачность: используется для регулирования степени прозрачности эффекта.
  • Радиус: используется для регулирования степени проявления эффекта.
  • Эффекты слоя: обеспечивает доступ к диалоговому окну Эффекты слоя, где представлены расширенные параметры и элементы управления.

В диалоговом окне Эффекты слоя можно изменить приведенные далее расширенные параметры.

  • Режим наложения: используется для определения того, как эффект должен влиять на содержимое, расположенное ниже текущего слоя.
  • Выравнивание: используется для выбора положения контура. Выберите этот вариант во всплывающем меню.
  • Стиль заливки: используется для выбора применяемого к контуру типа заливки.
  • Масштабировать относительно объекта: при выборе этого параметра (он установлен по умолчанию) в случае изменения размера объекта эффект масштабируется пропорционально объекту. Если этот параметр не выбран, при изменении размера объекта масштаб эффекта остается неизменным.
  • Непрозрачность заливки: используется для установки непрозрачности содержимого слоя, которая не влияет на примененные эффекты.

В параметре Стиль заливки можно выбрать тип По контуру. При выборе типа «По контуру» градиентная заливка применяется от внутреннего края контура к внешнему по всей ширине обводки.
