Инструмент «Перемещение»

С помощью инструмента «Перемещение» можно выделять объекты на странице. Инструмент «Перемещение» позволяет перемещать, поворачивать выделенные объекты и изменять их размер.

Используйте клавишу Shift одновременно с инструментом, чтобы ограничить объекты по вертикали, горизонтали или диагонали (45°).

Клавиша быстрого доступа к инструменту : V.

Параметры

В отсутствие выделенных объектов на контекстной панели инструментов доступны приведенные далее параметры.

  • Настройка документа: используется для отображения всплывающей панели, которая позволяет настраивать параметры документа.
  • Установки: используется для перехода к диалоговому окну «Установки» и выбора расширенных параметров приложения.

Если выделено несколько объектов, на контекстной панели инструментов доступны приведенные далее параметры.

  • Выровнять по горизонтали: используется для выравнивания объектов по левому или правому краям или по центру.
  • Выровнять по вертикали: используется для выравнивания объектов по верхнему или нижнему краям или по середине.
  • Группировать: используется для преобразования выделенных объектов в группу, чтобы упростить их выделение и изменение.*

Параметры на контекстной панели инструментов обновляются в зависимости от выделенного объекта. На панели могут быть представлены указанные ниже параметры.

  • Заливка: нажмите на образец цвета для отображения всплывающей панели и обновите цвет заливки.
  • Обводка: нажмите на образец цвета для отображения всплывающей панели и обновите цвет обводки.
  • Стиль линии и тип кисти: нажмите для отображения всплывающей панели и установите ширину и стиль линии или тип кисти.
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.
  • Разгруппировать: используется для разделения выделенной группы на отдельные объекты для целенаправленного редактирования.*
  • Закрепить дочерние: если установлен этот флажок, дочерние объекты выделенного разгруппированного слоя невозможно изменить при трансформации выделенного объекта. Отлично подходит для сохранения размера, положения и пропорций обрезанного изображения при изменении размеров родительского объекта.

*Вложенные группы можно создавать, когда существующая группа является частью группируемого выделения. Если эти вложенные группы разгруппировать, ранее сгруппированные объекты остаются в пределах своей исходной группы. Чтобы разделить вложенную группу на отдельные объекты (удаляя все предыдущие группы), в меню Слой выберите пункт Разгруппировать все.
