Инструмент «Карандаш»

С помощью инструмента «Карандаш» можно создавать нарисованные от руки изображения, проводя произвольные линии переменной ширины. Изображение будет иметь вид рисунка на бумаге.

По мере рисования автоматически создаются узлы, которые впоследствии можно отредактировать с помощью инструмента «Узел».

Линии переменной ширины можно регулировать либо посредством скорости (чаще всего может пригодиться при рисовании с помощью мыши), либо посредством нажима (используется при рисовании с помощью чувствительного к нажиму графического планшета).

Регулирование внешнего вида инструмента

Стили линий применяются с помощью панели Обводка.

К линиям, нарисованным с помощью инструмента «Карандаш», можно применить мазки кисти. Для этого нужно просто выделить нарисованную линию и выбрать дизайн на панели Кисти.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры кисти.

  • Начертание: цвет используемой кисти. Выберите необходимый образец цвета из сплошных или выбранных цветов, цветных градиентов, предустановленных или пользовательских образцов цвета.
  • Ширина: толщина линии в пикселях. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Контроллер: используется для регулирования степени чувствительности кисти к силе нажима (параметр «Нажим») или скорости движения мыши (параметр «Скорость» или «Инверсия скорости»). Чтобы не использовать ни один из параметров, выберите значение «Нет».
  • Стабилизатор: стабилизация мазка с помощью либо стабилизатора «Веревка», либо стабилизатора «Окно». При использовании первого инструмента потяните конец мазка с помощью «веревки», сглаживающей мазок, но позволяющей обеспечить острые углы при увеличении значений параметра Длина (Радиус) путем изменения направления ослабленной веревки. При использовании второго инструмента мазок сглаживается благодаря усреднению отобранных положений ввода в рамках окна с настраиваемым размером.
  • Использовать заливку: если этот параметр включен, вогнутый участок линии заполняется по мере рисования текущим цветом заливки (установленным на панели «Цвет»).
