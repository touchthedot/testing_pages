Цветовые модели

Для создания отображаемых цветов экран использует различное количество света. В физическом мире для создания цветов на странице используются чернила. Цвет сохраняется путем перевода цветов в числа. Цветовая модель используется для описания используемой числовой системы.

Поскольку разные устройства отличаются различными возможностями отображения цвета, используется цветовое пространство для определения гаммы (доступного диапазона) цвета. Работая в цветовом пространстве, соответствующем предназначенному устройству вывода, можно обеспечить надлежащее отображение цветов.

В Affinity Designer можно воспользоваться преимуществами цельного рабочего процесса по управлению цветами CMYK или Lab, выбрав эти режимы при создании документа.

О цветовых моделях

В разных цветовых моделях цвета представляются в виде чисел разными способами. При работе в Affinity Designer можно выбрать одну из четырех цветовых моделей.

Модель RGB

Модель RGB — это аддитивная цветовая модель. Основные цвета света (красный, зеленый, синий) в разных степенях смешиваются и образовывают другие цвета спектра.

Модель CMYK

Модель CMYK — это субтрактивная модель. Для создания каждого цвета смешиваются бирюзовый, пурпурный и желтый. Четвертый цвет чернила, черный, также используется для дополнительного контроля. Он может быть использован как отдельно для создания чистого черного, так и в сочетании с другими чернилами для создания насыщенного черного.

Способ внедрения цветовой модели зависит от выбранного цветового пространства. Для этого необходимо выбрать цветовой профиль.

Lab

Цветовая модель Lab представляет собой теоретический диапазон воспринимаемых человеческим зрением цветов с помощью трех каналов: канал светлоты (L) и два канала противоположных значений «красный — зеленый» (a) и «желтый — синий» (b). Она может пригодиться при творческой работе, в частности, благодаря тому, что значения светлоты можно изменять без воздействия на оттенок или насыщенность.

Чтобы выбрать цветовую модель нового документа, выполните указанные ниже действия.
  • При создании нового документа выберите вариант во всплывающем меню Формат цвета.

Формат цвета документа — это комбинация значений цветовой модели и установленной глубины цвета (8 или 16).

Чтобы в любое время изменить цветовую модель документа, выполните указанные ниже действия.
  1. В меню Файл выберите пункт Настройка документа.

  2. Во всплывающем меню Формат цвета выберите любую из доступных моделей, как описано выше.

    Affinity Designer преобразовывает каждый цвет старого формата в новый, в результате чего значения цветов и пикселей могут изменяться.
