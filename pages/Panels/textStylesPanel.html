Панель «Стили текста»

С помощью панели «Стили текста» можно получить доступ ко всем стилям текста в документе; пользователи могут применять, изменять, удалять имеющиеся стили или создавать новые.

О панели «Стили текста»

На панели «Стили текста» отображается информация о том, какие стили применены к текущему выделению, а также предоставляется возможность применить стили абзацев и символов в месте размещения курсора или к выделенному тексту.

На панели также имеются параметры для создания новых стилей абзацев и символов, переопределения существующих стилей или их удаления.

Кроме того, на панели имеются параметры для установки определенных стилей текста в виде стилей по умолчанию, а также их импорта из других проектов.

Дополнительную информацию об использовании стилей текста см. в теме Использование стилей текста.

Установки отображения

В установках панели существует несколько параметров, с помощью которых можно изменить способ отображения стилей на панели.

  • Показать иерархию: используется для вложенного представления стилей под стилями, на основе которых они были созданы.
  • Показать образцы: если этот параметр выбран (по умолчанию), оформление стиля будет являться зеркальным отображением того, что пользователи будут видеть на странице. Если этот параметр не выбран, все стили будут отображаться в виде обычного текста.
  • Сортировать по типу: если этот параметр выбран (по умолчанию), сначала в списке будут отображаться стили абзацев, а затем — стили символов. Если этот параметр не выбран, стили будут отображаться в списке исключительно в алфавитном порядке.

Параметры

На панели доступны приведенные далее параметры.

  • Текущее форматирование: используется для отображения форматирования, примененного к выделенному тексту или в текущем месте размещения курсора. Нажмите расположенную слева стрелку, чтобы отобразить информацию о выходе за границы (если необходимо).
  • Сбросить форматирование: используется для удаления операций перезаписи и локального форматирования из выделенного текста и обеспечения точного соответствия текущему форматированию.
  • Тип: используется для определения типа стиля текста (абзац, символ или группа).
  • Стиль текста: используется для отображения имен стилей текстов, которые в настоящий момент используются в документе.
  • Комбинация клавиш: используется для отображения комбинации клавиш, необходимой для применения стиля текста. Комбинацию клавиш необходимо сначала применить к стилю текста.
  • Меню «Параметры»: используется для предоставления доступа ко всплывающему меню, где расположены параметры создания и правки.
  • Создать стиль абзаца: используется для вывода диалогового окна Изменить стиль текста, которое применяется для создания нового стиля абзаца.
  • Создать стиль символов: используется для вывода диалогового окна Изменить стиль текста, которое применяется для создания нового стиля символов.
  • Создать стиль группы: используется для вывода диалогового окна Изменить стиль текста, которое применяется для создания нового стиля группы.
  • Обновить стиль абзаца: используется для переопределения текущего стиля абзаца с целью обеспечения соответствия локальному форматированию выделенного текста.
  • Обновить стиль символов: используется для переопределения текущего стиля символов с целью обеспечения соответствия локальному форматированию выделенного текста.
  • Удалить стиль: используется для удаления выделенного стиля абзаца или символов из документа, включая любые его копии.
