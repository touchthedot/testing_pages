Создание пиксельных выделений

Можно создавать пиксельные выделения для целенаправленного редактирования конкретных пиксельных областей. Границы выделения определяются в зависимости от включения или исключения отдельных пикселей.

Можно создать пиксельное выделение с помощью инструмента Выделяющая кисть или инструментов выделения в виде области либо расположить его на одном слое. Выделения, созданные с помощью инструментов выделения в виде области, выравниваются по отдельным пикселям.

Как только выделение будет создано, его можно инвертировать, чтобы исключить все включенные пиксели и наоборот.

Чтобы создать пиксельное выделение посредством рисования, выполните приведенные далее действия.
  1. На панели Инструменты выберите инструмент «Выделяющая кисть».
  2. Отрегулируйте параметры на контекстной панели инструментов.
  3. Перетащите инструмент по странице.

  • : используется для автоматического добавления областей к текущему выделению.
  • : используется для автоматического удаления областей из текущего выделения.
  • Перетащите инструмент с нажатыми левой и правой кнопками для автоматического добавления областей к текущему выделению.
Чтобы создать пиксельное выделение с помощью инструмента выделения в виде области, выполните перечисленные далее действия.
  1. На панели Инструменты выберите инструмент Прямоугольная область, Эллиптическая область, Область в виде строки или Область в виде столбца.
  2. Отрегулируйте параметры на контекстной панели инструментов.
  3. Перетащите инструмент по странице.

Далее перечислены клавиши-модификаторы, которые можно использовать для создания выделений при работе с инструментами выделения в виде области.

  • : используется для сохранения пропорций области.
  • : используется для автоматического добавления областей к текущему выделению.
  • : используется для автоматического удаления областей из текущего выделения.
  • Перетащите инструмент с нажатыми левой и правой кнопками для автоматического добавления областей к текущему выделению.
Чтобы нарисовать пиксельное выделение, выполните приведенные далее действия.
  1. На панели Инструменты выберите инструмент «Выделение произвольной формы».
  2. На контекстной панели инструментов выберите тип выделения.
  3. Выполните одно из перечисленных ниже действий.
    • После включения элемента Произвольная форма перетащите инструмент по странице, чтобы нарисовать края выделения, а затем отпустите кнопку мыши, чтобы замкнуть выделение.
    • После включения элемента Многоугольник щелкните по странице для определения начала выделения, а затем выполняйте последующие щелчки мышью для определения каждого изменения в направлении.
    • После включения элемента Магнит щелкните по странице для определения начала выделения, а затем щелкните повторно для определения положения пользовательского узла. Автоматические узлы будут размещаться вдоль отчетливых краев изображения по мере перемещения курсора.
  4. Дважды щелкните мышью для замыкания выделения (только элементы Многоугольник и Магнит).

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать для создания выделений при работе с инструментом «Выделение произвольной формы».

  • : используется для временного включения элемента Многоугольник во время работы с элементом Произвольная форма.
  • : используется для временного переключения между элементами Многоугольник и Магнит.
  • Во время работы с элементами Многоугольник и Магнит перетаскивание инструмента временно активирует элемент Произвольная форма.
Для создания пиксельного выделения из слоя выполните перечисленные далее действия.
  • На панели Слои щелкните по пиктограмме выделенного слоя, удерживая .
Для удаления пиксельного выделения выполните перечисленные далее действия.
  • В меню Выделить выберите параметр Отменить выделение.
Чтобы инвертировать выделение в виде области, выполните приведенные далее действия.
  • Создав выделение, в меню Выделить выберите Инвертировать пиксельное выделение.
