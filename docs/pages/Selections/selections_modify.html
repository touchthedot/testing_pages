Изменение пиксельных выделений

Созданное пиксельное выделение можно изменить несколькими способами.

Можно изменить пиксельное выделение посредством его увеличения или сжатия, применения размытия к краям и (или) сглаживания изгиба. Эти параметры можно выбрать в меню Выделить: Увеличить / сжать, Размытие краев и Сглаженность краев соответственно.

Параметры

В соответствующих диалоговых окнах можно настроить указанные ниже параметры.

  • Радиус: используется для регулирования степени проявления эффекта.
    • При выборе отрицательных значений для параметра Увеличить / сжать выделение будет сжиматься, а при выборе положительных значений — увеличиваться.
  • Круглые края: если этот параметр выключен (по умолчанию), при изменении размеров сохраняется оригинальная форма выделения. Если этот параметр включен, при увеличении значений радиуса форма выделения будет становиться более округлой.

Размытие краев можно также применить к выделению с помощью контекстной панели инструментов.

В дополнение к рассмотренным в этом разделе изменениям можно также уточнить края выделения. Этот параметр доступен на контекстной панели инструментов и в меню Выделить.
