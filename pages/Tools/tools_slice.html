Инструмент «Фрагмент»

Инструмент «Фрагмент» позволяет создавать и редактировать нарисованные области для экспорта, называемые фрагментами. Таким образом, пользователи могут выбирать части композиции на странице, которые будут экспортированы в виде отдельных графических изображений.

Этот инструмент используется не только для работы с нарисованными фрагментами: с его помощью можно также преобразовать выделенные объекты и слои во фрагменты с автоматическим определением размера.

Доступен только в режиме «Экспорт».

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Вернуть автоматический размер: используется для сброса измененных размеров фрагмента, созданного из слоя, в исходные значения.
