Инструмент «Монтажная область»

Инструмент «Монтажная область» позволяет добавлять, перемещать и изменять размер монтажных областей в документе.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Размер: используется для установки предустановленного размера монтажной области, которая будет добавлена на панель вставки.
  • Вставить монтажную область: используется для добавления новой монтажной области на панель вставки.
