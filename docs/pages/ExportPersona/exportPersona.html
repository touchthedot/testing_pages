Экспортирование с использованием режима «Экспорт»

Режим «Экспорт» представляет собой специальную рабочую область для экспорта монтажных областей, слоев, групп и объектов как фрагментов экспорта с указанием одновременно формата файла и размера изображения. Также можно экспортировать пользовательские нарисованные фрагменты.

Принцип работы

Для создания фрагментов в режиме «Экспорт» используются возможности нескольких панелей и инструментов. Фрагменты — это области документа, которые должны быть экспортированы.

В режиме «Экспорт» можно использовать возможности панелей «Слои», «Параметры экспорта» и «Фрагменты». Инструмент «Фрагмент» — это уникальный инструмент, доступный в режиме «Экспорт» и используемый для создания пользовательских фрагментов.

Использование панели «Слои»

Панель Слои отличается от панели «Слои» в режиме «Рисование». Она используется только на первом этапе для выбора монтажных областей, слоев, групп или объектов, из которых создаются фрагменты, добавляемые потом на панель Фрагменты.

При экспорте элемента панели «Слои» как фрагмента его размер автоматически меняется в соответствии с размером выделенного элемента. Символ круга в верхней правой части фрагмента свидетельствует о том, что его размер определен автоматически. Если вы решите изменить размер фрагмента, символ поменяется на квадрат, обозначающий внесение изменений в фрагмент. Фрагмент с измененным размером, созданный из элемента панели «Слои», можно в любой момент вернуть к автоматически настроенному размеру.

Использование панели «Параметры экспорта»

На панели Параметры экспорта можно установить параметры экспорта по умолчанию или параметры для текущего выбранного элемента перед созданием фрагмента.

Использование панели «Фрагменты»

На панели Фрагменты хранятся все фрагменты (на панели «Слои» или инструмента «Фрагменты»), готовые к экспорту непосредственно из панели. Каждый созданный фрагмент имеет исходный формат экспорта (например, PNG, JPG или SVG), связанный с ним в момент создания. При необходимости для фрагмента можно выбрать другие форматы экспорта. В каждом формате экспорта можно выбрать различное масштабирование размера или абсолютные размеры.

Можно использовать имена файлов на панели Фрагменты для того, чтобы указать (или создать) иерархию папок, в которой необходимо разместить экспортируемые файлы. Для этого используется знак косой черты с наклоном вправо или наклонная черта.

Например, главное изображение в формате PNG можно разместить в папке img, расположенной в папке assets, с использованием следующего синтаксиса: assets/img/hero.

Если какой-либо элемент структуры папок не существует, иерархия папок будет создана после экспорта соответствующих фрагментов с помощью Экспортировать фрагменты (n).

Можно экспортировать всю страницу документа с помощью пунктов Файл > Экспорт или посредством выбора предварительно определенной области распространения на панели Фрагменты.

Использование инструмента «Фрагмент»

С помощью инструмента «Фрагмент» можно легко и свободно создавать области экспорта любых размеров и в любой части документа.

Выберите параметр Принудительное выравнивание пикселей на панели инструментов для полного растягивания пикселей при создании, перемещении или изменении фрагментов.

Чтобы установить параметры экспорта по умолчанию, выполните приведенные далее действия.
  1. На панели Параметры экспорта выберите вкладку Настройки по умолчанию.
  2. Отрегулируйте параметры согласно требованиям.
  3. По завершении вернитесь на вкладку Выделение.

Параметры будут использоваться при создании следующего фрагмента.

Чтобы экспортировать область документа как изображение, выполните приведенные далее действия.
  1. Выполните одно из перечисленных ниже действий.
    • На панели Слои выберите монтажную область, слой, группу или объект и нажмите Создать фрагмент.
    • Выберите инструмент «Фрагмент» и перетаскивайте курсор для определения области экспорта.
  2. Выполните одно из перечисленных ниже действий.
    • На панели Параметры экспорта отрегулируйте параметры экспорта согласно требованиям.
    • Перейдите на панель Фрагменты, разверните запись фрагмента и измените формат экспорта или выберите Предустановка экспорта во всплывающем меню (только предустановки для PNG, JPEG и форматов проектирования значков Apple).
  3. (Необязательно.) На панели Фрагменты нажмите значок «+» под записью формата экспорта, чтобы добавить дополнительный графический формат экспорта.
  4. (Необязательно.) На панели Фрагменты нажмите значок «+», размещенный с отступом под записями размера экспортируемого фрагмента, чтобы добавить уровни увеличения разрешения (2x, 3x и т. д.) в формат экспорта.
  5. Для управления итоговым экспортированным файлом выполните одно из перечисленных далее действий.
    • Выберите Нажмите, чтобы экспортировать все форматы для этого элемента.
    • Выберите Нажмите, чтобы экспортировать этот размер. Это позволит экспортировать определенный формат экспорта для фрагмента определенного размера.
    • Нажмите Экспортировать фрагменты (n), чтобы экспортировать все форматы и размеры для всех выбранных фрагментов на панели «Фрагменты».
  6. Перейдите к папке для хранения экспортируемых изображений, выберите ее и нажмите кнопку Экспортировать.
  7. (Необязательно.) На панели Фрагменты выберите Непрерывно, если необходимо автоматически выполнять повторный экспорт областей экспорта при последующем изменении документа. Этот параметр доступен только после выбора элемента Экспортировать фрагменты (n) выше.

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать при работе с инструментом «Фрагмент».

  • С помощью клавиши можно сохранить квадратные пропорции области.
  • : нажатие этой клавиши во время вытаскивания области фрагмента позволяет рисовать в направлении от центра исходного положения перетаскивания (может использоваться в сочетании с ).
Чтобы восстановить исходные размеры вручную измененной области экспорта, созданной из слоя, выполните приведенные далее действия.
  1. Инструмент «Фрагмент»: выберите область экспорта, размер которой был изменен.
  2. На контекстной панели инструментов выберите пункт Вернуть автоматический размер.
