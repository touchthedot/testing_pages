Редактирование векторных кривых и фигур

Векторные кривые и фигуры можно легко редактировать с помощью одного из приведенных ниже способов.

  • С помощью при рисовании кривой или фигуры.
  • С помощью инструмента «Узел».

Воспользуйтесь первым способом для точной настройки и регулировки кривой во время рисования, а вторым — при выполнении более продолжительных операций редактирования.

Для редактирования кривых во время рисования выполните перечисленные далее действия.
  • Нажмите для перемещения узлов и регулирования управляющих маркеров, когда необходима точная настройка кривой.
    Кроме того, на панели «Цвет» можно изменить цвет и значение непрозрачности.

Нажмите и удерживайте и в это время нажимайте другие сегменты для того, чтобы редактировать их таким же образом.

Изменение кривизны сегмента

Выполните одно из перечисленных ниже действий.

  • Выберите узел и перетащите управляющие маркеры.

  • Перетащите непосредственно линию для того, чтобы установить ее в нужное положение.

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать для ускорения рабочего процесса с инструментом «Узел».

  •  — используется для ограничения шагов управляющего маркера до интервалов в 45 градусов. В интеллектуальном режиме использование этой функции приводит к установлению ограничений при расположении узла.
  •  — используется для изменения типа узла на резкий с целью создания точки перегиба на линии.
  • + — используется для расширения перетаскиваемого управляющего маркера с блокировкой направлений.

Инструмент «Узел» можно использовать в сочетании с возможностями панели Трансформация для точного позиционирования или масштабирования узлов.

Чтобы добавить узлы, выполните приведенные далее действия.
  • Выделите кривую и нажмите точку, в которой необходимо добавить узел.
Чтобы удалить узлы, выполните приведенные далее действия.
  • Выберите узел и нажмите .

Форма кривой автоматически изменится из-за удаления узла. Можно сохранить исходную геометрическую форму кривой, нажав + .

Чтобы замкнуть кривые, выполните приведенные далее действия.

Выполните одно из перечисленных ниже действий.

  • Выделите кривую и нажмите Замкнуть кривую на контекстной панели инструментов.
  • Перетащите конечный узел на начальный и отпустите кнопку, когда изменится указатель.
Чтобы разбить (разделить) кривые, выполните приведенные далее действия.
  • Выберите узел в точке, в которой необходимо разбить кривую и нажмите Разбить кривую на контекстной панели инструментов.
Чтобы соединить кривые, выполните приведенные далее действия.
  1. Выбрав инструмент «Узел» (нажмите и удерживайте , если используется инструмент «Перо»), нажмите и удерживайте , выбирая обе кривые.
  2. На контекстной панели инструментов нажмите Соединить кривые.
Чтобы обратить кривые, выполните приведенные далее действия.
  • На контекстной панели инструментов нажмите Обратить кривую. Начальный узел становится активным. Можно продолжать рисовать с противоположного конца кривой.
Для того чтобы преобразовать тип узла, выполните перечисленные далее действия.

Выполните одно из перечисленных ниже действий.

  • Выберите узлы, которые необходимо изменить, и нажмите один из типов узлов — Резкий, Интеллектуальный или Плавный — на контекстной панели инструментов.
  • Нажмите и удерживайте , а затем нажмите узел для того, чтобы преобразовать его в режим Резкий.
