Сегмент

Инструмент для рисования фигур Сегмент упрощает создание эллипса и предусматривает удаление секции на основании процентного соотношения.

Настройка

В инструменте «Сегмент» есть несколько вариантов выбора формы и параметров на контекстной панели инструментов, с помощью которых можно удалить часть верхней и нижней секций эллипса на основании процентного соотношения, а также регулировать угол эллипса.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Угол: используется для установки начального угла фигуры.
  • Нижняя / верхняя линия: используется для регулирования положения двух прямых линий фигуры. Таким образом определяется толщина сегмента.
  • Зеркальное отражение: используется для обновления указанных выше параметров и вывода инвертированной (перевернутой) версии текущей фигуры.
  • Негатив: используется для обновления указанных выше параметров и вывода обратной (противоположной) версии текущей фигуры.
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
