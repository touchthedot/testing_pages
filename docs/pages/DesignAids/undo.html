Использование функций «Отменить», «Повторить» и «История»

Можно легко отменить изменения, внесенные в файл, если была допущена ошибка или вам просто не нравится полученный результат.

Информация о любых правках документа сохраняется на панели «История». Для отмены правки можно впоследствии воспользоваться этой панелью или стандартными клавишами быстрого доступа. Можно также повторить правку, которая была недавно отменена.

Историю можно сохранить вместе с документом для того, чтобы можно было вернуться на более ранние этапы редактирования даже в случае, если документ был закрыт и вновь открыт.

Отмена или повтор изменения.

Выполните одно из перечисленных ниже действий.

  • В меню Правка нажмите Отменить для возврата к предыдущему действию или Повторить для повтора ранее отмененного действия.
  • На панели История нажмите одну из записей списка, чтобы перейти к этому состоянию.
  • На панели История перетащите ползунок влево для отмены изменения и вправо — для его повтора.
Для возврата в корень выполните перечисленные далее действия.
  • На панели История перетащите ползунок в самое крайнее левое положение.
Для сохранения истории вместе с документом выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Сохранить документ с историей.
  2. В диалоговом окне Сохранение с историей нажмите Да для того, чтобы согласиться с условиями, описанными в диалоговом окне.
  3. Сохраните документ.

Сохранение документа вместе с историей может привести к значительному увеличению размера файла проекта.

Функцию сохранения истории можно отключить, повторив шаг 1 описанной выше процедуры.

Для выбора установок панели «История» выполните перечисленные далее действия.
  1. В меню Affinity Designer выберите Установки.
  2. В меню Правка нажмите Установки.
  3. Нажмите Производительность.
  4. Перетащите ползунок Предел отмены влево для уменьшения количества сохраненных изменений или вправо для их увеличения.
  5. Нажмите Закрыть.
