Управление цветом

Информация о цвете и тоне цифрового документа сохраняется в виде цифр. При передаче документов с одного устройства на другое целевое устройство должно определить, как отображать цвет. Поскольку не все устройства могут одинаково отображать цветовую гамму, это может привести к разному представлению цветов на различных устройствах.

Для обеспечения одинакового представления цвета на всех устройствах используются цветовые профили, которые указывают устройству, как отображать или передавать информацию о цвете.

В Affinity Designer цветовой профиль открытого файла сохраняется по умолчанию. Можно преобразовать его в текущее рабочее цветовое пространство. При размещении изображений в существующем документе встроенный цветовой профиль каждого изображения будет всегда преобразовываться в текущее рабочее пространство документа.

Для обеспечения точного управления цветом при экспорте можно встроить назначенный цветовой профиль или профиль документа. Либо можно не встраивать назначенный профиль или профиль документа в экспортируемый файл, оставив его неотпрофилированным.

Назначение цветовых профилей

В Affinity Designer можно выбрать глобальные цветовые профили по умолчанию, назначить цветовой профиль при создании документа или в любой момент во время сеанса.

В большинстве коммерческих принтеров будет распознаваться пространство sRGB, поскольку на этапе печати эти принтеры могут выполнять собственное профилирование для максимально выгодного представления вашей работы.

В случае использования цветовой модели CMYK лучше всего проконсультироваться с партнером, который будет производить печать, чтобы получить рекомендацию о подходящем цветовом профиле CMYK.

Чтобы выбрать глобальные цветовые профили по умолчанию, выполните указанные ниже действия.
  • Сначала выберите Affinity Designer > Установки (параметр «Цвет»), а затем выберите во всплывающих меню цветовой профиль RGB, CMYK, «Оттенки серого» или LAB.
  • Выберите параметр Метод преобразования и установите флажок в поле Компенсация черной точки.

Выбранный профиль будет использоваться в виде текущего рабочего пространства, предлагаться при создании новых документов или применяться, когда пользователь выберет преобразование цветового пространства открытого файла (с удалением собственного цветового профиля).

Чтобы выбрать цветовой профиль нового документа, выполните указанные ниже действия.
  • При создании нового документа выберите параметр во всплывающем меню Цветовой профиль.
Чтобы преобразовать цветовое пространство открываемого файла в текущее рабочее пространство, выполните указанные ниже действия.
  • Прежде чем открыть файл, выберите Affinity Designer > Установки (параметр «Цвет»), а затем установите флажок в поле Преобразовать открытые файлы в рабочее пространство.
  • Прежде чем открыть файл, выберите Правка > Установки (параметр «Цвет»), а затем установите флажок в поле Преобразовать открытые файлы в рабочее пространство.

Среди имеющихся параметров можно выбрать, предупреждать ли о преобразовании рабочего пространства файла или о назначении профиля текущего рабочего пространства неотпрофилированному файлу.

Чтобы в любое время изменять цветовой профиль документа, выполните указанные ниже действия.
  1. В меню Файл выберите пункт Настройка документа.
  2. В диалоговом окне выполните указанные ниже действия.
    • Выберите вкладку Цвет.
    • Во всплывающем меню Цветовой профиль выберите профиль.
    • Выберите параметр Назначить или Преобразовать.
      Назначить: используется для применения нового профиля с неизменными значениями цветов и пикселей. Преобразовать: используется для преобразования каждого цвета старого профиля в новый, и, в результате, значения цветов и пикселей могут изменяться.
    • Нажмите ОК.

Как правило, в дизайне используются цветовые профили sRGB IEC61966-2.1 и Adobe RGB 1998, причем первый из них — для отображения на экране.

Чтобы встроить цветовой профиль в файл при экспорте, выполните указанные ниже действия.
  1. Перейдите в режим «Экспорт» и выберите необходимую предустановку на панели «Параметры экспорта».
  2. (Необязательно.) Выберите другой Профиль ICC во всплывающем меню. В противном случае будет встроен цветовой профиль документа.
  3. Установите флажок в поле Встроить профиль ICC.

Можно также встроить профиль ICC, выбрав Файл > Экспорт (нажмите в диалоговом окне Дополнительно >).

По умолчанию экспортируемые файлы не профилируются, т. е. цветовой профиль в них не встраивается. Благодаря этому создаются файлы небольшого размера, оптимизированного для использования в Интернете. При открытии файла ему присваивается рабочий цветовой профиль.

О цветопробе

Цветопроба имитирует вывод изображения при проектировании с учетом цветового профиля и бумажного носителя, на котором будет предположительно осуществляться печать.

В Affinity Designer для этого можно применить к проекту корректировку Цветопроба. Впоследствии можно воспользоваться предварительным просмотром, чтобы посмотреть, как будет выглядеть выводимое изображение, и не допустить появление неприятных сюрпризов во время печати.

Поскольку цветопроба применяется в виде корректировки, можно применить несколько таких корректировок, тем самым создавая цветопробы для нескольких устройств вывода.

Пример. Если вы хотите создать несколько разных типов выводимого изображения, при создании документа можно начать с выбора цветового профиля с более широкой гаммой (например, Adobe RGB 1998), после чего изменить профиль для соответствия назначению выводимого изображения. Но информация о цвете может быть потеряна, если выбрать менее широкую цветовую гамму — простой возврат к профилю с более широкой гаммой не восстановит информацию о дополнительных цветах. Применение корректировки «Цветопроба» позволяет этого избежать; пользователь может работать с изображением с более широкой гаммой, пока не будет готов перейти к выбранному профилю вывода.
