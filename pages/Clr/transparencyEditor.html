Редактор прозрачности

Как и при редактировании градиента, при редактировании прозрачности можно настраивать градиент, но только для альфа-канала композиции. Таким образом, можно создавать более сложные градиенты прозрачности, а затем применять их к заливке и обводке объектов.

Можно добавить более двух уровней альфа-канала в градиентном контуре, настроить положение каждого узла и (или) управлять цветовыми переходами. Это можно выполнить двумя способами:

  • непосредственно на объекте;
  • с помощью соответствующей контекстной панели инструментов.

В первом случае вы изменяете градиент прозрачности на глаз, а во втором — можете проектировать с высокой точностью и полностью контролируете процесс.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Тип: используется для определения типа градиента. Выберите этот вариант во всплывающем меню.
  • Положение: используется для контроля положения цветового узла в градиенте слева (0 %) направо (100 %), где 50 % — это центральная точка.
  • Средняя точка: используется для настройки распределения альфа-канала между выбранным узлом и узлом справа от него.
  • Непрозрачность: используется для определения степени прозрачности узла. Значение параметра 100 % соответствует полной непрозрачности, а 0 % — полной прозрачности.
  • Вставить: используется для добавления нового узла между выбранным и находящимся справа. Этому узлу назначается уровень альфа-канала его нового положения.
  • Копировать: используется для дублирования выбранного узла. Дубликат узла помещается между выбранным и находящимся справа.
  • Удалить: используется для удаления выбранного узла из градиента. Вследствие удаления конечного узла контур укорачивается до соседнего узла.
  • Обратить: используется для инвертирования градиента, то есть создания зеркального отражения.
