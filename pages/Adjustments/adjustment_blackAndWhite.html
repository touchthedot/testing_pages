Корректировка черного и белого

Преобразуйте цветное изображение в монохромное, полностью контролируя изменение отдельных цветов.

Применяя эту корректировку, можно воспользоваться удобным инструментом Пипетка, который предназначен для определения преобладающего цвета в выбранной области и автоматической регулировки соответствующего ползунка.

Можно применить цветовой тон, например сепию, к получившемуся изображению с оттенками серого. Для этого можно применить корректировку Изменить цвет.

Также можно создавать черно-белые изображения с помощью корректировок HSL, Градиентная карта и Пороговое значение.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • С помощью ползунков можно регулировать уровень светлоты определенных цветовых областей изображения. Для затемнения областей определенного цвета перетащите ползунок влево, а для их осветления — вправо.
  • Пипетка: используется для изменения корректировки посредством перемещения этого инструмента по изображению. Первый щелчок позволяет определить преобладающий цвет; при перемещении влево участки определенного цвета будут затемняться, а при перемещении вправо — осветляться. Положение соответствующего ползунка изменится.
