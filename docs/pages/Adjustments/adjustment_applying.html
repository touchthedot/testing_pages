Применение корректировок

Существует целый ряд корректировок, которые можно использовать в творческих целях или в целях исправлений во время работы с композицией и которые добавляются к изображению в виде нового слоя.

Доступные варианты корректировок
  • Черно-белое изображение
  • Яркость и контраст
  • Микширование каналов
  • Цветовой баланс
  • Кривые
  • Экспозиция
  • Градиентная карта
  • HSL
  • Инвертировать
  • Светофильтр
  • Уровни
  • Таблица перекодировки
  • OCIO (OpenColorIO)
  • Постеризировать
  • Изменить цвет
  • Выборочная цветокоррекция
  • Тени / светлые участки
  • Цветопроба
  • Раздельное тонирование
  • Пороговое значение
  • Красочность
  • Баланс белого

Для применения корректировок воспользуйтесь панелью Слои. Для большинства корректировок предусмотрена возможность использования пользовательских параметров, включая параметры общей коррекции.

Корректирующие слои затрагивают только расположенные ниже объекты слоя. Кроме того, корректировку можно применять к дочернему объекту, при этом изменения коснутся только этого объекта.

Некоторые корректировки (например, «Кривые», «Уровни» или «Микширование каналов») могут быть выполнены в любом цветовом пространстве, независимо от цветового пространства документа.

Параметры

Общие параметры, которые доступны во всех диалоговых окнах корректировок, описаны ниже.

  • Удалить: закрытие диалогового окна, удаление корректирующего слоя и отмена корректировок изображения.
  • Объединить: объединение текущего корректирующего слоя и слоя, который расположен сразу после него.
  • Сбросить: сброс всех параметров в диалоговом окне до значений по умолчанию.
  • Непрозрачность: определение степени прозрачности корректирующего слоя.
  • Режим наложения: определение характера взаимодействия пикселей корректирующего слоя и пикселей расположенного ниже слоя. Выберите тип режима во всплывающем меню.

Специальные диалоговые окна или пользовательские параметры предусмотрены не для всех типов корректировок.

Для применения корректировки выполните перечисленные далее действия.
  1. На панели Слои выполните одно из перечисленных ниже действий.
    • Выделите слой, к которому хотите применить корректировку, в качестве дочернего. Корректировка будет применяться ко всем объектам выделенного слоя, исходя из ее положения в стеке слоев.
    • Выделите только тот объект или группу, к которым должна применяться корректировка.
  2. Нажмите Корректировки и выберите нужный вариант корректировки во всплывающем меню.
  3. Если для выбранного типа корректировки появится диалоговое окно, выполните следующие действия.
    1. Выберите нужные параметры в диалоговом окне.
    2. Закройте диалоговое окно.

Корректировки также можно делать, воспользовавшись меню Слой и подменю Создать корректировку.
