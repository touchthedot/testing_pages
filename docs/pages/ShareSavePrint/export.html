Экспорт

В диалоговом окне «Экспорт» применяется оптимизированный подход к экспорту документа, монтажной области или текущей выделенной области в файл общего формата, предоставляя пользователю быстрый доступ к стандартным параметрам экспорта.

Дополнительную информацию об экспорте отдельных монтажных областей см. в теме Экспорт монтажных областей.

Общие параметры

В диалоговом окне «Экспорт» можно настроить указанные ниже общие параметры.

  • Формат: этот параметр используется для выбора пиктограммы формата экспортируемого файла. К форматам экспорта относятся PNG, JPEG, GIF, TIFF, PSD, PDF, SVG, EPS, EXR и HDR.
  • Предустановка: этот параметр используется для установки предварительно определенных параметров экспорта для выбранного формата файла. Выберите этот вариант во всплывающем меню.
    В этом всплывающем меню находятся не только предустановки приложения по умолчанию, в нем могут также находиться пользовательские предустановки.
  • Область: этот параметр позволяет вместо экспорта всего документа экспортировать либо монтажные области любого документа, либо текущий выбранный объект. В случае последнего варианта можно экспортировать объект с фоном или без него, используя настройки Выделение с фоном или Выделение без фона.
    Чтобы экспортировать нарисованные фрагменты или определенные слои, воспользуйтесь режимом «Экспорт».
  • Не экспортируйте слои, скрытые в режиме «Экспорт»: при выборе этого параметра слои, скрытые в режиме «Экспорт», исключаются из экспортируемого файла, даже если отображаются на странице. Если этот параметр не выбран, все объекты на странице (находящиеся в пределах выбранного выше параметра «Область») будут экспортированы вне зависимости от того, скрыты ли они в режиме «Экспорт».
  • Дополнительно: этот параметр используется для доступа к расширенным параметрам выбранного формата файла. Дополнительную информацию об этих параметрах см. в теме Панель «Параметры экспорта».

Дополнительные параметры

Для файлов формата PNG, JPEG, GIF, TIFF, EXR и HDR.

  • Размер: параметр по умолчанию, используется для отображения собственных размеров изображения. Введите значения, чтобы установить альтернативную ширину и (или) высоту для экспортируемого изображения.
  • Зафиксировать пропорции: когда этот параметр выбран (по умолчанию), сохраняются собственные пропорции изображения. Если этот параметр не выбран, ширина и высота экспортируемого изображения могут задаваться независимо друг от друга.

Если размер экспортируемого проекта превышает максимальные размеры формата файла, в который осуществляется экспорт, проект будет масштабироваться при экспорте для обеспечения соответствия максимальным размерам этого файла. Пользователь получит предупреждение об этом перед продолжением работы.

Для файлов формата PNG, JPEG, GIF, TIFF и PSD.

  • Передискретизация: выберите, какой следует использовать метод передискретизации (см. примечание ниже), если при экспорте разрешение изображения должно увеличиваться или уменьшаться.

Для файлов формата JPEG.

  • Качество: этот параметр используется для установки итогового качества экспортируемого изображения. Более высокое качество может значительно увеличить размеры файлов.

Для файлов формата SVG.

  • Экспортировать текст в виде кривых для обеспечения независимости от шрифта: если этот параметр выбран, текст в итоговом файле будет представлен в виде кривых (тем самым отображаясь в точности, как предполагалось, даже при просмотре с устройства, на котором не установлены используемые шрифты). Если этот параметр не выбран (по умолчанию), текст будет экспортироваться в виде текстового объекта, а для правильного отображения на устройстве просмотра должны быть установлены используемые шрифты.

Для файлов формата PDF, SVG и EPS.

  • Разрешение растровых изображений: этот параметр позволяет пользователю выбрать разрешение для растрирования при использовании эффектов, которые будут растрированы в ходе экспорта.

Для файлов формата EPS.

  • Уровень PostScript: этот параметр используется для выбора версии экспортируемого файла в формате PostScript. Выберите этот вариант во всплывающем меню.

Для файлов формата TIFF.

  • Сохранить слои Affinity: слои документа сохраняются в экспортированном изображении формата TIFF.

Методы передискретизации

Пользователю доступны приведенные далее параметры передискретизации.

  • Ближайший сосед: простая передискретизация с самой быстрой обработкой. Используйте для абстрактных изображений.
  • Билинейный: алгоритмическая передискретизация, применяемая при сжатии изображений.
  • Бикубический: алгоритмическая передискретизация, применяемая при увеличении изображений. Этот метод передискретизации обеспечивает большую плавность по сравнению с билинейным, но обработка осуществляется медленнее.
  • Фильтр Ланцоша 3: сложная алгоритмическая передискретизация, обеспечивающая наилучшие результаты, но имеющая наибольшее время обработки. Доступны «разделяемый» и «неразделяемый» варианты.

В зависимости от формата изображения в диалогом окне экспорта может предлагаться пункт Предположительный размер файла для экспортированного изображения.

Чтобы экспортировать проект в виде изображения, выполните указанные ниже действия.
  1. В меню Файл выберите пункт Экспортировать.
  2. Выберите нужные параметры в диалоговом окне.
  3. Нажмите Экспортировать.
Чтобы создать пользовательскую предустановку из расширенных параметров, выполните указанные ниже действия.
  1. В меню Файл выберите пункт Экспортировать.
  2. В диалоговом окне выберите Дополнительно.
  3. Выберите нужные параметры в диалоговом окне.
  4. Нажмите Управление предустановками и выберите Создать предустановку.
  5. Введите имя пользовательской предустановки и нажмите ОК. Впоследствии пользовательская предустановка появится во всплывающем меню.
