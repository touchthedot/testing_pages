Пиксельные выделения по образцу цвета

Можно создать пиксельное выделение путем отбора образцов цвета на растровом слое.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Допуск: используется для определения того, насколько пиксели, которые будут включены в выделение, должны соответствовать выбранному цвету. Чем ниже значение допуска, тем уже становится диапазон включаемых в выделение пикселей. Чем выше значение допуска, тем шире диапазон пикселей, включаемых в выделение. Перетащите ползунок для того, чтобы установить значение.
  • Модель: используется для определения цветовой модели, используемой при отборе образцов. Выберите этот вариант во всплывающем меню.
Для создания пиксельного выделения по образцу цвета выполните перечисленные далее действия.
  1. Выделите растровый слой.
  2. В меню Выделить выберите Выделить образец цвета.
  3. Щелкните по растровому слою для выделения образца цвета.
  4. Выберите нужные параметры в диалоговом окне.
  5. Нажмите Применить.
