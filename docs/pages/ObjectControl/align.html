Выравнивание объектов

Объекты можно с точностью выровнять на странице с помощью команд панели «Упорядочить».

Параметры

Можно настроить параметры всплывающей панели Упорядочить на панели инструментов.

  • Выровнять по: всплывающее меню, в котором установлены критерии выравнивания для операции. Например, можно выравнивать по границам выделенного, развороту, полю страницы или первому (либо последнему) выделенному объекту в случае выделения нескольких объектов (путем перетаскивания области выделения или выделения с помощью ). Если поля не установлены, выравнивание выполняется по краям страницы.

Всплывающая панель «Упорядочить» также содержит параметры для распределения объектов.

Для того чтобы выровнять объекты, выполните перечисленные далее действия.
  1. Выберите несколько объектов.
  2. Выполните одно из перечисленных ниже действий.
    • На панели инструментов нажмите Упорядочить, выберите параметр Выровнять по, затем Выровнять по горизонтали или Выровнять по вертикали на всплывающей панели. Нажмите Готово.
    • В подменю Выравнивание меню Слой выберите функцию выравнивания.

Кроме того, можно настроить указанные ниже параметры на контекстной панели инструментов, но всегда в пределах границ выделения.

  • Выровнять по горизонтали: используется для выравнивания выбранных объектов по левому краю, центральным точкам или правому краю объектов.
  • Выровнять по вертикали: используется для выравнивания выбранных объектов по верхнему краю, центральным точкам (По середине) или нижнему краю объектов.

Для того чтобы выровнять объекты на разных слоях, необходимо включить параметр Редактировать все слои.

Настройте панель инструментов согласно потребностям, чтобы воспользоваться большим количеством функций панели «Выравнивание» (Вид > Настроить панель инструментов).

Для включения/выключения параметра «Редактировать все слои» выполните указанные ниже действия.
  • На панели Слои нажмите Редактировать все слои.
