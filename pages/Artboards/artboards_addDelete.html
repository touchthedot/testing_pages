Добавление (и удаление) монтажных областей

Пользователь может в любое время создавать монтажные области в новом документе или же добавлять их в текущий документ: их исходный размер определяется предустановленными параметрами или же может быть изменен.

Добавление монтажных областей

При создании нового документа первая монтажная область может быть создана автоматически с помощью параметров в диалоговом окне «Новый документ».

После открытия документа можно добавлять монтажные области с помощью инструмента «Монтажная область».

Способы создания монтажных областей

  • С использованием размеров исходного документа — этот способ идеально подходит для создания многостраничных документов.
  • С использованием предустановленных размеров — этот способ идеален, если для работы используются определенные устройства.
  • С использованием размеров текущего выделенного фрагмента.
  • С использованием настраиваемых размеров.
  • Путем копирования или дублирования других монтажных областей.
  • Путем преобразования выделенного объекта.

Удаление монтажных областей

Монтажные области могут быть удалены из проекта в любое время. Любые объекты монтажной области также будут удалены.

В качестве альтернативы удалению монтажной области вы можете преобразовать ее в стандартный объект. В этом случае любые объекты монтажной области будут включены (вложены) в новый объект, что обеспечит сохранность композиции.

Для создания нового документа с монтажной областью выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Создать.
  2. В диалоговом окне выберите пункт Создать монтажную область.
  3. Выберите нужные параметры в диалоговом окне.
  4. Нажмите ОК.

Параметры Предустановка страницы или Ширина страницы и Высота страницы, установленные в описанном выше шаге 3, будут применяться к монтажной области и будут заданы в качестве параметров по умолчанию для будущих монтажных областей.

Для добавления монтажных областей с предустановленными размерами выполните перечисленные далее действия.
  1. (Необязательно.) С помощью инструмента «Перемещение» выделите один или несколько объектов.
  2. На панели Инструменты выберите инструмент «Монтажная область».
  3. Во всплывающем меню Размер контекстной панели инструментов выберите одно из указанного далее.
    • Документ: добавление на холст монтажной области с размерами текущего документа.
    • Выделение: добавление монтажной области в верхнюю часть стека слоев в том же размере и положении, что и выделенные объекты.
    • Любая другая предустановка — добавление другой монтажной области с размерами, предварительно установленными для определенного устройства.
  4. На контекстной панели инструментов нажмите Вставить монтажную область.
Для добавления монтажных областей с пользовательскими размерами выполните перечисленные далее действия.
  • Перемещая инструмент «Монтажная область» на панели вставки, определите размер монтажной области (или же выполните это действие в пределах другой монтажной области для создания вложенной панели).

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать при работе с инструментом «Монтажная область».

  •  — ограничение пропорций монтажной области в момент создания (квадрат) или при изменении размера.
  •  — изменение размера монтажной области относительно ее центра.
  •  — поворот монтажной области вокруг противоположной метки-манипулятора.
  • С помощью нажатия правой кнопки мыши можно вращать монтажную область вокруг противоположной метки-манипулятора.
Для копирования или дублирования монтажной области выполните перечисленные далее действия.
  • Выделите монтажную область и выполните одно из перечисленных ниже действий.
    • Перетащите монтажную область с помощью инструмента «Перемещение» или инструмента «Монтажная область».
    • В меню Правка выберите пункт Дублировать.
Для преобразования объекта в монтажную область выполните перечисленные далее действия.
  • Выделите один объект. В меню Слой выберите пункт Преобразовать объект в монтажную область.

После преобразования объекта в монтажную область вы по-прежнему сможете менять эту область, как будто это исходный объект. Например, монтажная область, преобразованная из замкнутой фигуры, по-прежнему может быть изменена как замкнутая фигура, а монтажная область, преобразованная из художественного текста, по-прежнему может быть отредактирована как текст.

Для удаления монтажной области выполните перечисленные далее действия.
  1. Выполните одно из перечисленных ниже действий.
    • С помощью инструмента «Монтажная область» нажатием выделите одну монтажную область или удерживайте и нажимайте несколько монтажных областей для их выделения.
    • Выделите одну или несколько монтажных областей на панели Слои.
  2. Выполните одно из перечисленных ниже действий.
    • Нажмите .
    • На панели Слои нажмите Удалить слой.
Для преобразования монтажной области в объект выполните перечисленные далее действия.
  • Выделите одну монтажную область. В меню Слой выберите пункт Преобразовать монтажную область в объект.
