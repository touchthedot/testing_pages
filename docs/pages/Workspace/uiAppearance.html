Изменение внешнего вида пользовательского интерфейса

По умолчанию в Affinity Designer используется темный интерфейс пользователя. Это обеспечивает точность просмотра и редактирования цветов в пользовательских проектах. Однако можно изменить стиль интерфейса пользователя с темного на светлый (и наоборот), отрегулировать оттенки серого для фона или общую гамму для изменения контрастности.

Чтобы изменить стиль интерфейса пользователя с темного на светлый (или наоборот) выполните приведенные далее действия.
  1. Выберите Affinity Designer > Установки.
  2. Выберите Правка > Установки.
  3. Нажмите метку Интерфейс пользователя.
  4. Для параметра Стиль интерфейса пользователя выберите «Темный» или «Свет».

Требуется перезапуск.

Чтобы изменить фон интерфейса пользователя, фон монтажной области или гамму, выполните приведенные далее действия.
  1. Выберите Affinity Designer > Установки.
  2. Выберите Правка > Установки.
  3. Нажмите метку Интерфейс пользователя.
  4. Перетащите ползунки, чтобы отрегулировать параметры Уровень серого в фоне, Уровень серого фона монтажной области и (или) Гамма интерфейса.
