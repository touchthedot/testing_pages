Панель «Символы»

С помощью панели «Символы» можно создавать и сохранять символы из выделенного на странице объекта или группы.

О панели «Символы»

После появления символов на панели их можно перетаскивать на страницу или монтажную область несколько раз. Эти экземпляры размещенных символов связаны между собой, и вследствие синхронизации любая правка отдельного экземпляра символа будет применена ко всем остальным экземплярам.

Панель «Символы» скрыта по умолчанию. Ее отображение можно включить, перейдя в меню Вид > Студия при работе в режимах «Рисование» или «Пиксельный».

Параметры

На панели доступны приведенные далее параметры.

  • Создать: используется для создания символа из текущего выделенного объекта или группы.
  • Открепить: используется для возврата экземпляра символа к виду стандартного объекта.
  • Синхронизировать: когда этот параметр включен, все экземпляры символов на странице обновляются автоматически при правке любого из экземпляров. Когда синхронизация отключена, правки применяются только к этому отдельному изолированному символу.
