Группирование объектов

Можно сгруппировать объекты для упрощения работы над проектом. Одну или несколько групп можно присвоить «родительской» группе.

Сгруппированные объекты расположены вместе, благодаря чему их можно легко выделять, перемещать или копировать, а также редактировать, как будто они являются одним объектом.

Группы и вложенные группы позже могут быть расформированы на отдельные компоненты или объекты. С помощью одной команды можно разгруппировать как отдельные группы, так и несколько групп, в том числе и вложенных.

Кроме того, потом можно преобразить группу в слой (при необходимости) для упрощения упорядочивания компонентов проекта.

Группы объектов отображаются в виде вложенных сгруппированных слоев на панели Слои.

Для создания группы выполните перечисленные далее действия.
  1. Выберите несколько объектов.
  2. Выполните одно из перечисленных ниже действий.
    • На контекстной панели инструментов нажмите Группировать.
    • В меню Слой выберите пункт Группировать.
Для того чтобы разгруппировать объекты, выполните приведенные далее действия.
  1. Выберите группу.
  2. Выполните одно из перечисленных ниже действий.
    • На контекстной панели инструментов нажмите Разгруппировать.
    • В меню Слой выберите пункт Разгруппировать.
Для того чтобы исключить из группы объект, выполните перечисленные далее действия.
  1. Выберите объект, который необходимо исключить из группы.
  2. Нажмите правую кнопку мыши на объекте и в меню выберите пункт Расформировать.
Для того чтобы разгруппировать все сгруппированные объекты, выполните приведенные далее действия.
  1. Выделите сгруппированные объекты.
  2. В меню Слой выберите пункт Разгруппировать все.

Все группы и вложенные группы будут разгруппированы, а ранее сгруппированные объекты преобразуются в отдельные, одиночные объекты.

Для того чтобы выбрать объекты (или группы) в группе, выполните приведенные далее действия.
  • Дважды нажмите на объект. Если объект принадлежит группе, сначала будет выделена группа. Снова дважды нажмите кнопку мыши, чтобы выбрать объект.
Для того чтобы преобразовать группу в слой, выполните приведенные далее действия.
  • В меню Слой выберите пункт Преобразовать группу в слой.
