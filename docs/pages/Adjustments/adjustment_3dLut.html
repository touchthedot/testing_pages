Корректировка таблицы перекодировки

Изменяя значение в таблице перекодировки, можно эмулировать цвета изображения, используемые в различных медиаплатформах.

Таблица перекодировки перераспределяет пиксельные значения цветов на основе определенной матрицы XYZ. Это позволяет быстро воссоздать внешний вид различных типов мультимедиа, например воспроизвести холодный синий оттенок, соотносимый с научной фантастикой и фильмами ужасов.

Таблица перекодировки хранится в файле, ранее экспортированном из Affinity Photo или других приложений (например, пакеты видеозаписей). Либо она может выводиться в процессе сравнения изображений.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Загрузить таблицу перекодировки: используется для определения файла таблицы перекодировки, который применяется к документу. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.
  • Вывести таблицу перекодировки: используется для применения корректировки таблицы перекодировки. Нет необходимости открывать отдельный файл таблицы перекодировки; процесс осуществляется посредством сравнения исходного изображения и соответствующего экспортированного изображения с коррекцией. После загрузки нового целевого изображения нажмите на элемент Вывести таблицу перекодировки, а затем во всплывающем диалоговом окне перейдите к исходному и откорректированному файлам, выберите их по очереди и нажмите кнопку Открыть.
  • Красный основной: если результаты не соответствуют ожидаемым, снимите флажок, чтобы возвратить неправильную матрицу «синий — зеленый — красный» XYZ к правильной матрице «красный — зеленый — синий» XYZ.
  • BGR: при выборе этого параметра значения оттенков инвертируются, а значения насыщенности и яркости, установленные таблицей перекодировки, сохраняются.

Доступны следующие типы файлов таблицы перекодировки: *.3dl, *.csp, *.cube и *.look.
