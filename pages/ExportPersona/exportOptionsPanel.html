Панель «Параметры экспорта»

Установите параметры экспорта для предварительно определенных областей экспорта с помощью панели «Параметры экспорта».

О панели «Параметры экспорта»

После того как была определена область экспорта, вы можете установить конкретные экспортные атрибуты для этой области с помощью панели «Параметры экспорта». Используя возможности этой панели, можно устанавливать разные параметры для разных областей экспорта на странице. Это дает вам полный контроль над экспортируемыми проектами.

Можно сохранить пользовательские параметры в виде предустановки для использования в будущем.

Общие параметры

На панели доступны приведенные далее параметры.

  • Режим: параметры экспорта по умолчанию могут быть изменены для выбранной области экспорта (для этого нужно предварительно активировать функцию «Выделение»). При выборе «Настройки по умолчанию» для новых фрагментов будут установлены параметры экспорта по умолчанию.
  • Предустановка: этот параметр позволяет установить предварительно определенные параметры экспорта для ряда стандартных форматов файлов, цветовых режимов и глубины бита. Выберите этот вариант во всплывающем меню.
  • Формат файла: этот параметр позволяет выбрать графический формат для экспортируемого изображения. Выберите этот вариант во всплывающем меню.
  • Передискретизация: этот параметр позволяет выбрать способ передискретизации (см. примечание ниже) для экспорта.
  • Встроить метаданные: при выборе этого параметра исходные метаданные любого растрового изображения сохраняются в экспортируемом файле. Если этот параметр отключен, все исходные метаданные удаляются.

Дополнительные параметры

Для файлов формата JPEG, PNG, TIFF, GIF, OpenEXR и Radiance HDR:

  • Пиксельный формат: этот параметр позволяет установить цветовой режим для экспортируемого изображения. Выберите этот вариант во всплывающем меню.
  • Подложка: этот параметр позволяет установить цвет фона для экспортируемого изображения. Выберите этот вариант на всплывающей панели.
  • Профиль ICC: по умолчанию установлен профиль ICC проекта (документа). Тем не менее профиль ICC проекта может быть перезаписан для этой области экспорта. Выберите этот вариант во всплывающем меню.
  • Встроить профиль ICC: при выборе этого параметра профиль ICC включается в данные экспортируемого изображения, что позволяет просматривать изображение с помощью соответствующего профиля на любом устройстве. Если этот параметр отключен, на устройстве просмотра должен быть установлен профиль ICC. В противном случае используется заменяющий профиль.

Для файлов формата JPG, PDF, SVG и EPS.

  • Качество: этот параметр используется для установки итогового качества экспортируемого изображения. Более высокое качество может значительно увеличить размеры файлов.

Для файлов формата PDF, SVG и EPS.

  • Растрировать: этот параметр используется для выбора варианта растеризации элементов композиции, которые не поддерживаются форматом файла. Выберите этот вариант во всплывающем меню.
    • Ничего: ни один элемент композиции не будет растрирован во время экспорта, поэтому неподдерживаемые элементы не включаются в экспортируемый файл.
    • Все: все элементы композиции растрируются для того, чтобы итоговый экспортируемый файл полностью соответствовал исходной композиции.
    • Неподдерживаемые свойства: только неподдерживаемые элементы растрируются в экспортируемом файле.
  • Понизить качество изображений: этот параметр используется для снижения качества растровых изображений в композиции.
  • Выше: если разрешение растровых элементов превышает установленное разрешение, оно будет снижено до этого установленного значения. Эта функция зависит от того, активен ли параметр Понизить качество изображений.
  • Использовать разрешение документа: этот параметр обеспечивает соответствие разрешения экспортируемого файла параметрам текущего проекта.
  • Использовать разрешение: этот параметр используется для отмены параметров разрешения текущего проекта для экспортируемого файла. Разрешение экспортируемого изображения устанавливается в смежном поле ввода.
  • Разрешить сжатие JPEG: при выборе этого параметра растровые элементы композиции будут сжаты с целью уменьшения размера экспортируемого файла. Если этот параметр отключен, растровые элементы композиции будут экспортированы в несжатом виде.

Для файлов формата PNG и GIF.

  • Палитровый: при выборе этого параметра выполняется кодировка экспортируемого изображения путем сопоставления его параметров с параметрами палитры и цветов, установленными ниже. (Этот параметр не может быть отключен для изображений в формате GIF.)
  • Палитра: по умолчанию значение этого параметра определяется автоматически. Тем не менее вы можете указать палитру кодировки самостоятельно. Выберите этот вариант во всплывающем меню.
  • Цвета: этот параметр используется для определения количества цветов в палитре. Выберите этот вариант во всплывающем меню.

Для файлов формата SVG и EPS.

  • Использовать относительные координаты: при выборе этого параметра объекты в экспортируемом файле имеют относительное положение для обеспечения максимально широких возможностей редактирования. Если этот параметр отключен, объекты имеют фиксированное положение в целях создания файла, который был бы оптимизирован для экранного отображения.

Для файлов формата JPG.

  • Прогрессивно: при выборе этого параметра во время загрузки экспортируемое изображение постепенно сжимается для оптимизации отображения файла.

Для файлов формата PSD.

  • Режим совместимости: при выборе этого параметра экспортируемый файл будет совместим с другими приложениями, которые не поддерживают некоторые компоненты (размер файла может также увеличиться). Если этот параметр отключен, экспортируемый файл может быть недоступен для чтения в других приложениях (в зависимости от компонентов, используемых в изображении).
  • Наименьшие размеры файлов: при выборе этого параметра будет выполняться максимально возможное сжатие элементов экспортируемого файла, однако он может быть недоступен для чтения в других приложениях. Если этот параметр отключен, сжатие экспортируемого файла не будет выполняться.
  • Растрировать все слои: при выборе этого параметра содержимое слоя растрируется в экспортируемом файле (структура слоев сохраняется). Если этот параметр отключен, растрирование не будет выполняться во время экспорта.
  • Стратегии. Этот параметр используется для определения способа экспорта особых атрибутов проекта. Выберите этот вариант во всплывающем меню.
    • Сохранять точный вид: указанный атрибут будет растрирован для сохранения целевой композиции.
    • Сохранить возможность редактирования: указанный атрибут будет экспортироваться с исходными параметрами для обеспечения удобства при редактировании.

Для файлов формата PDF.

  • Совместимость: этот параметр используется для определения версии и типа PDF для экспорта. Выберите этот вариант во всплывающем меню.
  • Цветовое пространство: этот параметр используется для выбора цветового пространства экспортируемого файла — текущего цветового пространства документа или выбранного цветового пространства. Выберите этот вариант во всплывающем меню.
  • Профиль: этот параметр используется для выбора цветового профиля экспортируемого файла — текущего цветового профиля документа или определенного цветового профиля. Выберите этот вариант во всплывающем меню.
  • Встроить профили: при выборе этого параметра (включен по умолчанию) выбранный цветовой профиль (или цветовой профиль документа) будет встроен в экспортируемый файл. Если этот параметр отключен, цветовой профиль не будет встроен в экспортируемый файл (для точного отображения экспортированного PDF-файла на устройстве просмотра должен быть установлен соответствующий цветовой профиль).
  • Сохранять плашечные цвета: при выборе этого параметра плашечные цвета композиции экспортируются такими, как они есть. Если этот параметр отключен, плашечные цвета будут преобразованы в эквивалентный цвет в пределах цветового пространства экспортируемого файла (см. выше).
  • Наложение черного: при выборе этого параметра для элементов композиции черного цвета CMYK будет задана печать поверх. Если этот параметр отключен, элементы композиции черного цвета CMYK будут неотличимы для других цветов во время печати.
  • Учитывать вылет под обрез: при выборе этого параметра любой выход за край, определенный в настройках документа, будет учитываться в выходном PDF-файле.
  • Включать метки принтера: при выборе этого параметра по краю страницы выходного PDF-файла будут находиться метки принтера. Все метки принтера добавляются по умолчанию. Тем не менее отображение определенных типов меток принтера можно отключить в зависимости от предпочтений пользователя. Эти метки перечислены далее.
    • Обрезные метки.
    • Метки приводки.
    • Панели цветов и оттенков серого.
    • Сведения о странице.
  • Встроить шрифты: этот параметр используется для установки параметра обработки шрифтов, используемых в документе.
    • Текст в виде кривых: этот параметр используется для преобразования всего текста в кривые. Это гарантирует то, что итоговый экспортируемый файл будет корректно отображаться, независимо от шрифтов, установленных на устройстве просмотра.
    • Все шрифты: любые шрифты, используемые в документе, встраиваются в экспортируемый файл. Это гарантирует то, что итоговый экспортируемый файл будет корректно отображаться, независимо от шрифтов, установленных на устройстве просмотра.
    • Нестандартные шрифты: используемые шрифты встраиваются в экспортируемый файл только в случае, если они не являются частью шрифтов, традиционно установленных на большинстве устройств. На устройстве просмотра должны быть установлены ожидаемые шрифты для отображения любых стандартных шрифтов в экспортируемом файле.
    • Без шрифтов: шрифты не встраиваются в экспортируемый файл. На устройстве просмотра должны быть установлены все используемые шрифты для точного отображения экспортируемого файла.
  • Подмножество шрифтов: при выборе этого параметра встроенные шрифты будут содержать только глифы, используемые в документе. Если этот параметр отключен, все глифы для используемых шрифтов будут встроены в экспортируемый файл независимо от того, содержатся они в документе или нет.
  • Разрешить дополнительные возможности: при выборе этого параметра все компоненты композиции, поддерживаемые форматом PDF-файла, экспортируются в виде векторов. Если этот параметр отключен, в зависимости от типа компоненты будут растрированы или преобразованы в кривые во время экспорта. Эти компоненты перечислены далее.
    • Художественный текст, который растянут горизонтально или вертикально.
    • Текст с обводкой.
    • Линейные и радиальные градиенты.
    • Области несплошной прозрачности.

    Если для экспорта выбран параметр Разрешить дополнительные возможности, при импорте итогового PDF-файла в другие приложения указанные выше компоненты могут быть растрированы или же неправильно отображены. При использовании приложения стороннего производителя также может отображаться сообщение об ошибке импорта PDF-файла.

Для файлов формата SVG.

  • Экспорт текста в виде кривых: при выборе этого параметра текст в итоговом файле будет отображаться в виде кривых (тем самым отображаясь в точности, как предполагалось, даже при просмотре с устройства, на котором не установлены используемые шрифты). Однако использование этого параметра приведет к увеличению размера файла, и вы не сможете редактировать текст как текстовый объект или использовать функцию преобразования текста в речь в других приложениях. Если этот параметр отключен, текст будет экспортироваться как текстовый объект, а для правильного отображения на устройстве просмотра должны быть установлены используемые шрифты.
  • Перенос длинного текста: при выборе этого параметра расположение текста будет определяться положением предшествующих строк текста (это приводит к уменьшению размеров файлов и упрощению их структуры). Если этот параметр отключен, расположение текста определяется абсолютными координатами.
  • Использовать шестнадцатеричные коды цветов: при выборе этого параметра цвета экспортируемого файла выражены в виде значений шестнадцатеричных кодов RGB (это приводит к уменьшению размера файла и снижению удобочитаемости). Если этот параметр отключен, цвета экспортируются как стандартные значения RGB.
  • Выполнить сведение трансформаций: при выборе этого параметра трансформированные объекты зафиксированы в экспортируемом файле. Это обеспечивает более точное отображение файла в различных приложениях. Если этот параметр отключен, объекты остаются динамически трансформированными для обеспечения более широких возможностей редактирования.
  • Использовать плиточные шаблоны: при выборе этого параметра растрированные области могут быть преобразованы в векторную фигуру с заполненной битовой картой для того, чтобы края были более сглаженными и резкими. Тем не менее эта функция может не поддерживаться некоторыми приложениями. Если этот параметр отключен, объекты будут существовать как единичные элементы в пределах экспортируемого файла.
  • Задать поле просмотра: при выборе этого параметра экспортируемый файл содержит координаты и размеры, которые определяют вид поля просмотра изображения. Если этот параметр отключен, данные поля просмотра не будут включены в экспортируемый файл. Область экспорта используется для определения параметров поля просмотра.
  • Добавить разрывы строк: при выборе этого параметра экспортируемый файл будет содержать код, оптимизированный для удобства просмотра и чтения пользователем. Если этот параметр отключен, экспортируемое изображение будет содержать однострочный код, что приведет к существенному уменьшению размера файла.

Для файлов формата EPS.

  • Уровень PostScript: этот параметр используется для выбора версии экспортируемого файла в формате PostScript. Выберите этот вариант во всплывающем меню.
  • Максимально уменьшить размер: при выборе этого параметра (включен по умолчанию) экспортируемый файл будет сжат для максимального уменьшения размера.

Методы передискретизации

Пользователю доступны приведенные далее параметры передискретизации.

  • Ближайший сосед: простая передискретизация с самой быстрой обработкой. Используйте для абстрактных изображений.
  • Билинейный: алгоритмическая передискретизация, применяемая при сжатии изображений.
  • Бикубический: алгоритмическая передискретизация, применяемая при увеличении изображений. Этот метод передискретизации обеспечивает большую плавность по сравнению с билинейным, но обработка осуществляется медленнее.
  • Фильтр Ланцоша 3: сложная алгоритмическая передискретизация, обеспечивающая наилучшие результаты, но имеющая наибольшее время обработки. Доступны «разделяемый» и «неразделяемый» варианты; последний дает немного лучшие результаты, но выполняется значительно медленнее по сравнению с «разделяемым».
Пользовательские параметры могут быть сохранены в качестве предустановки.
  1. Установите параметры экспорта по желанию.
  2. Нажмите «Установки панели» и выберите пункт Создать предустановку во всплывающем меню.
  3. Введите имя и нажмите OK.

Новая предустановка появится в конце списка всплывающего меню Предустановка и также будет доступна в диалоговом окне Экспорт.

Вы можете удалять и переименовывать пользовательскую предустановку, выбрав соответствующий параметр во всплывающем меню установок панели.
