Выноска: скругленный прямоугольник

Инструмент для рисования фигур «Выноска: скругленный прямоугольник» позволяет быстро создать выноску или облако текста в проекте.

Настройка

В инструменте «Выноска: скругленный прямоугольник» есть несколько вариантов выбора формы и параметров на контекстной панели инструментов, которые позволяют установить размер, форму и положение хвоста, радиус углов (включая возможность изменять каждый угол отдельно), а также глубину выноски в форме прямоугольника, подлежащую контролю.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Один радиус: если этот параметр включен (по умолчанию), пользователь может изменять все четыре угла одновременно посредством выбора одного значения радиуса. Если этот параметр выключен, можно установить значение радиуса каждого угла отдельно для создания независимых закругленных углов.
  • Абсолютные размеры: по умолчанию радиус угла указывается в виде процентного соотношения для объекта и масштабируется при изменении размера объекта. Если этот параметр включен, можно указать радиус угла в единицах измерения. При изменении размера объекта радиус угла не масштабируется вместе с объектом, а остается неизменным.
  • Радиус: используется для выбора радиуса скругленных углов прямоугольника.
  • Высота хвоста: используется для контроля высоты выступа фигуры относительно высоты самой фигуры.
  • Положение кончика хвоста: используется для регулирования положения острого конца выступа слева (0 %) направо (100 %), где 50 % — это центральная точка.
  • Положение хвоста: используется для регулирования положения выступа по основанию фигуры слева (0 %) направо (100 %), где 50 % — это центральная точка.
  • Ширина хвоста: используется для регулирования ширины выступа по основанию фигуры.
  • Показать центр поворота: используется для отображения перемещаемого центра поворота, вокруг которого можно поворачивать фигуру.
  • Скрыть выделение: если этот параметр включен, рамка выделения объекта временно скрывается при его трансформации. Если этот параметр выключен, рамка выделения остается видимой при трансформации. Выбранный вариант этого параметра применяется ко всем объектам до тех пор, пока не будет переключен вручную.
  • Сбросить рамку выделения: после изменения формы, поворота или наклона фигуры этот параметр используется для сброса рамки выделения в вертикальное положение.
  • Преобразовать в кривые: используется для преобразования выделенного объекта в несколько соединенных линий и узлов.

Чтобы сбросить настройки положения красного маркера до параметров по умолчанию, просто дважды щелкните маркер.
