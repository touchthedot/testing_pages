Корректировка «Цветопроба»

Предварительно просмотрите вид эффекта посредством вывода изображения на определенное цветовое пространство или устройство.

С помощью этой корректировки можно предварительно просмотреть различные варианты вывода для фотографии или проекта. Ее также можно творчески использовать для создания тональных эффектов. Поскольку она воспринимается как обычный корректировочный слой, перед экспортом или отправкой на печать ее необходимо скрыть или удалить. В противном случае эффект будет включен в выводимое изображение.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Профиль цветопробы: используется для определения используемого цветового профиля. Выберите его из меню или воспользуйтесь клавишами со стрелками вверх и вниз, чтобы циклически просматривать варианты.
  • Метод преобразования: используется для установки наглядной цели применения корректировки. Выберите этот вариант во всплывающем меню.
  • Компенсация черной точки: при выборе этого параметра (он установлен по умолчанию) черная точка проекта настраивается с учетом текущего контраста в текущем профиле цветопробы. Если этот параметр отключен, черная точка проекта не настраивается и контраст изображения может не учитываться.
  • Проверка цветовой гаммы: при выборе этого параметра цвета RGB без эквивалента в CMYK будут отображаться серыми.
