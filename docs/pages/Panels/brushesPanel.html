Панель «Кисти»

На панели «Кисти» содержится набор предустановленных кистей, которые можно выбрать для использования с инструментами «Векторная кисть» и «Кисть рисования» (в режимах «Рисование» и «Пиксельный» соответственно). Любые создаваемые вами пользовательские кисти можно сохранять в собственные категории панели. Можно импортировать файлы кистей ABR в отдельные категории.

О панели «Кисти»

Предлагаемые предварительно разработанные кисти доступны в виде пиктограмм, которые можно выбрать. При выбранном инструменте на основе кистей выделите пиктограмму, чтобы определить тип кисти для использования. Кисти упорядочены по категориям.

В режиме «Рисование» представлены только векторные кисти (инструмент «Векторная кисть»), а в режиме «Пиксельный» — исключительно пиксельные кисти (инструмент «Кисть рисования»).

На панели отображается приведенное далее содержимое.

  • Всплывающее меню категории. Для отображения пиктограмм кистей для выбранной категории.
  • Изменить: загрузка выбранной кисти в диалоговом окне Изменение с возможностью ее настройки и сохранения как новой предустановленной кисти.
  • Пиктограммы кистей для текущей категории.

В меню «Установки панели» доступны приведенные далее параметры.

  • Создать новую категорию: используется для добавления новой, пустой категории во всплывающее меню.
  • Переименовать категорию: используется для переименования текущей выбранной категории кистей.
  • Удалить категорию: используется для удаления текущей выбранной категории кистей.
  • Импортировать кисти: используется для импорта набора векторных или растровых кистей, в том числе в формате ABR. Импортированные векторные кисти будут доступны в новой категории на панели «Кисти» в режиме «Рисование», а импортированные растровые кисти будут расположены аналогичным образом в режиме «Пиксельный».
  • Экспортировать кисти: используется для экспорта текущей выбранной категории кистей, готовой для передачи другим пользователям.

Использование векторных кистей.

  • Создать сплошную кисть: используется для создания базового сплошного векторного мазка.
  • Создать текстурированную кисть интенсивности: используется для создания мазка кисти на основании значений непрозрачности растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.
  • Создать текстурированную кисть с изображением: используется для создания мазка кисти на основании значений цвета растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

Дополнительную информацию о создании пользовательских кистей см. Создание пользовательских кистей.

Использование пиксельных кистей.

  • Создать кисть интенсивности: используется для создания мазка кисти на основании значений непрозрачности растрового изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.
  • Создать круглую кисть: используется для создания мазка кисти с мягкими и размытыми краями.
  • Создать квадратную кисть: используется для создания мазка кисти с резким краем.
  • Создать кисть с изображением: используется для создания мазка кисти на основе изображения. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

Дополнительную информацию о создании пользовательских кистей см. Создание пользовательских пиксельных кистей.

При переключении в режим «Пиксельный» вместо векторных кистей будут отображаться пиксельные кисти (например, кисть распыления).
