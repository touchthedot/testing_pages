Использование снимков

Снимки — это идеальный способ сохранить текущую работу пользователя в любой момент времени; они скорее напоминают «стоп-кадр». Это можно делать заранее перед выполнением более сложных операций, когда может понадобиться возврат к предыдущему моменту времени (если что-то пойдет не по плану).

Если вы экспериментируете с различными подходами, можно также создавать разные снимки для каждого подхода, благодаря чему можно изучить, сравнить и оценить каждый из них по очереди. Впоследствии у вас будет возможность выбрать предпочтительный снимок, с которого стоит продолжать работу.

Если сравнивать с панелью «История», на которой можно выполнять пошаговый переход к предыдущей правке, снимки позволяют определить этап сеанса, с которого можно восстановить работу.

При необходимости можно также перенести снимок в новый документ.

О панели «Снимки»

Панель «Снимки» скрыта по умолчанию. Ее отображение можно включить, перейдя в меню Вид > Студия при работе в режимах «Рисование» или «Пиксельный».

Снимки сохраняются с документом и могут использоваться в будущих сеансах.

Для создания снимка выполните перечисленные далее действия.
  • На панели Снимки нажмите Добавить снимок.

Снимок с отображением даты и времени добавляется на панель.

Для восстановления снимка выполните перечисленные далее действия.
  1. Выберите снимок из списка на панели Снимки.
  2. Нажмите Восстановить снимок.
Для создания нового документа из снимка выполните перечисленные далее действия.
  1. Выберите снимок из списка на панели Снимки.
  2. Нажмите Новый документ из снимка.
Для удаления снимка выполните перечисленные далее действия.
  1. Выберите снимок из списка на панели Снимки.
  2. Нажмите Удалить.
