Соединение объектов

Можно соединить объекты для создания безграничного разнообразия фигур с помощью булевых операций.

Операции соединения

Доступные операции приведены далее.

Добавить: используется для создания нового объекта, получаемого путем объединения выбранных объектов.

Вычесть: используется для удаления перекрывающихся участков нижнего объекта. Все остальные выделенные объекты удаляются.

Пересечение: используется для создания нового объекта, получаемого из перекрывающихся участков выделенных объектов.

Разделить: используется для разделения участков объектов на отдельные объекты; объект, полученный из области пересечения, сохраняет цвет верхнего объекта.

Объединить: используется для объединения выделенных объектов в один составной объект, на котором в прозрачной области перекрываются участки с заливкой.

На новых объектах сохраняются свойства нижнего объекта в выделении.

Чтобы провести операции соединения, выполните указанные ниже действия.
  1. Выберите несколько объектов.
  2. Выполните одно из перечисленных ниже действий.
    • На панели инструментов выберите Добавить, Вычесть, Пересечение, Разделить или Объединить.
    • В подменю Геометрия меню Слой выберите команду операции.

Нажмите и удерживайте при выборе параметра Добавить, Вычесть, Пересечение или Объединить для создания неразрушающей составной формы.
