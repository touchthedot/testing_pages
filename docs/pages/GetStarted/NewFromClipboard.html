Создание новых документов из содержимого буфера обмена

Можно создать новый документ на основе актуального содержимого буфера обмена. Эта функция вставки в виде нового документа позволяет начать новый проект со скопированного элемента (слоя или выделения) другого проекта в Affinity Designer или других поддерживаемых приложениях.

Для создания нового документа на основе содержимого буфера обмена выполните перечисленные далее действия.
  • В меню Файл выберите пункт Создать из данных буфера обмена.
