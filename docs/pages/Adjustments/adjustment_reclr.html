Корректировка «Изменить цвет»

Создайте монохромное изображение, воспользовавшись полным спектром цветов.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Оттенок: используется для регулирования цвета, применяемого к изображению. Перетаскивайте ползунок, чтобы изменять цвета по всему спектру.
  • Насыщенность: используется для регулирования интенсивности цветов изображения. Для уменьшения интенсивности цвета перетащите ползунок влево, а вправо — для увеличения интенсивности.

Создание черно-белых изображений и изображений с эффектом сепии

  • Для создания обесцвеченного (черно-белого) изображения установите ползунок Насыщенность в крайнее левое положение.
  • Для создания изображения с эффектом сепии установите ползунок Оттенок в любой точке участка спектра оранжевого цвета, а ползунок Насыщенность — в любой точке слева от центра.
