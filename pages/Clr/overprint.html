Печать поверх

В профессиональной полиграфии можно использовать печать глобальных цветов поверх. Применяя цвет печати поверх к выборочным объектам, можно контролировать печать поверх.

О печати поверх

Печать поверх предусматривает печать одним цветом чернил поверх другого вместо «стирания» (удаления) цвета ниже (по умолчанию).

Печать поверх характерна для профессиональной полиграфии и применяется при публикации файлов PDF с использованием совместимости цветового пространства CMYK и формата PDF/X.

Вам не нужно напрямую задавать печать поверх для черного цвета, черного текста или черной графики, поскольку это задано по умолчанию. При публикации файлов PDF можно регулировать печать черного цвета поверх с помощью параметра Наложение черного на панели «Параметры экспорта» (для любых параметров экспорта файлов PDF).

Образец цвета печати поверх обозначен изогнутым ярлычком в правом верхнем углу поля с образцом цвета.

С помощью панели «Цвет» можно проверить, назначен ли цвет печати поверх обводке или заливке объекта.

Перед использованием этой функции обратитесь к поставщику полиграфических услуг за консультацией о принципе ее применения.

Для создания цвета печати поверх с нуля выполните указанные ниже действия.
  1. На панели Образцы выберите палитру документа во всплывающем меню палитры. Если палитра документа отсутствует, можно создать ее с помощью меню «Установки панели».
  2. В меню Установки панели выберите пункт Добавить глобальный цвет.
  3. Выберите нужные параметры в диалоговом окне.
  4. Выберите параметр Печатать поверх.
  5. Нажмите Добавить.
Для наложения существующего глобального цвета при печати выполните указанные ниже действия.
  • На панели Образцы выполните щелчок мышью по пиктограмме образца глобального цвета и выберите пункт Печатать поверх.
