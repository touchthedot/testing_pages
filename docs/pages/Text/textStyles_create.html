Создание стилей текста и управление ими

Стили текста могут быть созданы на панели «Стили текста», после чего их можно сразу применять к тексту. Кроме того, ими можно управлять и изменять их для улучшения проектирования и рабочего процесса.

Существует несколько способов создания нового стиля текста.

  1. Создание с нуля: новый стиль текста изначально не будет связан с каким-либо другим стилем текста.
  2. Дублирование из другого стиля: новый стиль текста изначально будет точно таким же, как и выбранный стиль текста, но не будет связан с этим стилем.
  3. Создание на основе существующего стиля: выбранный стиль текста будет использоваться в качестве основы для нового стиля текста. Между стилями сохраняется связь, и устанавливается иерархия.

Стиль текста может быть изменен двумя способами.

  • Непосредственное изменение стиля текста.
  • Обновление стиля текста таким образом, чтобы он совпадал с выбранным локально отформатированным текстом.

Кроме того, пользователь может установить собственные комбинации клавиш для применения определенных стилей текста к тексту.

Чтобы создать стиль текста с нуля, выполните перечисленные далее действия.
  1. (Необязательно) Выберите часть текста или нажмите внутри абзаца.
  2. На панели Стили текста выберите:
    • Создать стиль абзаца — для создания нового стиля абзаца;
    • Создать стиль символов — для создания нового стиля символов;
    • Создать стиль группы — для создания нового стиля группы.
    Дополнительные сведения см. в разделе Типы стилей текста.
  3. Выберите нужные параметры в диалоговом окне.
  4. (Необязательно) В диалоговом окне Изменить стиль текста выберите параметр Применить стиль к выделенному, чтобы применить стиль к тексту, выделенному на этапе 1.
  5. Нажмите ОК.
Чтобы создать стиль текста на основе существующего стиля, выполните перечисленные далее действия.
  1. На панели Стили текста нажмите меню параметров перечисленных стилей и выберите:
    • Дублировать — чтобы начать работу с точно такими же параметрами, как и в выбранном стиле;
    • Создать стиль на основе — чтобы автоматически установить параметр в диалоговом окне На основе на выбранный стиль.
  2. Выберите нужные параметры в диалоговом окне.
  3. Нажмите ОК.
Чтобы изменить существующий стиль текста, выполните перечисленные далее действия.
  1. На панели Стили текста нажмите меню параметров перечисленных стилей и выберите Правка.
  2. Выберите нужные параметры в диалоговом окне.
  3. Нажмите ОК.
Чтобы обновить стиль текста, выполните перечисленные далее действия.
  1. Выберите часть текста или нажмите внутри абзаца и выполните любые локальные изменения с помощью контекстной панели инструментов, панели «Символ» или панели «Абзац».
  2. На панели Стили текста:
    • Выберите Обновить стиль абзаца, чтобы обновить стиль абзаца, применяемый к текущему тексту, таким образом, чтобы он совпадал с локальным форматированием.
    • Выберите Обновить стиль символов, чтобы обновить стиль символов, применяемый к текущему тексту, таким образом, чтобы он совпадал с локальным форматированием.
    • В любом меню параметров перечисленных стилей выберите Обновить ... Из параграфов или Обновить ... из символов.
Чтобы назначить комбинацию клавиш для текстового стиля, выполните перечисленные ниже действия.
  1. На панели Стили текста нажмите меню параметров перечисленных стилей и выберите Правка.
  2. В диалоговом окне выберите раздел Стиль, а затем нажмите в поле Комбинация клавиш.
  3. Нажмите требуемую комбинацию клавиш.
  4. Нажмите ОК.

Если в поле отображается предупреждающий значок, значит эта комбинация клавиш уже назначена для другого действия или стиля текста. Наведите курсор на значок, чтобы узнать, что это за действие или стиль текста.

Чтобы удалить комбинацию клавиш из стиля текста, нажмите на значок в виде крестика в поле Комбинация клавиш.

Параметры

В разделе Стиль диалогового окна Изменить стиль текста доступны приведенные далее параметры.

  • На основе — устанавливает параметры по умолчанию для этого стиля. С точки зрения иерархии стиль текста «На основе» является основным стилем текста по отношению к подчиненному стилю.
  • Следующий стиль — для стилей абзацев, определяет стиль текста, автоматически применяемый для следующего абзаца (см. примечание).
  • Комбинация клавиш — устанавливает комбинацию клавиш для применения этого стиля текста.
  • Тип — определяет преобладающий характер стиля текста.
  • Показать на обеих панелях — если этот параметр отключен (по умолчанию), стиль текста может быть применен только в преобладающем режиме (определяется установленным выше типом). Если параметр выбран, стиль абзаца может быть применен в качестве стиля символов и наоборот.
  • Сбросить форматирование — удаляет все параметры, примененные в разделах «Символ», «Типографика» и «Абзац» в этом диалоговом окне. Параметры в разделе «Стиль» не меняются.

Функция Следующий стиль активируется только при нажатии во время набора текста или использования параметра Применить ... , затем «Следующие стили» на панели Стили текста.

Дополнительные параметры диалогового окна «Изменить стиль текста».
  • Имя стиля — определяет название стиля текста, которое будет использоваться в приложении.
  • Символ — назначает атрибуты уровня символа для стиля текста или перезаписывает атрибуты, определенные стилем «На основе». Подробности см. на панели «Символ».
  • Типографика — активирует элементы шрифтов OpenType для стиля текста или перезаписывает атрибуты, определенные стилем «На основе». Подробности см. в разделе Элементы шрифтов OpenType.
  • Абзац — назначает атрибуты уровня абзаца для стиля текста или перезаписывает атрибуты, определенные стилем «На основе». Подробности см. на панели «Абзац».
  • Параметры стиля — содержит список всех атрибутов, примененных к текущему стилю текста.
  • Применить стиль к выделенному — применяет текущий стиль текста к любому выделенному тексту (если текст был выделен до открытия диалогового окна).

Если [изменения] не отображаются, атрибут остается без изменений по сравнению со стилем На основе.
