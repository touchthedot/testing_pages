Корректировка уровней

Откорректируйте значения тона и цветовой баланс изображения, установив черную точку, белую точку и значение гаммы. Корректировка уровней влияет на распределение пикселей в изображении.

Корректировки можно выполнять в любом цветовом режиме, вне зависимости от текущего цветового режима документа.

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Выберите цветовой режим из первого всплывающего меню.
  • Укажите один цветовой канал, к которому будут применяться корректировки; можно выбрать в том числе альфа-канал слоя. Основной (выбран по умолчанию): используется для применения корректировки ко всем каналам. Выберите вариант во втором всплывающем меню.
  • Уровень черного: используется для определения диапазона пикселей в изображении, рассматриваемых как абсолютно черные. Перетащите ползунок вправо для включения пикселей в диапазон (и таким образом увеличивая тени) и влево для исключения пикселей (и таким образом уменьшая тени).
  • Уровень белого: используется для определения диапазона пикселей в изображении, рассматриваемых как абсолютно белые. Перетащите ползунок влево для включения пикселей в диапазон (и таким образом увеличивая количество светлых участков) и вправо для исключения пикселей (и таким образом уменьшая количество светлых участков).
  • Гамма: используется для определения распределения пикселей средних тонов в изображении. Перетащите ползунок влево, чтобы перераспределить пиксели в сторону черной точки, и вправо — в сторону белой.

Чтобы создать негативное (инвертированное) изображение, разместите ползунок Уровень черного по правую сторону от ползунка Уровень белого.

Нажимайте и удерживайте при изменении уровней черного или белого, чтобы включать предварительный просмотр областей, информация о которых будет утеряна вследствие сокращения диапазона, в режиме реального времени.
