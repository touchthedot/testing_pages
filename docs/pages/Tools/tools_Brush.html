Инструменты «Кисть»

Инструмент «Векторная кисть» доступен в режиме «Рисование», тогда как в режиме «Пиксельный» используется Кисть рисования. Инструменты работают по-разному в зависимости от текущего режима работы.

Линии переменной ширины можно регулировать либо посредством скорости (чаще всего может пригодиться при рисовании с помощью мыши), либо посредством нажима (используется при рисовании с помощью чувствительного к нажиму устройства).

Режим «Рисование»

С помощью инструмента «Векторная кисть» можно создавать нарисованные от руки изображения, рисуя произвольные линии и фигуры переменной ширины. Изображение будет иметь вид рисунка на бумаге.

Инструмент создает рисунок путем масштабирования или повторения изображения вдоль контура (линии).

По мере рисования автоматически создаются узлы, которые впоследствии можно отредактировать при рисовании с помощью или в любое время с помощью инструмента «Узел».

Инструмент использует многие параметры стандартной линии для контроля внешнего вида наносимого мазка. Типы кисти устанавливаются с помощью панели Кисти.

Режим «Пиксельный»

Кисть рисования наносит пиксели на страницу, создавая мазки со сглаженными краями. Это создает естественный переход между мазком и окружающими пикселями.

Другие инструменты на основе кисти в режиме «Пиксельный» используют те же параметры для контроля внешнего вида применяемых пикселей, хотя допускаются незначительные вариации.

Большинство кистей по умолчанию мягкие и круглые. Альтернативные стили можно выбрать на панели Кисти.

При любом выбранном инструменте «Кисть» можно быстро уменьшить или увеличить размер (ширину) кисти, удерживая и и перетаскивая, соответственно, влево или вправо. Если кисть обладает атрибутом жесткости, его можно настроить аналогичным образом, удерживая и и перетаскивая вверх или вниз.

При любом выбранном инструменте «Кисть» можно быстро уменьшить или увеличить размер (ширину) кисти, удерживая нажатой правую кнопку мыши, затем левую кнопку мыши и перетаскивая на странице, соответственно, влево или вправо. Если кисть обладает атрибутом жесткости, его можно настроить перетаскиванием вверх или вниз.

При любой выбранной кисти в режиме «Пиксельный» с помощью клавиш с цифрами можно быстро менять значение непрозрачности кисти. Дополнительную информацию см. в теме Прозрачность.

Параметры

Можно настроить указанные ниже параметры векторной кисти на контекстной панели инструментов.

  • Цвет: цвет мазка кисти. Выберите необходимый образец цвета из сплошных или выбранных цветов, цветных градиентов, предустановленных или пользовательских образцов цвета.
  • Ширина: используется для определения размера кисти (мазка) в пикселях. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Непрозрачность: используется для определения степени прозрачности мазков кисти. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Дополнительно: при нажатии этой кнопки отображается диалоговое окно «Кисти», в котором можно установить дополнительные параметры кисти.
  • Стабилизатор: стабилизация мазка с помощью либо стабилизатора «Веревка», либо стабилизатора «Окно». При использовании первого инструмента потяните конец мазка с помощью «веревки», сглаживающей мазок, но позволяющей обеспечить острые углы при увеличении значений параметра Длина (Радиус) путем изменения направления ослабленной веревки. При использовании второго инструмента мазок сглаживается благодаря усреднению отобранных положений ввода в рамках окна с настраиваемым размером.
  • Режим наложения: используется для изменения характера взаимодействия цвета мазка с имеющимися цветами слоя. Выберите этот вариант во всплывающем меню.
  • Контроллер: определяет реакцию мазка кисти на различные входные значения.
    • Настройки кисти по умолчанию: используются параметры отклонения кисти и контроллера кисти, сохраненные для текущей выбранной кисти. Чтобы получить доступ, нажмите Дополнительно на панели инструментов.
    • Автоматически: обнаружено устройство ввода (планшет, устройство Force Touch, мышь), и кисть будет работать с этим устройством автоматически, используя параметры отклонения кисти, но не учитывая параметр контроллера кисти. На устройствах можно работать поочередно.
    • Нажим: мазок кисти реагирует на нажим только при работе с планшетом; при этом используются параметры отклонения кисти, но не учитывается параметр контроллера кисти.
    • Скорость: мазок кисти реагирует только на скорость движения мыши.

Для пиксельного инструмента «Кисть рисования» параметры кисти на контекстной панели инструментов включают перечисленное ниже.

  • Поток: используется для определения скорости наложения эффектов кисти (1 % — очень медленно, 100 % — мгновенно). Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Жесткость: используется для определения степени жесткости краев мазка, поставленного пиксельной кистью. При уменьшении процентного значения жесткость кисти снижается. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Принудительный нажим для регулирования размера: используется для регулирования размера мазка кисти посредством нажима в случае применения чувствительных к нажиму устройств. При выборе этого параметра настройки кисти по умолчанию отменяются.
  • Влажные края: при выборе этого параметра происходит скопление цвета вдоль контуров мазка, нанесенного пиксельной кистью, в результате чего возникает эффект акварельного рисунка.
  • Защитить альфа-канал: при выборе этого параметра нельзя рисовать на прозрачных областях текущего слоя.
