Корректировка цветового баланса

Корректировка «Цветовой баланс» предназначена для изменения соотношения отдельных цветов тонального диапазона.

Параметры

Параметры, которые могут быть изменены, описаны ниже.

  • Тональный диапазон: позволяет определять тона, которые должны быть изменены. Выберите этот вариант во всплывающем меню.
  • С помощью ползунков можно регулировать баланс определенных цветов в выбранном тональном диапазоне. Перетащите ползунок в сторону цвета, который хотите сделать более ярким.
  • Сохранить светимость: если этот параметр отключен (по умолчанию), то при применении этой корректировки исходное значение светлоты пикселей не будет учитываться. При выборе этого параметра корректировки цвета пикселей меняются в сравнении с исходным значением светлоты.
