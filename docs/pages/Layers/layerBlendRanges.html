Диапазоны смешивания слоев

Диапазоны наложения предназначены для определения диапазона цветов на текущем слое, которые смешиваются с цветами расположенных ниже слоев.

О диапазонах наложения

Диапазоны наложения позволяют определить, каким образом должно происходить наложение значений тонов верхнего и нижних слоев. Можно установить диапазон значений для тонов, которые должны быть изменены, а также отрегулировать уровень непрозрачности для определенного диапазона (от непрозрачного до прозрачного).

Диапазон смешивания выделенного и нижележащих слоев регулируется в диалоговом окне Диапазоны смешивания.

Можно изменить диапазон наложения для отдельных цветовых каналов в диалоговом окне.

О гамме наложения и сглаживании (только документы RGB)

В диалоговом окне «Диапазоны наложения» можно регулировать гамму наложения выделенного слоя. Благодаря этому вы можете работать в линейном цветовом RGB-пространстве (гамма 1.0), выполнять смешивание цветов в стандартном пространстве sRGB (гамма 2.2) или применять любую другую гамму до версии 3.0. Другими словами, вы можете полностью управлять взаимодействием тонов полупрозрачных объектов или объектов со сглаженными краями и цветов расположенных ниже слоев.

По умолчанию для слоев с текстом устанавливается гамма 1.45, а для слоев всех остальных типов — гамма версии 2.2 (смешивание цветов в стандартном пространстве sRGB). Эти параметры по умолчанию можно изменить в меню Установки (параметр «Инструменты»).

Сглаживание: это устранение эффекта «зубчатости» линий на пиксельной сетке. Эффект сглаживания достигается за счет добавления полупрозрачных пикселей вдоль линии для того, чтобы сделать переход от края линии к объектам фона более плавным. Эту область перехода иногда называют линейным градиентом сглаживания или областью покрытия сглаживания.

В диалоговом окне «Диапазоны наложения» можно отрегулировать параметры линейного градиента (покрытия) сглаживания для выделенного слоя.

Параметры

В диалоговом окне «Диапазоны наложения» доступны приведенные далее параметры.

  • Гамма: используется для регулирования гаммы наложения слоя.
  • Карта покрытия: используется для регулирования параметров линейного градиента сглаживания слоя.
  • Каналы: используется для выбора канала, который будет изменен при регулировке диапазона наложения. Выберите этот вариант во всплывающем меню.

Далее приведены параметры для элементов Диапазоны исходного слоя и Диапазоны базовой композиции, которые можно изменить.

  • График — используется для регулирования изменяемого тонального диапазона и степени непрозрачности в пределах определенного диапазона.
  • Вход: используется для установки горизонтального положения выделенного узла. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Выход: используется для установки вертикального положения выделенного узла. Для установки значения введите его непосредственно в текстовое поле или перетащите всплывающий ползунок.
  • Линейный: если выбран этот параметр (по умолчанию), то сегмент между двумя узлами будет иметь линейную форму (т. е. узлы на графике будут соединены с помощью прямых линий). Если этот параметр отключен, узлы будут соединены с помощью сглаженных кривых.
  • Сбросить: используется для того, чтобы вернуть графику вид по умолчанию (прямая линия между двумя узлами, расположенными в верхней части сетки).

При регулировании параметров графиков в диалоговом окне следует отметить следующее.

  • На графиках тональные значения представлены от наиболее темных (слева) до наиболее светлых (справа).
  • Объекты на выделенном слое становятся менее заметными при перемещении узлов на графике Диапазоны исходного слоя вниз.
  • Объекты на расположенных ниже слоях становятся более заметными при перемещении узлов на графике Диапазоны базовой композиции вниз.
Для изменения диапазонов смешивания, гаммы смешивания и линейного градиента сглаживания выполните перечисленные далее действия.
  1. Выделите слой на панели Слои, а затем нажмите Диапазоны наложения.
  2. Выберите нужные параметры в диалоговом окне.
  3. Закройте диалоговое окно.
Для изменения графика напрямую выполните перечисленные далее действия.

Выполните любое из перечисленных ниже действий.

  • Перетаскивайте узел по горизонтали для того, чтобы изменения затронули большее или меньшее количество пикселей.
  • Перетаскивайте узел по вертикали для того, чтобы влиять на видимость пикселей на участке с выбранным значением тона.
  • Чтобы добавить дополнительные узлы, нажмите точку на кривой.
  • Нажмите узел, который необходимо выбрать, а затем нажмите , чтобы его удалить.
Для изменения линейного градиента сглаживания выполните перечисленные далее действия.
  1. Нажмите пиктограмму Карта покрытия.
  2. На появившейся диаграмме выделите узел на линии профиля и установите его в новое положение путем перемещения по вертикали или горизонтали.
  3. Повторите эти действия для других узлов по мере необходимости.

В случае с более сложными профилями нажмите точку на линии профиля для добавления узла, который может быть размещен как любой созданный узел.

Для удаления сглаживания установите прямую горизонтальную линию профиля в верхней части диаграммы.

Для обратного преобразования линейного градиента сглаживания в линейный формат выполните перечисленные далее действия.
  1. Нажмите пиктограмму Карта покрытия.
  2. Во всплывающем диалоговом окне отобразившейся диаграммы нажмите Сбросить.
Для сохранения профиля карты покрытия выполните перечисленные далее действия.
  • Нажмите Сохранить профиль под диаграммой. Профиль отображается под диаграммой.
Для применения пользовательской карты профиля выполните перечисленные далее действия.
  1. Нажмите пиктограмму Карта покрытия.
  2. Выберите пиктограмму пользовательского профиля, которая находится ниже диаграммы. Диаграмма обновится, и отобразится выбранный профиль.
