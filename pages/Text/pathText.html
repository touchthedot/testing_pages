Текст по контуру

Текст вдоль контура представляет собой разновидность художественного текста и может располагаться вдоль линии, кривой или контура фигуры.

Операция размещения текста вдоль контура расширяет возможности для работы с текстовыми элементами композиции. Контур может быть создан из любой линии, кривой или фигуры, нарисованной с использованием любого инструмента для создания линий или фигур. К ним относятся, кроме прочего, такие инструменты: инструмент «Перо», инструмент «Карандаш» и инструмент «Эллипс». Единственным критерием является то, что линия, кривая или фигура должна быть создана на странице до наложения текста вдоль контура.

После преобразования линии, кривой или фигуры в контур, вдоль которого будет размещен текст, исходный объект удаляется. Если вы хотите сохранить исходный объект, рекомендуется сначала дублировать его.

В меню Текст выберите Вставить замещающий текст для придания тексту естественного вида во время работы над композицией.

Изменение текста вдоль контура

После создания контура можно менять положение его крайних точек с тем, чтобы растянуть или сжать сегмент контура, где будет размещен текст. Крайние точки сегмента контура обозначены оранжевыми треугольниками.

Если текст выходит за пределы сегмента, отмеченного крайними точками, он перейдет на второй контур (если таковой имеется) или перевернется на другую сторону и отразится в обратном направлении. В этом случае отобразятся дополнительные крайние точки контура, благодаря чему вы сможете по отдельности управлять новыми сегментами. Если для создания контура использовалась фигура замкнутой формы, например кольцо, можно будет работать со вторым контуром.

Кроме того, можно регулировать расстояние между текстом и контуром (то есть положение базовой линии) и направление текста вдоль контура.

Для сегментов контура можно устанавливать разные значения смещения базовой линии.

Изменение формы объекта с контуром

Можно изменить форму объекта с контуром в любой точке с помощью инструмента «Узел». Если для создания объекта с контуром использовался один из инструментов «Фигура», его форму можно менять, как если бы это была исходная геометрическая фигура. Положение текста, уже размещенного на контуре, будет пересчитано в зависимости от того, как меняется контур преобразуемого объекта.

Дополнительную информацию об изменении формы объекта с контуром см. в разделах Редактирование кривых и фигур и Рисование и редактирование фигур.

Для создания текста вдоль контура выполните перечисленные далее действия.
  1. Выделите ранее нарисованную линию, кривую или фигуру.
  2. На панели Инструменты выберите инструмент «Художественный текст».

  3. Поместите курсор рядом с контуром линии, кривой или фигуры.

    Курсор изменится. Это свидетельствует о том, что будет создан текст вдоль контура.

  4. Нажмите кнопку мыши или перетащите курсор по странице, чтобы сохранить размер текста по умолчанию или изменить его в соответствии с потребностями.

    Линия, кривая или фигура будет преобразована в контур, вдоль которого можно размещать текст.

  5. Выполните одно из перечисленных ниже действий.
    • Введите текст.
    • Вставьте предварительно скопированный текст.
    • В меню Текст выберите Вставить текст из файла. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

Либо выделите ранее нарисованную линию, кривую или фигуру, а затем в меню Слой выберите Преобразовать в контур для текста.

Для изменения размещения текста вдоль контура выполните перечисленные далее действия.

Выделите текст вдоль контура и выполните одно из перечисленных ниже действий.

  • Перетаскивая, меняйте положение одной или нескольких крайних точек на контуре.
  • На контекстной панели инструментов установите значение для параметра Базовая линия.
  • На контекстной панели инструментов нажмите Обратить контур для текста.

При активном инструменте «Текст» для применяется описанная далее логическая схема.

  1. Если используется при выбранных глифах (знаках), с глифов снимается выделение, но каретка (курсор) остается внутри текстового объекта. (Можно продолжать редактирование текста в текстовом объекте.)
  2. Если каретка находится внутри текстового объекта, но глифы не выбраны, использование приведет к удалению каретки и выделению текстового объекта. (Нажмите V для перехода к инструменту «Перемещение» и изменению положения текстового объекта.)

Изменение размера объекта с контуром

При изменении размера объекта с контуром можно контролировать приведенные далее условия.

  • Сохранение заданного размера текста, а также пересчет его положения вдоль контура.
  • Изменение масштаба текста при изменении размера контура.
Для того чтобы пересчитать положение текста, выполните перечисленные далее действия.

Выделите объект с контуром и выполните одно из перечисленных ниже действий.

  • Для того чтобы одновременно менять высоту и ширину, перетягивайте угловые маркеры объекта.

  • Для того чтобы менять высоту и ширину по отдельности, перетягивайте боковые маркеры объекта.

Для того чтобы изменить масштаб текста, выполните перечисленные далее действия.

  • Выделите объект с контуром и перетащите маркер масштабирования объекта (тянется от нижнего правого угла выделения).

Использование замещающего текста

Если вы выполняете проектирование без сохранения полной копии, можно использовать замещающий текст (заполнитель), благодаря которому вы сможете продолжать работу над проектом.

Для добавления замещающего текста выполните перечисленные далее действия.
  • Выделите объект, вдоль контура которого будет размещен текст, и в меню Текст выберите Вставить замещающий текст.
