Панель «Эффекты»

Применяйте эффекты слоя непосредственно к объектам или слоям с помощью панели «Эффекты».

О панели «Эффекты»

Эффекты слоя можно применять к текущему выделенному объекту или всему слою. Также эти эффекты являются полностью неразрушающими, что позволяет менять их настройки в любое время.

На панели доступны приведенные далее элементы управления и эффекты.

  • Непрозрачность заливки: используется для регулирования непрозрачности обводки и заливки объекта (слоя) без изменения уровня непрозрачности применяемых эффектов слоев. Это особенно важно для дальнейшего применения эффекта к художественному тексту или фигурам.
  • Размытие по Гауссу: используется для размытия объекта или слоя.
  • Внешняя тень: используется для добавления тени позади объекта или слоя.
  • Внутренняя тень: используется для добавления тени по внутреннему краю объекта или слоя.
  • Внешнее свечение: используется для добавления цветного свечения по внешнему краю объекта или слоя.
  • Внутреннее свечение: используется для добавления цветного свечения, которое направлено от внутреннего края (слоя).
  • Контур: используется для создания контура по краю объекта или слоя.
  • Трехмерность: используется для добавления освещения с целью создания иллюзии трехмерности.
  • Скос/тиснение: используется для добавления различных комбинаций светлых участков и теней.
  • Перекрытие цвета: используется для наложения сплошного цвета на объект или слой.
  • Градиентное перекрытие: используется для наложения линейного градиента на объект или слой.

Дополнительные расширенные элементы управления для эффектов наложения тени и эффектов трехмерности, скоса, тиснения и перекрытия доступны в меню Слой > Эффекты слоя.
