Скос/тиснение

Эффект «Скос/тиснение» используется для придания краям округлой формы и добавления теней с целью создания иллюзии трехмерной рельефности.

В зависимости от установленного типа скоса или тиснения закругленный край может находиться внутри или снаружи объекта, может быть выпуклой или вогнутой формы, а также на него может накладываться тень.

Для всех эффектов скоса и тиснения можно применять предустановленный или пользовательский профиль, который определяет угол скоса или тиснения.

Параметры

На панели Эффекты представлены указанные ниже параметры.

  • Непрозрачность: используется для регулирования степени прозрачности эффекта.
  • Радиус: используется для регулирования степени проявления эффекта.
  • Эффекты слоя: обеспечивает доступ к диалоговому окну Эффекты слоя, где представлены расширенные параметры и элементы управления.

По умолчанию при использовании панели «Эффекты» применяется эффект «Выпуклый рельеф». Для создания внутреннего или внешнего скоса и наложения основного эффекта тиснения следует использовать возможности диалогового окна «Эффекты слоя».

В диалоговом окне Эффекты слоя можно изменить приведенные далее расширенные параметры.

  • Тип: используется для создания внутреннего или внешнего скоса, а также наложения эффекта «Тиснение» или «Выпуклый рельеф». Выберите этот вариант во всплывающем меню.
  • Глубина: используется для установки глубины эффекта. Этот параметр можно привязать к радиусу или устанавливать независимо от него.
  • Смягчение: используется для размытия теней и светлых участков изображения.
  • Профиль: используется для определения способа падения света и прорисовывания неровностей, углублений и выступов, затеняемых в процессе.
  • Инвертировать: используется для инвертирования эффекта от применения профиля освещения.
  • Удалить профиль: используется для возврата настроек профиля по умолчанию, когда свет падал равномерно.
  • Направление: используется для обозначения положения источника света, тени или градиента. Для корректировки направления нажмите поле или перетащите указатель (и установите значение параметров Азимут и Возвышение автоматически).
  • Азимут: используется для определения направления источника света, тени или градиента.
  • Светлый участок: используется для установки режима наложения, цвета и непрозрачности для светлого участка. Режим наложения по умолчанию — «Экран».
  • Тень: используется для установки режима наложения, цвета и непрозрачности для темного участка. Режим наложения по умолчанию — «Умножение».
  • Масштабировать относительно объекта: при выборе этого параметра (он установлен по умолчанию) в случае изменения размера объекта эффект масштабируется пропорционально объекту. Если этот параметр не выбран, при изменении размера объекта масштаб эффекта остается неизменным.
  • Непрозрачность заливки: используется для установки непрозрачности содержимого слоя, которая не влияет на примененные эффекты.
Для того чтобы применить предустановку или пользовательский профиль, выполните перечисленные далее действия.
  1. Нажмите пиктограмму профиля.
  2. Выберите на всплывающей панели один из вариантов ниже.
    • Одну из пиктограмм стандартного профиля.
    • Пользовательский профиль. Нажмите точку на кривой для того, чтобы добавить узел, а затем перетащите узел или часть кривой для придания ей формы.
  3. Установите флажок в поле «Линейная» для создания заостренной кривой или снимите флажок для создания сглаженной кривой.
  4. Для того чтобы применить профиль, нажмите в любом месте за пределами всплывающей панели.
