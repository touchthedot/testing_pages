Инструмент «Заливка»

Инструмент «Заливка» позволяет настроить цвета заливки и линий, применяемые для векторных и текстовых объектов.

Хотя инструмент «Заливка» можно использовать для применения сплошного цвета к заливке или обводке объекта, однако его истинная сила заключается в возможности применять и изменять градиенты.

Можно также применить градиент к растровым слоям, корректировочным слоям и маскам слоя, воспользовавшись инструментом «Заливка».

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Контекст: используется для корректировки цвета заливки или обводки выбранного объекта.
  • Тип: используется для преобразования типа цвета объекта. Например, тип Нет можно изменить на тип Линейный.
  • Выделяйте, выбирайте и изменяйте сплошной или градиентный цвет объекта. Для этого необходимо нажать на образец цвета, чтобы отобразилась всплывающая панель. Дополнительную информацию о доступных параметрах см. в темах Панель «Цвет», Панель «Образцы» и Редактирование градиента.
  • Повернуть градиент: используется для поворота примененного градиента на 90°.
  • Обратный градиент: конечные узлы меняются местами. (Все промежуточные узлы также меняют свое положение соответственно.)
  • Сохранить коэффициент заливки: если этот параметр выключен (по умолчанию), можно изменять положение конечных узлов в индивидуальном порядке, что приводит к изменению коэффициента заливки. Если этот параметр включен, конечные узлы фиксируются, обеспечивая сохранение коэффициента заливки (т. е. при изменении положения одного узла автоматически обновится другой). Этот параметр оказывает воздействие только на эллиптическую и растровую заливки.
