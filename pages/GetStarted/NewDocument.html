Создание новых документов

Для создания новых документов можно использовать как предустановки, так и пользовательские параметры, определенные в соответствии с вашими конкретными потребностями. Для повышения эффективности рабочего процесса документы можно создавать непосредственно в iCloud Drive.

При создании нового документа вы указываете различные параметры для цветов, в том числе размеры страницы в зависимости от назначения документа. Эти параметры представлены ниже.

  • Тип: используется для определения назначения и конечного результата проекта (для быстрой настройки приведенных ниже параметров). Помимо печати готовых для вывода на печать изображений CMYK, вы можете работать с конкретными размерами фотопечати и определенными разрешениями экрана для отображения веб-страниц, а благодаря функции «Устройства» также можете создавать документы, соответствующие спецификациям iPad, iPhone и Nexus.
  • Предустановка страницы: используется для настройки размеров в зависимости от характера использования экрана и типа мобильных устройств, а также параметров фотопечати и печати документов по стандартам, принятым в США и Великобритании. Параметр «Пользовательский» позволяет определить, если для страницы указаны нестандартные размеры.
  • Единицы измерения документа: используется для отображения линеек и размеров объекта в пикселях, точках или физических единицах измерения.
  • Создать монтажную область: используется для автоматического создания одной монтажной области документа.
  • Формат цвета: используется для установки цветового режима RGB или «Серый» (8 или 16 бит), CMYK (8 бит) или Lab (16 бит).
  • Цветовой профиль: используется для установки цветовой гаммы для ранее выбранного формата цвета.
  • Прозрачный фон: если для этого параметра установлен флажок, фон страницы будет прозрачным.
  • Ширина / Высота страницы: используется для изменения этих значений с целью настройки пользовательского размера страницы. (Является альтернативой выбору описанного выше параметра «Предустановка страницы».)
  • Разрешение: используется для установки разрешения документа. Например, для получения профессионального качества печати установите разрешение 300 тчк/дюйм.
  • Масштабирование «Фактический размер»: используется для определения устройства, которое имитируется при представлении документа в фактическом размере (меню Вид). Этот параметр доступен только тогда, когда в качестве единиц измерения документа указаны пиксели или точки. Выберите этот вариант во всплывающем меню.
  • Портретная: если флажок не установлен, документ будет представлен в альбомной ориентации.
  • Учитывать поля: если для этого параметра установлен флажок, включено отображение полей страниц.
  • Левое поле / Правое поле / Верхнее поле/ Нижнее поле: используется для установки полей принтера, отображаемых в виде непечатаемых синих линий. Нажмите Использовать для поля настройки принтера для того, чтобы применить настройки принтера по умолчанию.

При создании нового документа точный размер документа на экране определяется настройками параметра «Единицы измерения документа». Например, если выбраны пиксели, при экспорте и во время просмотра на экране в масштабе 100 % документ будет отображаться в своем точном размере в пикселях. Аналогичным образом, если установлены физические единицы измерения (например, миллиметры), документ будет представлен в точном размере печати в масштабе 100 %.

Кроме того, новый документ может быть создан на основе содержимого буфера обмена.

Для создания нового документа выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Создать.
  2. В диалоговом окне выберите параметры документа.
  3. Нажмите ОК.
Для создания нового документа в iCloud Drive выполните перечисленные далее действия.
  1. В меню Файл выберите пункт Открыть.
  2. На боковой панели в iCloud Drive выберите папку Affinity Designer, а затем пункт Новый документ.
  3. В диалоговом окне выберите параметры документа.
  4. Нажмите ОК.
