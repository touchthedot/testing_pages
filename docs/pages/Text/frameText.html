Текстовая область

Текстовая область — это лучший инструмент для создания абзацев с фиксированной структурой и макетом. Если необходимо представить текст в столбцах, текстовая область — это идеальное решение.

Можно заполнить содержимое рамки текстовой областью, поставив курсор в нарисованной рамке и напечатав текст. Если текста слишком много и он выходит за нижние границы области, можно или уменьшить размер шрифта текста на контекстной панели инструментов, или увеличить размер текстовой области. С помощью обоих способов можно добиться размещения текста в пределах рамки.

Текстовую область можно создать внутри прямоугольных (квадратных) рамок с помощью инструмента «Текстовая область» или внутри какой-либо кривой или замкнутой фигуры.

В меню Текст выберите Вставить замещающий текст для придания тексту естественного вида во время работы над композицией.

Чтобы создать текстовую область, выполните приведенные далее действия.

Выберите инструмент «Текстовая область».

  1. Перетащите курсор по странице. Таким образом задается начальный размер текстовой области.
  2. Выполните одно из перечисленных ниже действий.
    • Введите текст.
    • Вставьте (V) предварительно скопированный текст.
    • В меню Текст выберите Вставить текст из файла. Во всплывающем диалоговом окне перейдите к файлу, выберите его и нажмите кнопку Открыть.

При копировании текста из внешних приложений воспользуйтесь функцией вставки без форматирования (меню Правка) для сохранения «чистого» неформатированного текста.

Чтобы создать текстовую область из кривых или фигур, выполните приведенные далее действия.
  1. Выделите ранее нарисованную кривую или фигуру.
  2. На панели Инструменты выберите инструмент «Текстовая область».

  3. Поместите курсор рядом с контуром линии или внутри фигуры.

    Курсор изменится. Это свидетельствует о том, что будет создана текстовая область внутри фигуры.

  4. Нажмите кнопку мыши, чтобы преобразовать кривую или фигуру в текстовую область внутри фигуры.
  5. Введите текст.

Либо выделите ранее нарисованную линию, кривую или фигуру, а затем в меню Слой выберите Преобразовать в текстовую область.

Клавиши-модификаторы

При использовании инструмента «Текстовая область» можно использовать приведенные далее клавиши-модификаторы.

  • : ограничение пропорций области в момент создания (квадрат) или при изменении размера.
  • : изменение размера области относительно ее центра.
  • С помощью клавиши можно вращать размещенную текстовую область вокруг противоположного маркера.
  • С помощью нажатия правой кнопки мыши можно вращать размещенную текстовую область вокруг противоположного маркера.

При активном инструменте «Текст» для применяется описанная далее логическая схема.

  1. Если используется при выбранных глифах (знаках), с глифов снимается выделение, но каретка (курсор) остается внутри текстового объекта. (Можно продолжать редактирование текста в текстовом объекте.)
  2. Если каретка находится внутри текстового объекта, но глифы не выбраны, использование приведет к удалению каретки и выделению текстового объекта. (Нажмите V для перехода к инструменту «Перемещение» и изменению положения текстового объекта.)
  3. При выбранном текстовом объекте нажмите , чтобы отменить выделение текстового объекта.

Изменение размера текстовых областей

При изменении размера области можно контролировать приведенные далее условия.

  • Сохранение заданного размера текста, а также пересчет его положения в области.
  • Изменение масштаба текста при изменении размера области.
  • Лишнее пространство в нижних границах области следует удалять после изменения размера.
Для того чтобы пересчитать положение текста, выполните перечисленные далее действия.

Выделите текстовую область и выполните одно из перечисленных ниже действий.

  • Для того чтобы одновременно менять высоту и ширину, перетягивайте угловые маркеры объекта.

  • Для того чтобы менять высоту и ширину по отдельности, перетягивайте боковые маркеры объекта.

Для того чтобы изменить масштаб текста, выполните перечисленные далее действия.

  • Выделите текстовую область и перетащите маркер масштабирования объекта (тянется от нижнего правого угла выделения).

Чтобы растянуть область по размеру текста, выполните приведенные далее действия.
  • Дважды щелкните маркер края, расположенный по центру нижней границы текстовой области. Лишнее пространство между последней строкой текста и нижним краем области сбудет удалено.

Выравнивание текста в текстовых областях

Выравнивание по горизонтали — это атрибут абзаца, который можно изменить на контекстной панели инструментов «Текст». Выравнивание текста по вертикали в области — это атрибут области. Эта функция подходит для выравнивания текста по центу вместе со смежными графическими элементами.

Чтобы выровнять текстовую область по вертикали, выполните приведенные далее действия.
  • В меню Текст выберите вариант в подменю Выравнивание по вертикали.

Использование замещающего текста

Если вы выполняете проектирование без сохранения полной копии, можно использовать замещающий текст (заполнитель), благодаря которому вы сможете продолжать работу над проектом.

Для добавления замещающего текста выполните перечисленные далее действия.
  • Выделите текстовую область и в меню Текст выберите Вставить замещающий текст.
