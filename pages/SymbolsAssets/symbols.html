Символы

Символ — это интеллектуальный объект, который может быть многократно размещен в документе. При изменении любого из этих символов на странице автоматически и незамедлительно обновляются все экземпляры этого символа.

Символы

Символы обеспечивают высокоэффективный способ работы, подходящий для «повторяющихся» элементов, подлежащих изменению, например логотипов, фирменной символики и кнопок. Благодаря отсутствию необходимости изменять один и тот же элемент несколько раз, символ может быть изменен всего один раз; при этом все экземпляры этого символа будут обновлены автоматически, даже в нескольких монтажных областях. Кроме того, существует возможность изменять любой экземпляр символа независимо от других.

Связывание

При создании нескольких экземпляров символа автоматически создаются связи символа. Связывание имеет фундаментальное значение для функциональности символа и, по умолчанию, обеспечивает автоматическое отображение изменений символа во всех экземплярах.

Синхронизация

Синхронизация позволяет вносить изменения во все символы, тогда как отсутствие синхронизации означает, что все последующие изменения ограничиваются текущим объектом и не применяются к другим экземплярам символа, пока снова не будет включена синхронизация.

Если больше нет необходимости, чтобы экземпляр символа оставался символом, он может быть отсоединен (преобразован в стандартный объект), что не повлияет на синхронизацию. Если больше нет необходимости, чтобы все экземпляры символа оставались символами, их все можно снова преобразовать в стандартные объекты.

Изменение

К символам можно применять такие же изменения, как и к любому стандартному объекту. Можно выполнять приведенные ниже действия.

  • Изменять форму, обводку/заливки и добавлять эффекты/корректировки. Обратите внимание, что глобальные цвета также могут использоваться для изменения цвета объекта независимо от символов.
  • Добавлять «несимвольные» объекты к символу перетаскиванием в группу символа на панели «Слои».

Ключевой функцией символов является возможность изменять атрибуты объекта независимо друг от друга. это означает, что при отсутствии синхронизации отдельный атрибут (размер, заливка, эффект, шрифт и т. д.) может быть изменен, тогда как остальные атрибуты сохранят функциональность символов.

В качестве визуальной подсказки символы отмечены вертикальной оранжевой чертой рядом с записями об их слоях на панели «Слои». Если отображенный в виде символа объект был изменен при отключенной синхронизации, отображается пунктирная оранжевая черта.

Панель «Символы» скрыта по умолчанию. Ее отображение можно включить, перейдя в меню Вид > Студия при работе в режимах «Рисование» или «Пиксельный».

Для создания символа выполните перечисленные далее действия.
  1. Выберите объект или группу на странице или с помощью панели Слои.
  2. На панели Символы нажмите Создать.

Нельзя создать символ из объекта, присутствующего в существующем символе.

Для создания экземпляра символа

выполните одно из перечисленных ниже действий.

  • На панели Символы перетащите выбранный символ на страницу.
  • Дублируйте уже присутствующий на странице символ.

Когда на странице уже присутствует несколько экземпляров, связывание символов позволяет выполнить синхронизацию между всеми объектами.

Для изменения экземпляра символа выполните указанные ниже действия.
  • На панели Слои разверните выбранную запись символа, затем измените содержащиеся в ней объекты.

Если символ, содержащий несколько объектов, используется для создания экземпляров символа на странице, группировка объектов осуществляется по умолчанию. В результате необходимо изменить содержащиеся в нем объекты для применения изменений ко всем экземплярам символа.

Для отключения синхронизации выполните указанные ниже действия.
  • На панели Символы нажмите Синхронизировать, чтобы отключить; таким образом, каждый объект может быть изменен отдельно. При включении изменения синхронизируются во всех экземплярах символа.
Для отсоединения объектов от символов выполните указанные ниже действия.
  1. Выберите символ на странице или с помощью панели Слои.
  2. На панели Символы нажмите Открепить.

При отсоединении символ снова преобразуется в стандартный объект, но синхронизация остается включенной.

Для удаления символа выполните указанные ниже действия.
  • На панели Символы нажмите символ и выберите Удалить символ.

Все экземпляры символа снова преобразуются в стандартные объекты.
