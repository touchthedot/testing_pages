Панель «32-битный просмотр»

На панели «32-битный просмотр» содержатся средства управления экспозицией и гаммой, которые можно настроить, для предварительного просмотра широкого диапазона тонов 32-битного документа без изменения тональности. Они могут пригодиться, в частности, для рабочих процессов с применением чисел с плавающей точкой при необходимости избежать потерь информации (например, 3D-рендеринг), когда требуется вносить изменения в изображение без уменьшения доступного тонального диапазона вследствие тонального сжатия или корректировок.

О панели «32-битный просмотр»

На панели можно откорректировать приведенные далее параметры.

  • Экспозиция: используется для увеличения или уменьшения экспозиции в предварительном просмотре изображения, с помощью которого можно получить лучшее наглядное представление о тональных диапазонах светлых участков или теней для редактирования.
  • Гамма: используется для изменения значения коррекции гаммы.
  • Трансформация отображения (параметр OpenColorIO): используется для переключения между неуправляемым линейным цветовым пространством RGB и трансформацией отображения в другое цветовое пространство:
    • Трансформация дисплея ICC: используется для трансформации с помощью назначенного монитору профиля ICC.
    • Не управляется: трансформация не применяется.
    • OCIO-трансформация отображения: используется для выбора профиля устройства вывода для предварительного просмотра документа в этом цветовом пространстве. Как правило, необходимо, чтобы этот профиль соответствовал профилю отображения выбранной панели (например, P3 или Rec.2020 для особым образом отпрофилированных мониторов).

Для возможности пользоваться параметрами Трансформация отображения необходим действительный конфигурируемый файл OpenColorIO и заданный каталог. Дополнительную информацию см. в разделе Использование OpenColorIO.

Если вам необходимо выполнить тональную компрессию или изменить тональность изображения с целью сжатия тонального диапазона, скорее всего, не потребуется использовать эту панель.
