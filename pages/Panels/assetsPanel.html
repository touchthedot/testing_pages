Панель «Ресурсы»

Панель «Ресурсы» предназначена для хранения элементов проектирования, к которым можно получить доступ из любого документа Affinity Designer. По умолчанию эта панель заполняется элементами проектирования интерфейса для разработки проектов для iOS.

Панель «Ресурсы» по умолчанию скрыта. Ее отображение можно включить, перейдя в меню Вид > Студия при работе в режимах «Рисование» или «Пиксельный».

Ресурсы перечисляются на панели «Ресурсы» в порядке добавления. Однако можно вручную изменить порядок ресурсов на панели, перетаскивая их при удержании клавиши .

Параметры

На панели доступны приведенные далее параметры.

  • Категория: используется для отображения текущей активной категории. Выберите этот вариант во всплывающем меню.
  • Подкатегория: используется для отображения всех ресурсов в этот подкатегории. Нажмите заголовок подкатегории, чтобы свернуть или развернуть подкатегорию.
  • Поиск: используется для сокращения количества ресурсов на панели до тех, чьи названия совпадают с критериями поиска. Вводите текст непосредственно в текстовое поле.

В установках панели доступны приведенные далее параметры.

  • Создать новую категорию: используется для добавления новой, пользовательской категории ресурсов на панель.
  • Переименовать категорию: используется для переименования текущей выбранной категории.
  • Удалить категорию: используется для удаления текущей выбранной категории (и всех ее ресурсов) из панели.
  • Создать подкатегорию: используется для добавления новой подкатегории в текущую выбранную категорию.
  • Импортировать ресурсы: используется для добавления новой категории на панель, предварительно заполненную подкатегориями и ресурсами.
  • Экспортировать ресурсы: используется для создания файла *.afassets, содержащего структуру ресурсов и подкатегории, из текущей выбранной категории.
  • Показать в виде списка: если этот параметр выключен (по умолчанию), ресурсы отображаются в виде пиктограмм в сетке со всплывающими подсказками, отображающими их имена. Выбранные ресурсы отображаются в виде пиктограмм с именами в списке.

В меню параметров подкатегории доступны приведенные далее параметры.

  • Переименовать: используется для переименования подкатегории.
  • Удалить: используется для удаления подкатегории (и всех ее ресурсов) из панели.
  • Переместить вверх: используется для перемещения подкатегории на позицию выше предыдущей подкатегории, чтобы она находилась выше в списке на панели. Если этот параметр неактивен, подкатегория находится в самой верхней позиции панели.
  • Переместить вниз: используется для перемещения подкатегории на позицию ниже следующей подкатегории, чтобы она находилась ниже в списке на панели. Если этот параметр неактивен, подкатегория находится в самой нижней позиции панели.
  • Добавить из выделенного: используется для добавления всех текущих выделенных на странице объектов в подкатегорию в качестве ресурсов. Сгруппированные объекты добавляются как один ресурс.
