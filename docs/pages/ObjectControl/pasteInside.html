Вставить внутри

С помощью этой функции можно вставить объект (или несколько выделенных объектов) внутри другого объекта.

Для копирования и вставки выполните перечисленные далее действия.
  1. Выделите один или несколько объектов.
  2. В меню Правка выберите пункт Вырезать.*
  3. Выделите объект, который будет выступать в качестве родительского объекта.
  4. В меню Правка выберите пункт Вставить внутри.

* Либо же объект можно скопировать (а не вырезать). В меню Правка выберите пункт Копировать.
