Инструмент «Вид»

Инструмент «Вид» используется для перемещения видимой части документа в представлении документа.

Контекстная панель инструментов «Вид» также содержит инструменты для изменения уровня масштабирования документа.

Можно установить масштаб «По размеру», дважды щелкнув по значку инструмента «Вид» на панели инструментов.

Клавиша быстрого доступа к инструменту : H

Панорамирование горячей клавишей:

Панорамирование ограничено областью видимости страницы, независимо от текущего уровня масштабирования.

Параметры

Параметры масштабирования можно изменить на контекстной панели инструментов.

  • Установите уровень масштаба. Выберите этот вариант во всплывающем меню.
  • Перетащите ползунок, чтобы отрегулировать уровень масштабирования.
  • Установите единицу измерения. Выберите этот вариант во всплывающем меню.
