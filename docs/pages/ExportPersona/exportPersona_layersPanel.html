Панель «Слои» (режим «Экспорт»)

На панели «Слои» в режиме «Экспорт» можно создавать фрагменты из целых монтажных областей, слоев, групп или выделенных объектов.

О панели «Слои»

На этой панели можно работать с монтажными областями, слоями, группами или объектами композиции, которые необходимо экспортировать. Вы можете выбирать и преобразовывать их во фрагменты на этой же панели. С помощью параметра «Создать фрагмент» можно создавать фрагменты для панели «Фрагменты», откуда затем экспортируются растровые изображения.

Параметры

На панели доступны приведенные далее параметры.

  • Видимый во время экспорта*. Если этот параметр включен, элемент виден и может быть экспортирован. Если параметр выключен, элемент скрыт и не может быть экспортирован.
  • Сделать элемент видимым*. Если этот параметр включен, элемент виден. Если параметр выключен, элемент скрыт. Экспорт элемента возможен в обоих случаях.
  • Редактировать все слои: этот параметр используется для выделения всех объектов во всех слоях (а не только в текущем).
  • Создать фрагмент. Этот параметр используется для создания фрагмента из выделенной монтажной области, слоя, группы или объекта, которые могут быть экспортированы.

Если для родительского объекта соответствующий параметр видимости выключен, то параметры видимости для дочерних объектов могут отображаться как частично неактивные.
