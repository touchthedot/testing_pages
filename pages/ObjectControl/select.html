Выделение объектов

Прежде чем перемещать и изменять векторные объекты, их необходимо выделить.

Выделить один или нескольких объектов можно различными способами. Если объект уже выделен, можно быстро выделить следующий или предыдущий объект в z-последовательности (стеке).

«Редактировать все слои» и «Выделить все»

Действие, выполняемое при использовании команды «Выделить все», зависит от настройки параметра «Редактировать все слои» на панели «Слои».

  • Если этот параметр включен, при использовании этой команды будут выделены все объекты во всех слоях и подслоях.
  • Если этот параметр отключен, будут выделены только объекты на текущем слое. Объекты на подслоях выделены не будут.

По умолчанию параметр «Редактировать все слои» включен, но его можно отключить в любой момент.

Для включения/выключения параметра «Редактировать все слои» выполните указанные ниже действия.
  • На панели Слои нажмите Редактировать все слои.
Чтобы выделить объект, выполните указанные ниже действия.

Выполните одно из перечисленных ниже действий.

  • Выберите инструмент «Перемещение» и нажмите на объект или группу на странице, чтобы выделить их.
  • На панели Слои нажмите запись объекта.
  • Если объект уже выделен, в меню Выделить выберите параметр Выделить следующий или Выделить предыдущий.

С помощью параметра Выделить следующий / предыдущий в качестве изначально выделенного объекта можно выделить только те объекты, которые расположены на одном слое или в пределах одной группы.

Выделение нескольких объектов.

Выполните одно из перечисленных ниже действий.

  • Выберите инструмент «Перемещение» и, удерживая , поочередно нажмите на каждый объект на странице, чтобы выделить их.
  • Выберите инструмент «Перемещение» и перетащите его, чтобы нарисовать область выделения вокруг объектов. *
Выделение всех объектов на слое.

Выполните одно из перечисленных ниже действий.

  • В меню Выделить выберите Выделить все.
  • На панели Слои щелкните по слою или подслою.
Для того чтобы выбрать объекты (или группы) в группе, выполните приведенные далее действия.
  • Дважды нажмите на объект. Если объект принадлежит группе, сначала будет выделена группа. Снова дважды нажмите кнопку мыши, чтобы выбрать объект.

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать для выделения объектов при работе с инструментом «Перемещение».

  • : используется для выделения нескольких объектов путем их поочередного нажатия.
  • : используется для выделения объектов, расположенных непосредственно позади текущего выделенного объекта.
  • : используется для выделения объектов, которые лишь частично охвачены областью выделения. *
  • По щелчку правой кнопкой мыши выделяются объекты, которые лишь частично охвачены областью выделения. *

* Можно поменять местами операцию выделения в виде области и операцию клавиши-модификатора, установив соответствующие параметры интерфейса пользователя в диалоговом окне Установки.

К выделенным объектам можно применять параметр Фиксация / отмена фиксации на панели Слои или в меню Слой.
