Просмотр

Существуют различные возможности просмотра документов, обеспечивающих максимальную гибкость при создании композиций.

Режимы просмотра

Каждый режим просмотра имеет две отличительные характеристики:

  • каким образом объекты отображаются на экране;
  • как представлена страница проекта.

Режимы

Векторный

Нарисованные объекты по умолчанию отображаются как векторные. Это означает, что, независимо от текущего уровня масштаба, объекты (и примененные эффекты) всегда отображаются с плавными векторными краями и переходами.

Пиксельный

В процессе проектирования векторные объекты отображаются, как состоящие из отдельных пикселей. С помощью этого режима вы получите точное представление о том, как будет выглядеть созданная вами композиция в режиме просмотра изображений после экспорта.

Пиксельный (Retina)

Этот режим аналогичен описанному выше пиксельному режиму, но используется для просмотра на дисплеях с высоким разрешением и разрешением Retina.

Контур

В процессе проектирования объекты на странице отображаются исключительно в виде контуров (без заливки или обводки).

Вид

Единое представление

Представление в выбранном режиме изолированно на одной странице.

Комбинированный вид

Представление двух режимов на одной странице одновременно. Можно менять положение разделителя-ползунка для отображения разных областей композиции в обоих режимах.

Обрезать по холсту

Параметр «Обрезать по холсту» (установлен по умолчанию) ограничивает обзор документа. Отображаются только объекты, размещенные на странице. Если объекты выходят за пределы страницы, область на странице останется видимой, а область на панели вставки будет скрыта. Объекты, которые полностью находятся на панели вставки, будут скрыты.

Если параметр Обрезать по холсту отключен, панель вставки остается видимой (слегка менее затененной), как и все расположенные на ней объекты.

Параметр «Обрезать по холсту» можно использовать в любом из представленных выше режимов просмотра.

Этот вариант недоступен в документах, содержащих монтажные области.

Для отображения определенных режимов выполните перечисленные далее действия.

Выполните одно из перечисленных ниже действий.

  • Нажмите Пиксельный режим просмотра для отображения векторных композиций в виде отдельных пикселей.
  • Нажмите Пиксельный режим просмотра (Retina) для отображения векторных композиций в виде отдельных пикселей. Используется для дисплеев с поддержкой разрешения Retina.
  • Нажмите Контурный режим просмотра для отображения композиции исключительно в виде контуров. В сложных проектах его можно использовать для упрощения нацеливания незафиксированных и зафиксированных объектов с целью выделения.
  • Прежде чем вернуться в векторный режим, убедитесь в том, что со всех значков снято выделение.

Кроме того, можно переключаться между режимами просмотра с помощью меню Вид (подменю Режим просмотра).

Для работы в комбинированном виде выполните перечисленные далее действия.
  1. В меню Вид перейдите в подменю Режим просмотра и выберите параметр Комбинированный вид.

  2. Нажмите разделитель-ползунок с одной стороны.

    Метка разделителя-ползунка отображается черным цветом со стороны нажатия (иначе отображается серым цветом).

  3. На панели инструментов или в меню Вид выберите режим.

Полноэкранный режим отображения

Перейти в полноэкранный режим просмотра очень просто с помощью кнопки в окне, параметра меню или комбинации клавиш.

Переход в полноэкранный режим.

Выполните одно из перечисленных ниже действий.

  • Нажмите кнопку «Полноэкранный режим» в верхней части окна (в операционной системе OS X Yosemite это зеленая кнопка, расположенная в левой верхней части экрана).
  • В меню Окно выберите пункт Включить/выключить полноэкранный режим.

Создать вид

Время от времени может пригодиться просмотр композиции с разными уровнями масштабирования одновременно. Для этого необходимо открыть текущий проект в области «Создать вид». Впоследствии можно установить для вида разные уровни масштабирования. Изменения, внесенные в любой из видов, дублируются в других.

Для области «Создать вид» также можно установить разные режимы просмотра и масштабирования.

Для открытия текущего проекта в области «Создать вид» выполните перечисленные далее действия.
  • В меню Вид выберите пункт Создать вид. Проект откроется на новой вкладке или в новом окне, в названии которых будет указано числовое обозначение.
Переключение между открытыми проектами и используемым видом.

Выполните одно из перечисленных ниже действий.

  • Перейдите на вкладку проекта или вида.
  • Нажмите заголовок окна проекта или вида в плавающем режиме.
  • В меню Вид выберите проект или вид в подменю Виды.
