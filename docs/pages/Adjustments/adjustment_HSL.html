Корректировка HSL

Настройте или даже полностью измените цвета на изображении путем регулирования оттенка, насыщенности и яркости (светлоты).

Параметры

В диалоговом окне можно изменить приведенные далее параметры.

  • Канал: позволяет изменить все цвета сразу (Основной) или выбрать отдельный набор цветов (например, Оттенки желтого).
  • HSV: если флажок для этого параметра установлен, то вместо цветовой модели HSL (оттенок, насыщенность, яркость) используется модель HSV (оттенок, насыщенность, значение). Функции ползунков «Насыщенность» и «Яркость» отличаются в зависимости от того, какая модель применяется.
  • Сдвиг оттенка: используется для регулирования оттенка цвета пикселей на изображении. Перетаскивайте нижний ползунок для того, чтобы менять цвета по всему спектру. (См. примечание ниже.)
  • Сдвиг насыщенности: используется для регулирования интенсивности цветов на изображении. Для уменьшения интенсивности цвета перетащите ползунок влево, а вправо — для увеличения интенсивности.
  • Сдвиг яркости: используется для регулирования общей яркости изображения. Для уменьшения яркости перетащите ползунок влево, а вправо — для ее увеличения.

Почему существуют два спектра для параметра «Сдвиг оттенка»?

Верхний спектр содержит цвета пикселей исходного изображения, где бирюзовый цвет выступает в качестве средней точки настройки гаммы. Вы не можете изменить этот спектр путем перемещения ползунка.

Верхний спектр позволяет оценить степень цветового сдвига пикселей изображения после изменения положения нижнего ползунка. До корректировки спектры идентичны.

Если, например, вы перетащите ползунок вправо, зеленая область нижнего спектра окажется под средней бирюзовой точкой. Все пиксели бирюзового цвета на изображении станут зелеными. Все остальные пиксели изображения изменятся соответственно (например, синий станет бирюзовым, а зеленый — оранжевым).

Для создания обесцвеченного (черно-белого) изображения установите ползунок Сдвиг насыщенности в крайнее левое положение.
