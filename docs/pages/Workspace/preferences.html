Установки

В разделе «Установки» находится целый ряд прочих параметров, применяемых в приложении. Их можно использовать для настройки собственного стиля работы.

Общие параметры
  • Выберите, нужно ли восстанавливать последний сеанс при запуске приложения.
  • Выберите, нужно ли открывать файлы PSD с редактируемыми текстовыми объектами.
  • Выберите, нужно ли показывать или скрывать разрешения файлов.
  • Выберите, нужно ли создавать файлы SVG при копировании объектов для улучшения взаимодействия между платформами.
  • Можно предотвратить сохранение пиктограммы вместе с документом.
  • Выберите, нужно ли перезаписывать импортированные файлы PSD при использовании команды Сохранить вместо создания файла *.afdesign. Прежде чем выбирать этот параметр, рекомендуется прочесть предупреждение, указанное в диалоговом окне.
  • Язык: используется для установки языка пользовательского интерфейса независимо от операционной системы. Выберите этот вариант во всплывающем меню.
Цветовые параметры
  • Можно настроить цветовые профили RGB, 32-битный RGB, CMYK, «Оттенки серого» и LAB по умолчанию для применения в будущих документах. Выберите варианты во всплывающих меню.
  • Метод преобразования: используется для выбора метода преобразования изображений. Выберите этот вариант во всплывающем меню.
  • Выберите, нужно ли применять компенсацию черной точки при открывании изображения.
  • Выберите, нужно ли преобразовывать цветовое пространство открытого файла в рабочее цветовое пространство, а также определите, выводить ли предупреждение о подобном преобразовании.
  • Выберите, нужно ли выводить предупреждение при назначении рабочего профиля неотпрофилированному изображению.
  • Выберите файл конфигурации OpenColorIO.
  • Выберите папку поиска OpenColorIO.
  • Выберите, нужно ли преобразовывать цветовые пространства с помощью OCIO на основе имени файла, а также определите, выводить ли предупреждение о подобном преобразовании.
Параметры производительности
  • Предел использования ОЗУ: используется для установки предпочтения касательно оптимизации производительности приложения при выполнении проектов пользователя.
  • Предупреждение об использовании места на диске при: используется для выбора предала, при превышении которого пользователю начнет выводиться предупреждение об использовании места на диске.
  • Предел отмены: используется для выбора длительности истории, к которой у пользователя будет доступ.
  • Качество просмотра: используется для выбора метода вывода изображения при внесении изменений. Выберите этот вариант во всплывающем меню.
  • Выберите, нужно ли сглаживать градиенты во время работы над проектами для увеличения производительности.
  • Выберите параметр обрезки для оптимизации производительности.
  • Интервал восстановления файла: используется для выбора интервала сохранения временных данных для открытых документов. Если произошел сбой в работе приложения, то при его повторном запуске пользователю будет предложено восстановить документ.
  • Отображение: используется для выбора аппаратного ускорения (OpenGL или Metal) или программного ускорения при снижении производительности компьютера. Ускорение Metal используется для Sierra macOS (10.12.x) и выше. Если в компьютере имеется встроенный графический процессор, то в случае выбора параметра Использовать только встроенный графический процессор доступ ко всем дополнительным дискретным видеокартам будет запрещен. Таким образом, снижается потребление электроэнергии и продлевается время работы аккумулятора (это полезная функция для не подключенных к сети ноутбуков MacBook Pro).
  • Модуль рендеринга: используется для выбора метода рендеринга. Выберите этот вариант во всплывающем меню.
    • По умолчанию: используется для применения метода рендеринга по умолчанию (обычно с помощью установленной графической платы).
    • Адаптер графического дисплея: указанное здесь название зависит от установленной графической платы и ее драйвера. Если у вас установлено несколько графических плат, здесь может быть представлено несколько вариантов.
    • WARP: используется платформа Windows Advanced Rasterization Platform. Если у вас возникли проблемы с производительностью при использовании параметра по умолчанию, попробуйте выбрать этот вариант.
  • Рендеринг Retina: используется для выбора метода рендеринга. Выберите этот вариант во всплывающем меню.
    • Автоматически (наилучший способ): используется для рендеринга сначала как на устройствах без экрана Retina, а затем как на устройствах с экраном Retina для обеспечения сбалансированной производительности и качества.
    • Низкое качество (самый быстрый способ): используется исключительно для рендеринга, как на устройствах без экрана Retina, для обеспечения максимального уровня производительности, но с потерей качества.
    • Высокое качество (самый медленный способ): используется исключительно для рендеринга, как на устройствах с экраном Retina, для обеспечения высокого качества, но с возможным снижением производительности.
Параметры интерфейса пользователя
  • Уровень серого в фоне: используется для регулирования уровня оттенков серого на панели вставки, окружающей страницу.
  • Уровень серого фона монтажной области: используется для регулирования уровня оттенков серого на панели вставки, окружающей монтажные области.
  • Гамма интерфейса: используется для регулирования контрастности пользовательского интерфейса.
  • Размер шрифта элементов интерфейса: используется для необязательного увеличения размера шрифта элементов интерфейса.
  • Стиль интерфейса пользователя: отображение интерфейса пользователя в темном или светлом стиле.
  • Задержка подсказки: используется для установки временного промежутка перед появлением подсказки при наведении курсора на элемент интерфейса.
  • Количество десятичных знаков для типов единиц измерения: используется для регулирования числа десятичных знаков, которые допускаются для каждой единицы измерения документа.
  • Выберите, нужно ли зафиксировать импортированное изображение в виде фонового слоя или оставить его незафиксированным.
  • Выберите, как будет отображаться ширина (или толщина) линии: в точках или в единицах измерения документа.
  • Выберите, как будет отображаться текст: в точках или в единицах измерения документа.
  • Выберите, будет ли в курсоре при выборе кисти отображаться предпросмотр подлежащих размещению пикселей.
  • Выберите, будет ли на курсоре при выборе кисти отображаться перекрестие для улучшения нацеливания.
  • Выберите, будут ли значки отображаться с оттенками серого.
  • Выберите, нужно ли разворачивать содержимое слоя для отображения записи выделенного объекта, когда этот объект выделен на странице.
  • Выберите, нужно ли активировать поддержку сенсорной панели.
Параметры инструментов
  • Размер маркера инструмента: используется для уменьшения или увеличения маркеров ограничительной рамки содержимого выделенного слоя и узлов (и маркеров) кривой.
  • Выберите, будут ли после нажатия и щелчка мышью отображаться контекстные меню (альтернатива нажатию правой кнопки многокнопочной мыши).
  • Выберите, нужно ли включать контекстное меню Force Touch; при включении доступ к меню осуществляется посредством сильного нажатия трекпада, совместимого с Force Touch.
  • Выберите, нужно ли при частичном покрытии объекта областью выделения добавлять его к выделенному, или же объект должен быть окружен областью выделения.
  • Выберите, будет ли средняя кнопка мыши выполнять прокрутку или масштабирование.
  • Выберите, нужно ли при перетаскивании инструмента «Масштаб» увеличивать область вокруг курсора (масштабирование перетаскиванием), или масштабировать с помощью выделяющей рамки.
  • Выберите, нужно ли поворачивать холст при использовании трекпада.
  • Ограничение пропорций для инструмента перемещения: используется для управления ограничивающим поведением по умолчанию при изменении размера объектов с помощью инструмента «Перемещение». Выберите этот вариант во всплывающем меню.
  • Создать текст с гаммой наложения: используется для управления гаммой наложения по умолчанию, которая применяется к тексту при его создании.
  • Расстояние перемещения курсорными клавишами: используется для установки расстояния сдвига, на которое перемещается объект при использовании клавиш со стрелками. Выберите единицы измерения во всплывающем меню.
  • Модификатор расстояния перемещения курсорными клавишами: используется для установки расстояния сдвига, на которое перемещается объект при использовании клавиш со стрелками вместе с .
  • Папка дополнительного словаря: используется для добавления папки, в которую можно дополнительно скопировать файлы словарей. При запуске каталог сканируется и добавляются обнаруженные словари.
Параметры комбинаций клавиш
  • Используйте эти параметры для определения собственных комбинаций клавиш. См. тему Настраивание комбинаций клавиш.
Прочие параметры
  • Текущие установки для заливок, кистей, стилей объектов, настроек пользователя по умолчанию и шрифтов можно сбросить до заводских настроек по умолчанию.
Чтобы задать установки, выполните перечисленные далее действия.
  • В меню Affinity Designer выберите Установки.
  • В меню Правка выберите пункт Установки.

Для перехода к определенным параметрам установок воспользуйтесь клавишами со стрелками, всплывающим меню или инструментом поиска в верхней части диалогового окна.

С помощью установок системы Apple можно управлять элементами интерфейса пользователя Affinity, например режимом полосы прокрутки.
