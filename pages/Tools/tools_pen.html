Инструмент «Перо»

Инструмент «Перо» используется для точного рисования кривых и фигур.

Клавиша быстрого доступа к инструменту : P

В этом инструменте есть четыре режима: Режим «Перо», Интеллектуальный режим, Режим «Многоугольник» и Режим «Линия»; их можно выбрать на контекстной панели инструментов. В каждом режиме используется свое начертание линии.

Режим «Перо»

Самый эффективный и точный режим, используемый для создания кривых типа «Безье» и фигур с плавными или острыми углами и узлами.

Интеллектуальный режим

Легко создавайте плавные кривые и фигуры посредством нажатия мыши и размещения узлов.

Режим «Многоугольник»

Используется для рисования прямых линий с резкими узлами и фигур с прямыми краями.

Режим «Линия»

Используется для рисования прямых линий, состоящих из одного сегмента.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Режим: используется для переключения между режимами Перо, Интеллектуальный, Многоугольник и Линия (см. выше).
  • Преобразовать: используется для преобразования выделенного узла в Резкий, Плавный или Интеллектуальный узел.
  • Действие: используется для выполнения операций с кривыми.
    • Разбить кривую: размыкает фигуру в выделенном узле.
    • Замкнуть кривую: соединяет начальный и конечный узлы для создания замкнутой фигуры.
    • Сгладить кривую: изменяет линию или фигуру, добавляя или удаляя узлы, для придания более эстетичного вида.
    • Соединить кривые: соединяет две отдельные кривые друг с другом для создания одной кривой. Обе кривые необходимо выделить с помощью и инструмента «Узел» или по мере рисования.
    • Обратить кривую: позволяет рисовать с противоположного конца кривой. Начальный узел становится активным, и можно продолжать рисовать.
  • Привязать: используется для регулирования привязки узлов.
    • Выровнять по узлам выделенных кривых: используется для привязки любого перетаскиваемого узла к любому другому узлу на той же или другой кривой.
    • Привязать к геометрии выделенных кривых: используется для привязки перетащенных узлов к тому же или другому контуру или узлу кривой.
    • Привязать все выделенные узлы при перетягивании: используется для привязки нескольких выделенных узлов при перетягивании к целевому узлу на любой выделенной кривой.
    • Привязать маркеры, расположенные вне кривых: используется для выравнивания перетянутого управляющего маркера с прилегающим узлом.

    Эти параметры совершенно не зависят от глобальных параметров привязки.

  • Использовать заливку: если этот параметр включен, вогнутый участок линии заполняется по мере рисования текущим цветом заливки (установленным на панели «Цвет»).
