Выделение и перемещение монтажных областей, а также изменение их размера

Выделенные монтажные области можно перемещать по панели вставки, а также можно изменять их размер в любой момент в соответствии с потребностями.

Монтажные области также можно поворачивать и наклонять непосредственно на панели вставки с помощью инструмента «Монтажная область» или с помощью инструментов точного редактирования на панели «Трансформация».

Выделенные монтажные области также можно изменять разнообразными способами с помощью разных инструментов в зависимости от изначального способа создания монтажной области. Например, монтажная область, преобразованная из замкнутой фигуры, по-прежнему может быть отредактирована как замкнутая фигура с помощью инструмента «Узел», а монтажная область, преобразованная из художественного текста, по-прежнему может быть отредактирована как текст с помощью инструмента «Художественный текст».

Выбор монтажных панелей.

Выполните одно из перечисленных ниже действий.

  • Выберите инструмент «Монтажная область» и нажмите монтажную область.
  • На панели Слои выберите запись монтажной области.
  • Выберите инструмент «Перемещение» и выполните одно из перечисленных ниже действий.
    • Нажмите метку монтажной области.
    • Нажмите любую пустую монтажную область.

Путем нажатия в ходе выполнения одной из указанных выше операций можно выбрать несколько монтажных областей.

Изменение положения монтажной области.

Выполните одно из перечисленных ниже действий.

  • Перетащите выбранную монтажную область с помощью инструмента «Монтажная область» или инструмента «Перемещение».
  • Перетащите метку монтажной области с помощью инструмента «Перемещение».
  • Выберите монтажную область и на панели Трансформация откорректируйте параметры X и (или) Y.
Изменение размера монтажной области.

Выделите монтажную область и выполните одно из перечисленных ниже действий.

  • Перетащите маркеры выделения на монтажной области с помощью инструмента «Монтажная область» или инструмента «Перемещение».
  • На панели Трансформация откорректируйте параметры Ш (ширина) и (или) В (высота).

Клавиши-модификаторы

Далее перечислены клавиши-модификаторы, которые можно использовать при работе с инструментом «Монтажная область».

  •  — ограничение пропорций монтажной области в момент создания (квадрат) или при изменении размера.
  •  — изменение размера монтажной области относительно ее центра.
  •  — поворот монтажной области вокруг противоположной метки-манипулятора.
  • С помощью нажатия правой кнопки мыши можно вращать монтажную область вокруг противоположной метки-манипулятора.
Поворот, наклон и отражение монтажной области.

Выделите монтажную область и выполните одно из перечисленных ниже действий.

  • Выберите инструмент «Монтажная область» или инструмент «Перемещение» и перетащите маркер поворота монтажной области, или установите курсор возле одного из угловых маркеров монтажной области и перетащите его по странице.
  • Выберите инструмент «Монтажная область» или инструмент «Перемещение», установите курсор возле одного из боковых маркеров монтажной области и перетащите его по странице.
  • На панели инструментов выберите параметр отражения или поворота.
  • На панели Трансформация откорректируйте параметры П (поворот) и (или) Н (наклон).
