Инструмент «Прозрачность»

Инструмент «Прозрачность» позволяет применить градиенты прозрачности к векторным и текстовым объектам и изменить их.

Градиентная прозрачность представлена в виде контура, соединенного узлами оттенков серого. Значение оттенков серого для узла представляет собой значение непрозрачности / прозрачности.

  • Черный цвет представляет собой полную непрозрачность (100 % непрозрачности / 0% прозрачности).
  • Белый цвет представляет собой полную прозрачность (0 % непрозрачности / 100 % прозрачности).

Инструмент «Прозрачность» влияет только на градиентную прозрачность объекта — обводка и заливка затрагиваются в равной степени. В этом случае отсутствует параметр цвета, в отличие от инструмента «Заливка».

Сплошную прозрачность можно применить на панели «Цвет» или «Образцы» с помощью ползунка Непрозрачность. Чтобы перейти от градиента к сплошной прозрачности, выберите «Нет» во всплывающем меню на контекстной панели инструментов «Тип», а затем примените непрозрачность из любой вкладки к выделенному объекту.

Параметры

На контекстной панели инструментов можно настроить указанные ниже параметры.

  • Тип: используется для преобразования типа прозрачности объекта: Нет, Линейный, Эллиптический, Радиальный или Конический.
  • Выделяйте, выбирайте и изменяйте градиентную прозрачность объекта. Для этого необходимо нажать на образец, чтобы отобразилась всплывающая панель. Дополнительную информацию о доступных параметрах см. в теме Редактирование градиента.
  • Повернуть градиент: используется для поворота примененного градиента на 90°.
  • Обратный градиент: конечные узлы меняются местами. (Все промежуточные узлы также меняют свое положение соответственно.)
  • Сохранить коэффициент заливки: если этот параметр выключен (по умолчанию), можно изменять положение конечных узлов в индивидуальном порядке, что приводит к изменению коэффициента заливки. Если этот параметр включен, конечные узлы фиксируются, обеспечивая сохранение коэффициента заливки (т. е. при изменении положения одного узла автоматически обновится другой). Этот параметр оказывает воздействие только на эллиптическую и растровую заливки.
