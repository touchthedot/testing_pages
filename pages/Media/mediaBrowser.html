Медиабраузер

Медиабраузер — это место хранения растровых изображений, где пользователь может быстро найти сохраненные файлы.

Перейти в медиабраузер можно с помощью меню Вид.

Параметры

В диалоговом окне доступны приведенные далее параметры.

  • В верхней части отображаются добавленные папки, а в нижней — пиктограммы изображений из этих папок. Папка «Изображения» добавлена по умолчанию.
    • Перетащите ползунок, чтобы отрегулировать уровень масштабирования.
    • Введите текст в поле поиска, чтобы отобразить только изображения, соответствующие критериям поиска.
Параметры диалогового окна, доступ к которым осуществляется с помощью контекстных меню (посредством выполнения щелчка мышью)
  • Удалить: используется для удаления выбранной папки из медиабраузера. Папка и ее содержимое не удаляются с компьютера пользователя.
  • Перезагрузить: используется для синхронизации с выбранной папкой, обеспечивающей отображение актуального содержимого в медиабраузере.
  • Показать в Finder: используется для открытия выбранной папки или файла в Finder.
  • Открыть с помощью Preview.app: используется для открытия выбранного файла в режиме предварительного просмотра.
  • Быстрый просмотр: используется для открытия выбранного файла в режиме быстрого просмотра.
Добавление фотографий в медиабраузер.

Выполните одно из перечисленных ниже действий.

  • Перетащите папку непосредственно в медиабраузер.
  • Выполните щелчок мышью по верхней части диалогового окна, выберите Добавить, а затем перейдите к папке, выберите ее и нажмите Открыть.
Добавление мультимедиа из медиабраузера.

Выполните одно из перечисленных ниже действий.

  • Перетащите пиктограмму изображения из медиабраузера непосредственно на страницу.
  • Дважды нажмите пиктограмму изображения. Изображения добавляются в виде нового документа.
