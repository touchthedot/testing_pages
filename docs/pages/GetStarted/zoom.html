Масштабирование

Масштабирование отдельных областей документа — это важная часть работы над композицией. Возможны несколько вариантов масштабирования.

Увеличение или уменьшение масштаба документа.

Выполните одно из перечисленных ниже действий.

  • Нажмите инструмент «Масштаб» (клавиша быстрого доступа Z) и выполните перечисленные ниже действия.
    • Перетащите курсор по странице: вправо для увеличения масштаба или влево для его уменьшения.*
    • Нажмите в любом месте страницы, чтобы увеличить масштаб на один шаг.
    • Нажмите и, не отпуская, щелкните левой кнопкой мыши в любом месте страницы, чтобы уменьшить масштаб на один шаг.
    • Нажмите и, не отпуская, перетащите курсор по странице, чтобы увеличить масштаб определенной области.
    • На контекстной панели инструментов выберите уровень масштабирования.
  • На контекстной панели инструментов нажмите инструмент «Вид» (клавиша быстрого доступа H) и выберите уровень масштабирования.
  • Выберите сначала меню Вид, а затем параметры масштабирования.
  • Воспользуйтесь и колесом прокрутки мыши (если есть).
  • Увеличивайте или уменьшайте масштаб с помощью параметров панели Навигатор.

* Это действия по умолчанию. В пункте Установки (параметры инструментов) можно изменить настройку и выбрать увеличение масштаба определенной области.

Комбинации клавиш:

  • Увеличить: с помощью +;
  • Уменьшить: с помощью –;
  • По размеру окна: с помощью и цифры 0 или двойного щелчка по значку инструмента «Вид»;
  • Масштаб 100 %: с помощью и цифры 1 или двойного щелчка по значку инструмента «Масштаб» (при этом пиксели документа сопоставляются непосредственно с пикселями экрана);
  • Выделение на весь экран: с помощью 0 или двойного щелчка по пиктограмме панели «Слой»;
  • Выделение на весь экран: с помощью 0 или двойного щелчка по пиктограмме панели «Слой»;
  • Фактический размер: с помощью и цифры 8;
  • Размер пикселя: с помощью и цифры 9.

Информацию для пользователей, работающих с трекпадом или мышью Magic Mouse, см. в разделе Жесты.

При создании нового документа точный размер документа на экране определяется настройками параметра «Единицы измерения документа». Например, если выбраны пиксели, при экспорте и во время просмотра на экране в масштабе 100 % документ будет отображаться в своем точном размере в пикселях. Аналогичным образом, если установлены физические единицы измерения (например, миллиметры), документ будет представлен в точном размере печати в масштабе 100 %. Размер печати определяется разрешением (тчк/дюйм), установленным в элементе Настройка документа.
