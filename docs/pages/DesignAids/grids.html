Сетки

На изображении может отображаться непечатаемая сетка, которая не будет видна в экспортированном документе и которая предназначена для более точного размещения объектов.

Сетка накладывается поверх страницы с целью выравнивания положения объектов. По умолчанию сетки имеют серый цвет, однако можно выбрать любой другой цвет.

Сетки могут быть автоматическими или фиксированными. В случае с первым типом (выбран по умолчанию) частота промежуточных делений меняется при увеличении или уменьшении масштаба. Если же используется сетка второго типа, то частота промежуточных делений всегда остается неизменной (независимо от уровня масштабирования).

Наибольшая эффективность достигается при использовании сеток в сочетании с функцией привязки, в частности, когда включен параметр «Привязать к сетке». Для сеток можно установить любые единицы измерения, и они идеально совпадут с линейками (если включено их отображение).

Для того чтобы показать или скрыть сетку, выполните перечисленные далее действия.
  • В меню Вид выберите пункт Показать сетку.
Для регулирования интервала автоматической сетки выполните перечисленные далее действия.

Выберите инструмент «Масштаб» и выполните одно из перечисленных ниже действий.

  • Увеличьте масштаб для отображения более мелких делений сетки.
  • Уменьшите масштаб для отображения более крупных делений сетки.

При любом уровне масштабирования сетка отображается в виде блоков, разделенных на промежуточные деления.

Для создания фиксированной сетки квадратной формы выполните перечисленные далее действия.
  1. В меню Вид выберите пункт Диспетчер сеток и осей.
  2. Снимите флажок Использовать автоматическую сетку.
  3. Установите значения для параметров Интервал и Деления.
  4. Нажмите Закрыть.
Для настройки цвета сетки выполните перечисленные далее действия.
  1. В меню Вид выберите пункт Диспетчер сеток и осей.
  2. Нажмите Линии сетки или Линии промежуточного деления для отображения всплывающей панели и установки цвета линии.
  3. Перетащите ползунки для того, чтобы установить значение непрозрачности цвета для одной или обеих линий.
  4. Нажмите Закрыть.

Если вы хотите, чтобы цвет или степень непрозрачности линий промежуточных делений и основных линий сетки отличалась, нажмите символ Связать цвета сетки рядом с образцами.

Для создания сетки прямоугольной формы выполните перечисленные далее действия.
  1. В меню Вид выберите пункт Диспетчер сеток и осей.
  2. Снимите флажок Использовать автоматическую сетку.
  3. Выберите Расширенные и снимите флажок Равномерно.
  4. Выполните одно из перечисленных ниже действий.
    • Установите значения для параметров Интервал и Деления для первой и второй осей.
    • Выберите Фиксированные пропорции, установите значения для параметров Интервал и Деления для первой оси, а затем установите значения для параметров Пропорции и Деления для второй оси.
  5. Нажмите Закрыть.
Для создания фиксированной угловой сетки выполните перечисленные далее действия.
  1. В меню Вид выберите пункт Диспетчер сеток и осей.
  2. Снимите флажок Использовать автоматическую сетку.
  3. Выберите Расширенные.
  4. Во всплывающем меню Тип сетки выберите вариант Двухосевая пользовательская.
  5. Настройте угол для одной из осей.
  6. Нажмите Закрыть.

Для настройки изометрических углов установите для первой и второй осей значения 30° и 150° соответственно.

Для изменения единиц измерения документа выполните перечисленные далее действия.

Выполните одно из перечисленных ниже действий.

  • Предварительно активировав инструмент «Вид» или инструмент «Масштаб», выберите единицу измерения во всплывающем меню на контекстной панели инструментов.
  • В меню Файл выберите Настройка документа, а в диалоговом окне Единицы измерения документа всплывающего меню выберите единицу измерения.
