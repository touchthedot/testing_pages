Корректировка «Инвертировать»

Создайте негативное изображение, инвертировав все цветовые каналы.

Для этой корректировки пользовательские параметры не предусмотрены.

Можно использовать инвертирование в рамках создания масок краев и для применения повышения резкости и других корректировок к выбранным участкам изображения.

Чтобы воспользоваться большим количеством функций для создания негативного (инвертированного) изображения, примените корректировку Уровни.
